#### 3.1.1 素質配點建議 (2025年最新)

*   **STR (力量)**：主點，初期優先點滿90-100，後期可繼續提升至120，最大化物理輸出。
*   **AGI (敏捷)**：副點，建議點至70-90，確保足夠的攻擊速度與閃避，提升輸出效率與生存。
*   **DEX (靈巧)**：點至40-60，保證命中率，避免攻擊MISS影響輸出節奏。
*   **VIT (體質)**：點至30-50，提供基礎血量與防禦，防止被秒殺。
*   **LUK (幸運)**：初期可不點，若有餘力或搭配暴擊裝備，可考慮點至30-40提升暴擊率。
*   **INT (智力)**：不點。

#### 3.1.2 核心技能選擇與搭配

*   **劍術修練 (Sword Mastery)**：必滿，提升劍類武器攻擊力。
*   **雙手劍修練 (Two-Handed Sword Mastery)**：若使用雙手劍，必滿。
*   **猛擊 (Bash)**：前期主要輸出技能，後期作為銜接或單點技能。
*   **挑釁 (Provoke)**：PVE拉怪、PVP降低敵方防禦，重要輔助技能。
*   **霸體 (Endure)**：PVE衝鋒、PVP免受技能中斷，核心生存/輔助技能。
*   **狂擊 (Magnum Break)**：範圍輸出技能，PVE清怪效率高。
*   **集中攻擊 (Concentration)**：提升自身攻擊力與命中，爆發前必開。
*   **進階二轉後 (騎士/十字軍)**：
    *   **騎士**：騎乘攻擊、連刺攻擊、螺旋擊刺 (極致單體爆發)。
    *   **十字軍**：聖十字攻擊、盾擊、盾牌猛擊 (若轉型為半輸出半坦)。

    *   **單手劍**：+10以上「卡特爾之劍」(高攻擊、攻速)，搭配盾牌提升生存。
*   **盾牌**：若使用單手劍，推薦「巴基力之盾」(高防禦、屬性減傷)。

*   **披肩**：「巴基力披肩」(減傷、屬性抗性)、「惡魔波利斗篷」(高迴避)。
*   **靴子**：「戰士長靴」(STR、HP)、「變形鞋」(HP、SP)。
*   **飾品**：「力量戒指」(STR)、「將軍的戒指」(STR、ATK)。
*   **卡片**：
    *   **武器**：邪骸戰士卡(ATK)、海盜之王卡(STR)。

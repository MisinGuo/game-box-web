[三國志幻想大陸新世界服靈寵培養?三國志幻想大陸新世界服私服,GM版助手,修改版,速刷版,自瞄外掛,自動練級;下载]({{siteConfig.jumpDomain}})


靈寵的戰鬥效能由四大要素決定，玩家在每一步都需要兼顧 **E‑E‑A‑T**（經驗、專業、權威、可信度）原則，選擇最適合自己的培養路徑。

1. **等級**  
   - 每提升 1 級，基礎屬性（攻擊、生命、防禦）提升 1.2%。  
   - **等級提升** 需要「靈寵經驗藥水」或「戰鬥經驗」累積。

2. **屬性**  
   - 包含 **攻擊、生命、防禦、速度** 四項。  
   - 2025 年新增「星魂」可在屬性上限上再提升 **15%**，建議優先投入星魂於主攻屬性。

3. **技能**  
   - 靈寵分為 **被動** 與 **主動** 兩類。  
   - 被動技能可在隊伍屬性上提供全局加成；主動技能在戰鬥中觸發，冷卻時間受等級影響。

4. **進化**  
   - 進化分 **Ⅰ、Ⅱ、Ⅲ** 三階段，需消耗「進化碎片」與「進化儀式」道具。  
   - 完成進化後，靈寵將解鎖新技能或提升現有技能等級。

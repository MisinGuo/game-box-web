#### 攻擊與防禦屬性：輸出與生存的基石

*   **物理傷害/抗性 (Physical Damage/Resistance)：** 最基礎的傷害類型。高物理傷害能有效應對缺乏特定抗性的敵人，而高物理抗性則是面對大量近戰或物理投射物敵人的生存保障。
*   **元素傷害/抗性 (Elemental Damage/Resistance)：** 包括火焰、冰霜、閃電等。地牢中許多怪物和環境會帶有特定的元素傷害或抗性。玩家需要平衡自身的元素抗性，並準備多樣化的元素傷害手段。
*   **混沌傷害/抗性 (Chaos Damage/Resistance)：** 一種無視大部分物理和元素減傷的特殊傷害類型。混沌傷害來源相對稀少但極具威脅，因此提升混沌抗性在面對某些特定敵人時至關重要。
*   **護甲 (Armor)：** 主要減少物理傷害。在高難度地牢中，足夠的護甲能顯著提升物理傷害的減免效率。
*   **閃避 (Evasion)：** 提供躲避攻擊的機率。對於敏捷型角色，高閃避率是其主要生存手段。
*   **格擋 (Block)：** 提供格擋攻擊的機率，通常與盾牌相關。格擋成功可以完全抵擋傷害，是坦克型角色的核心防禦屬性。

#### 輔助與特殊屬性：戰術深度的來源

*   **暴擊率/暴擊傷害 (Critical Chance/Damage)：** 提升傷害爆發力的關鍵。高暴擊率配合高暴擊傷害能讓你的輸出能力質變。
*   **攻擊速度/施法速度 (Attack Speed/Cast Speed)：** 影響技能和普攻的頻率。更高的速度意味著在相同時間內造成更多傷害或觸發更多效果。
*   **生命值/能量護盾 (Life/Energy Shield)：** 兩種主要的生命池。生命值是所有角色通用的生存屬性，而能量護盾則提供額外的保護層，通常與智力型角色相關。
*   **抗性穿透 (Resistance Penetration)：** 降低敵人特定抗性的效果，這比單純提升傷害更為有效，尤其是在面對高抗性敵人時。
*   **異常狀態機率/效果 (Ailment Chance/Effect)：** 觸發燃燒、冰緩、感電、中毒等異常狀態的能力。適當的異常狀態能有效控場或提升傷害。
*   **移動速度 (Movement Speed)：** 提升在地牢中探索和規避傷害的效率。

#### 獨特地牢詞綴與其對屬性的影響

2025年的《DoE流亡地牢》引入了更多動態詞綴，這些詞綴會直接影響地牢內所有生物（包括玩家）的屬性。例如：

*   **「元素共鳴」：** 地牢內所有怪物獲得額外元素抗性，同時玩家的元素穿透效果提升。
*   **「混沌侵蝕」：** 玩家混沌抗性降低，怪物混沌傷害提升。
*   **「堅韌光環」：** 所有怪物獲得額外護甲和生命，但攻擊速度降低。

玩家在進入地牢前，務必仔細閱讀當前地牢的詞綴，並據此調整自己的屬性配置或戰術。

*   **核心：** 在輸出與生存屬性之間取得平衡。例如，適中的暴擊率與暴擊傷害，搭配足夠的生命值、滿抗性，以及一定的護甲/閃避。
*   **策略：** 這種配置適合不確定地牢詞綴或希望應對各種挑戰的玩家。它可能無法達到極致的輸出或生存，但在大多數情況下都能有穩定的表現。

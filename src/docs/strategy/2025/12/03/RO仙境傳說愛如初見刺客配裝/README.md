刺客的能力值配點應圍繞其輸出特性進行。

*   **力量 (STR)：** 核心輸出屬性，直接影響物理攻擊力。建議優先點滿或點至觸發裝備加成門檻。
*   **敏捷 (AGI)：** 提升攻速與閃避率。雙刀刺客需高敏捷以達到攻速上限；拳刃刺客也需一定敏捷來保證技能銜接流暢。
*   **靈巧 (DEX)：** 少量DEX能縮短技能詠唱時間、提升命中率。對於技能型刺客有一定價值，但通常不會主點。
*   **體質 (VIT)：** 提升生命值與物理防禦。在追求生存的PVP或高難度PVE中，適量點數能顯著提升容錯率。
*   **智力 (INT)：** 提升魔法攻擊與SP上限。對刺客基本無用。
*   **幸運 (LUK)：** 提升暴擊率、暴擊傷害與完全迴避。暴擊流刺客的核心屬性，建議在STR和AGI達標後優先投入。

**2025年推薦配點方向：**

*   **雙刀暴擊流：** STR點至90-100 → AGI點至攻速達標 (約100-120) → LUK點滿 → 剩餘點數分配至VIT或補足STR。
*   **拳刃技能流 (PVE)：** STR點滿 → AGI點至70-90 (確保技能後搖流暢) → LUK點至50-70 (輔助暴擊) → 剩餘點數分配至VIT。
*   **拳刃爆發流 (PVP)：** STR點滿 → VIT點至50-70 (提升生存) → AGI點至60-80 → LUK點至50-70。

A3：最推薦的搭配是「4 達爾文 + 2 費列特」。達爾文提供輸出，費列特強化治療與輔助。若資源有限，可先湊齊達爾文4件套，再考慮費列特。

**Q4：蝕暗在暗隊中，和薇拉．緋耀如何取捨？**
A4：兩者定位不同。薇拉．緋耀是純粹的暗屬性增傷輔助，而蝕暗是治療與混合傷害輔助。在需要高治療量時選擇蝕暗，需要極致暗屬性增傷且生存壓力不大時選擇薇拉。通常情況下，暗隊會同時攜帶這兩位輔助。

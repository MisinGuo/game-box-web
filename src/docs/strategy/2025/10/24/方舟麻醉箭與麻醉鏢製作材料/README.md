了解兩者的差異，才能在不同情境下做出最佳選擇。

| 特性     | 麻醉箭 (Tranquilizer Arrow)                               | 麻醉鏢 (Tranquilizer Dart)                                 |
| :------- | :------------------------------------------------------- | :------------------------------------------------------- |
| **所需技術** | 早期遊戲，只需弓、十字弓等基礎武器。                     | 中後期遊戲，需步槍、長管步槍等進階武器。                   |
| **製作成本** | 較低，主要為基礎資源。                                   | 較高，涉及金屬、火藥、稀有資源。                           |
| **麻痺值** | 中等，對目標造成較多物理傷害。                           | 高，對目標造成極低物理傷害。                               |
| **射程** | 較短，受弓箭拋物線影響。                                 | 較長，彈道筆直，精準度高。                                 |
| **適用對象** | 小型、中型、血量較低的恐龍，或對傷害不敏感的生物。       | 大型、高等級、血量高或對傷害敏感的恐龍，追求高效率馴服。 |
| **優點** | 易於製作，早期即可使用，成本低廉。                       | 麻痺值高，傷害低，馴服效果穩定，適合高價值目標。           |
| **缺點** | 傷害較高，可能意外擊殺目標；麻痺值較低，馴服時間長。     | 製作成本高昂，需要更多進階資源和設施。                   |

**選擇建議：**

*   **遊戲初期至中期：** 主要使用麻醉箭配合十字弓，馴服如三角龍、劍龍、甲龍等。
*   **遊戲中後期：** 當你擁有熔爐、化學實驗室和長管步槍後，應優先轉向麻醉鏢，以應對更強大的馴服挑戰。

## 提升麻醉效率的額外技巧 (2025 最新建議)

除了正確製作和選擇麻醉工具，以下技巧能進一步提升你的馴服效率：

1.  **瞄準頭部：** 對大部分生物而言，射擊頭部會造成額外的麻痺值，大幅縮短馴服時間。
2.  **利用地形與陷阱：** 將目標生物引誘至預設陷阱（如恐龍門、石牆圍欄）中，可有效限制其行動，避免逃跑或反擊，讓你安全地進行射擊。
3.  **麻醉藥/黑色漿果的儲備：** 在馴服過程中，即使生物昏迷，其麻痺值也會緩慢衰減。務必準備足夠的麻醉藥或黑色漿果，放入昏迷生物的背包中，強制餵食以維持麻痺狀態。
4.  **觀察生物麻痺值衰減：** 透過望遠鏡或馴服追蹤器，密切監控目標生物的麻痺值衰減速度，適時補充麻醉箭或麻醉鏢。
5.  **避免不必要的傷害：** 盡量減少麻醉工具以外的物理傷害，因為過高的傷害可能導致生物死亡，或降低馴服效率。麻醉鏢的低傷害特性使其在這方面表現更優。
6.  **團隊合作：** 馴服大型或高難度生物時，與隊友分工合作，一人負責拉怪、一人負責射擊，能大幅提高成功率。
7.  **考慮生物的抗性：** 某些生物（如泰坦巨蟒）對麻醉箭有較高抗性，而對麻醉鏢效果更佳。馴服前可查閱相關資料。

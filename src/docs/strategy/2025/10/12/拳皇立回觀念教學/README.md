「立回」（たちまわり, Tachima-wari）源自日語，在格鬥遊戲中意指「中距離的攻防、移動、牽制與心理博弈」。它不是指一套固定的連段，而是一種動態的、以「控制」為核心的戰略思維。在KOF中，立回的目標是：

*   **控制空間：** 透過走位、跳躍、前後 dash 等移動，限制對手的行動範圍，或為自己創造有利的進攻或防禦位置。
*   **製造機會：** 利用牽制技、假動作、防禦反擊等手段，引誘對手露出破綻，從而發動連段或造成傷害。
*   **讀取對手：** 觀察對手的習慣、反應模式，預判其下一步行動。

簡單來說，立回就是一場無形的「棋局」，雙方玩家在螢幕中段不斷試探、博弈，爭奪主動權，為最終的擊殺創造條件。

*   **壓制與退讓：**
    *   **壓制（Pressure）：** 利用攻擊迫使對手防禦，限制其行動。例如，用波或長手普攻讓對手不敢輕舉妄動。
    *   **退讓（Zoning/Keepaway）：** 透過遠距離攻擊或移動，保持安全距離，消耗對手或等待機會。這是許多波昇角色的立回核心。
    *   **引誘（Whiff Punish）：** 故意在對手攻擊範圍外晃動，引誘其出招落空（whiff），然後抓住破綻進行懲罰（punish）。這是高端立回的常見技巧。

*   **假動作與預判：**
    *   **假動作（Feint）：** 透過看似要攻擊卻取消，或做出特定動作引誘對手反應。例如，假裝要跳入，卻在空中緊急迴避落地。
    *   **預判（Prediction）：** 根據對手的習慣（例如，喜歡在特定情況下跳入或發波），提前做出應對。這需要大量的對戰經驗和觀察力。

#### 2.2 幀數知識與應用

雖然KOF的幀數資訊不如一些遊戲那麼普及，但了解「有利幀」和「不利幀」的概念，能讓你更精準地判斷攻防時機。

*   **有利幀（Plus Frames）：** 當你的攻擊命中或被防禦後，你比對手更快恢復行動，這時你處於有利幀。可以繼續壓制或發動另一次攻擊。
*   **不利幀（Minus Frames）：** 當你的攻擊命中或被防禦後，你比對手慢恢復行動，這時你處於不利幀。如果對手反應夠快，可能會反擊你。

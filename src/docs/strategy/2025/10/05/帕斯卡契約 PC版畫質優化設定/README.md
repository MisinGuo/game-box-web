*   **紋理品質 (Texture Quality)**：
    *   **影響**：遊戲中物體表面紋理的清晰度。主要消耗顯示卡記憶體（VRAM）。

        *   **低**：若VRAM ≤ 4GB，則建議設定為低，以避免卡頓。
*   **陰影品質 (Shadow Quality)**：
    *   **影響**：遊戲中陰影的精細度、解析度和距離。對CPU和GPU都有較大影響。
    *   **建議**：這是最影響FPS的設定之一。
        *   **高**：若追求畫面最佳效果且硬體強勁。
        *   **中**：性價比較高，能保留大部分陰影細節，同時提升不少FPS。
        *   **低**：若FPS不佳，可大幅提升效能。
*   **抗鋸齒 (Anti-Aliasing)**：
    *   **影響**：消除物體邊緣的鋸齒感，使畫面更平滑。對GPU效能有較大影響。
    *   **建議**：
        *   **TAA (Temporal Anti-Aliasing)**：遊戲內預設選項，效果較好，但可能導致畫面輕微模糊。
        *   **SMAA/FXAA**：如果遊戲提供其他選項，這些通常效能開銷較小，但效果不如TAA。
        *   **關閉**：若FPS不足，可直接關閉，但鋸齒感會明顯。在高解析度下，鋸齒感會相對不明顯。
*   **後處理效果 (Post-Processing Effects)**：
    *   **影響**：包括景深 (Depth of Field)、泛光 (Bloom)、動態模糊 (Motion Blur) 等。這些效果能增強畫面氛圍，但對GPU有一定負擔。
    *   **建議**：
        *   **景深 (Depth of Field)**：模擬相機焦點效果。建議**關閉**，尤其是在戰鬥中，模糊背景反而會影響判斷，也能提升FPS。
        *   **泛光 (Bloom)**：使光源周圍產生光暈。可依個人喜好決定，對效能影響較小。
        *   **動態模糊 (Motion Blur)**：在快速移動時產生模糊效果。建議**關閉**，許多玩家認為會讓畫面不清晰，且可能引起暈眩，關閉後也能提升FPS。
*   **環境光遮蔽 (Ambient Occlusion - AO)**：
    *   **影響**：模擬物體之間和物體褶皺處的陰影，讓畫面更有立體感。對GPU有顯著影響。
    *   **建議**：
        *   **開啟 (低/中)**：如果硬體允許，開啟可大幅提升畫面真實感。建議從「中」開始嘗試。
        *   **關閉**：若FPS不足，優先關閉此選項。
*   **視距 (View Distance)**：
    *   **影響**：遠處物體和景物的渲染距離。對CPU和GPU都有影響。
    *   **建議**：
        *   **高**：若硬體強勁，可設定為高，遠景更清晰。
        *   **中**：平衡選項，能看到大部分重要景物，同時節省資源。
        *   **低**：若FPS不佳，可適當降低。
*   **植被細節 (Foliage Detail)**：
    *   **影響**：樹木、草地等植被的數量和精細度。對CPU和GPU都有影響。
    *   **建議**：
        *   **高**：若硬體強勁且喜歡豐富的自然景觀。
        *   **中/低**：若FPS不佳，降低此選項能獲得不錯的效能提升。

1.  **作為素材的綠色潛能武器（舊武器）：**
    *   **潛能顏色：** 必須是「綠色字體」的潛能。這通常意味著武器至少達到傳說（Legendary）或神話（Mythic）等級，且潛能已透過潛能方塊洗練至綠色。

3.  **繼承石：**
    *   這是進行綠色繼承的關鍵道具。繼承石通常可透過遊戲內活動、里程商店、特殊禮包或拍賣場獲得。請務必確認您擁有所需數量的繼承石。
    *   **注意事項：** 繼承石有時效性，請留意使用期限。

    *   繼承過程需要消耗一定數量的楓幣，費用會根據武器的星力等級和潛能等級而有所不同。請預留足夠的楓幣。

*   如果素材武器的潛能是「綠色字體」，則會完整繼承。

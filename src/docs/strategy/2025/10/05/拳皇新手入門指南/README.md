    *   `←`：後退/防禦

*   **攻擊鍵**：

    *   **緊急迴避**：`LP+LK` 或 `HP+HK` (依版本而異)，向前或向後閃避攻擊，消耗少量能量。
    *   **吹飛攻擊**：`HP+HK`，擊退對手並造成硬直，可破防。
    *   **爆氣/MAX模式**：消耗能量槽，進入強化狀態，可使用EX特殊技或強化連段。

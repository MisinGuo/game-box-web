*   **解鎖Tek：** Tek傳送器、Tek複製器。

#### 2. 大猩猩（Megapithecus）

*   **地點：** 藍色方舟（Blue Obelisk）或傳送器。
*   **解鎖Tek：** Tek發電機、Tek燈、Tek水上氣墊船。
*   **推薦戰鬥恐龍：**
    *   **霸王龍（Rex）：** 同樣是主力輸出。
    *   **恐狼（Direwolf）/斑龍（Allosaurus）：** 攻擊速度快，適合圍攻。
    *   **猛獁象（Mammoth）：** 血量高，可作為肉盾。
*   **貢品：** 毒液、巨齒鯊牙、泰坦巨蟒毒液、傷齒龍牙、獸皮。
*   **攻略要點：** 大猩猩會投擲石頭，造成擊退和高額傷害。戰鬥場地較大，注意恐龍走位，避免被推下懸崖。

#### 3. 噴火龍（Dragon）

*   **地點：** 紅色方舟（Red Obelisk）或傳送器。
*   **解鎖Tek：** Tek護甲（全套）、Tek步槍、Tek劍、Tek手榴彈。
*   **推薦戰鬥恐龍：**
    *   **南方巨獸龍（Giganotosaurus）：** 高傷害，但體型大，不易控制。
    *   **霸王龍（Rex）：** 依然是穩定的選擇。
    *   **異特龍（Allosaurus）：** 群體攻擊有流血效果。
    *   **建議：** 噴火龍是飛行Boss，且有火焰吐息，對非抗火恐龍傷害極高。建議使用高培育的霸王龍群，並搭配一隻**豬鱷（Kaprosuchus）**或**袋鼠（Procoptodon）**吸引火力。**務必**準備高品質的**抗火藥劑（Lazarus Chowder）**和**仙境傳說（Fairy Tale）**。
*   **貢品：** 元素礦、龍鱗、毒液、巨齒鯊牙、泰坦巨蟒毒液、傷齒龍牙、獸皮。
*   **攻略要點：** 龍的火焰吐息傷害非常高，盡量讓恐龍從側面或後方攻擊。玩家可以騎乘**帝鱷（Sarco）**或**棘背龍（Spino）**在水中躲避火焰。

#### 4. 監察者（Overseer）與 Tek洞穴（Tek Cave）

*   **地點：** 方舟島火山頂部的Tek洞穴。
*   **解鎖Tek：** 所有Boss Tek藍圖的Alpha版本解鎖後，會解鎖Tek洞穴內部的Tek傳送門。擊敗監察者後，將解鎖**Tek洞穴傳送器**以及**Tek克隆機**。
*   **推薦戰鬥恐龍：**
    *   **霸王龍（Rex）：** 數量越多越好，高品質鞍具是必須的。
    *   **猛獁象（Mammoth）/三角龍（Triceratops）：** 作為肉盾。
    *   **建議：** Tek洞穴是方舟島的最終挑戰，裡面充滿了高等級的野生生物。需要大量高品質的恐龍，並做好長期戰鬥的準備。洞穴內無法騎乘飛行生物。
*   **貢品：** 擊敗前面三個Boss的Alpha難度後獲得的方舟神器（Artifacts）和Boss戰利品。
*   **攻略要點：**
    *   **時間限制：** 洞穴內有時間限制，必須在規定時間內抵達監察者。
    *   **高溫：** 洞穴內部溫度極高，需準備抗熱裝備和補給。
    *   **多階段戰鬥：** 監察者會變身為育母蜘蛛、大猩猩和噴火龍的形態，需要多種戰術應對。
    *   **玩家等級：** 玩家必須達到特定等級才能進入Tek洞穴（Gamma 80級, Beta 90級, Alpha 100級）。

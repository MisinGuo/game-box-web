守護者是團隊的核心坦克，擁有極高的生存能力和控制技能。對於無課玩家來說，守護者的裝備屬性主要堆疊生命、防禦、韌性，這些屬性相對容易通過副本掉落和活動獲取。在團隊中，即使輸出不高，其強大的坦度和保護能力也能讓你在任何PVE和PVP場合都備受歡迎。2025年版本中，守護者的坦度依然是遊戲頂尖。

**核心技能與玩法：**
*   **核心坦傷：** 利用「聖光」系列技能吸引仇恨、減傷，保護隊友。
*   **控制能力：** 「制裁」等技能提供擊飛、暈眩，打斷敵人施法。
*   **團隊增益：** 部分技能能為隊友提供護盾或減傷。
*   **無課養成策略：**
    *   **裝備：** 優先選擇增加生命、雙防、韌性的裝備。龍語方面，優先選擇防禦類、減傷類。

    *   **言靈：** 優先選擇「時間零」、「不朽」等增強生存能力的言靈。
    *   **夥伴：** 搭配增加防禦、生命或治療的夥伴。

**無課優勢：**
執行者以其獨特的雙形態（人形輸出/異形控制）和高爆發而聞名。對於無課玩家而言，執行者在PVE單體輸出和PVP的戰場收割上都有不錯的表現。雖然對裝備有一定要求，但由於其靈活的機制和高額的瞬間爆發，在資源有限的情況下，透過操作彌補戰力差距的空間較大。

**核心技能與玩法：**

*   **異形：** 近戰形態，擁有更強的控制、位移和生存能力。
*   **靈活切換：** 根據戰況在兩種形態間切換，是玩好執行者的關鍵。
*   **無課養成策略：**
    *   **裝備：** 優先選擇增加智力、洞察、暴擊和多重打擊的裝備。龍語方面，注重提升輸出和暴擊。
    *   **血統：** 選擇提升爆發傷害、技能冷卻縮減或形態切換增益的血統。
    *   **言靈：** 優先選擇「冥界之門」、「風之加護」等提升輸出或機動性的言靈。

**無課優勢：**
獵鷹是遊戲中典型的遠程物理輸出職業，擁有極高的持續傷害和廣闊的攻擊範圍，同時具備一定的自保能力。對於無課玩家來說，獵鷹的輸出循環相對穩定，且在PVE副本中能提供可靠的傷害。在PVP中，雖然身板較脆，但優秀的走位和控制技能可以有效風箏敵人。2025年版本中，獵鷹的PVE地位依然穩固。

*   **陷阱與控制：** 「衛星射線」、「脈衝炸彈」等技能提供範圍控制和傷害。
*   **機動性：** 「閃現」等技能提供快速位移，保持安全距離。
*   **無課養成策略：**
    *   **裝備：** 優先選擇增加力量、靈巧、暴擊和多重打擊的裝備。龍語方面，注重物理攻擊和暴擊。
    *   **血統：** 選擇提升持續傷害、暴擊率或技能冷卻縮減的血統。

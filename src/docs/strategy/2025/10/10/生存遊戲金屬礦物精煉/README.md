## 四、核心金屬礦物及其用途（2025年通用參考）

雖然不同遊戲設定有差異，但以下幾種金屬礦物在生存遊戲中通常扮演著核心角色：

*   **鐵礦 (Iron Ore)**：最基礎、需求量最大的金屬。精煉後得到鐵錠，用於製作工具、武器、基礎建築、機械零件。
*   **銅礦 (Copper Ore)**：重要性僅次於鐵礦。精煉後得到銅錠，主要用於電線、電路板、電機等電氣設備，以及部分合金。
*   **錫礦 (Tin Ore)**：常與銅礦結合製成青銅（Bronze），是初期比鐵更堅硬的合金，用於工具、武器。
*   **金礦 (Gold Ore)**：精煉後得到金錠，主要用於高科技電路、貴重物品、交易貨幣，或作為特殊合金成分。
*   **銀礦 (Silver Ore)**：精煉後得到銀錠，與金類似，用於電子元件、特殊裝飾品或交易。
*   **鋁土礦 (Bauxite)**：精煉後得到鋁錠，用於輕量化結構、飛機零件、高壓電纜。
*   **鉑礦 (Platinum Ore)**：稀有且堅硬。精煉後得到鉑錠，用於頂級工具、裝甲、高階電子元件和催化劑。
*   **鈦礦 (Titanium Ore)**：極其堅固且耐腐蝕。精煉後得到鈦錠，是製造太空船、深海潛艇、末日裝甲的關鍵材料。
*   **鈾礦/鈽礦 (Uranium/Plutonium Ore)**：核能發電的燃料，精煉過程複雜且危險，但能提供巨量電力。

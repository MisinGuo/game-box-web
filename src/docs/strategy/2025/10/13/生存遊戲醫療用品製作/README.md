以下是2025年生存遊戲中，幾種最常見且實用的醫療用品**製作流程與材料**分析，幫助玩家掌握核心技術。

#### 1. 繃帶/布條 (Bandages/Rags)

*   **用途：** 止血、治療輕微割傷與擦傷，部分遊戲可緩解感染初期症狀。
*   **核心材料：**
    *   **布料 (Cloth)：** 最常見，可從廢棄衣物、窗簾、沙發等處拆解獲得。
    *   **植物纖維 (Plant Fiber)：** 部分遊戲允許使用植物纖維直接製作簡易繃帶。
*   **製作流程：** 通常在玩家背包、基礎工作台或裁縫台直接合成。這是遊戲初期最易取得的醫療品。
*   **進階提示：** 盡早囤積布料，因為它是許多中後期醫療品的基礎。

#### 2. 基礎藥膏/消毒劑 (Basic Salves/Disinfectants)

*   **用途：** 治療輕微燒傷、感染、皮膚病變，或用於傷口消毒防止感染擴散。
*   **核心材料：**
    *   **植物藥材 (Herbs/Mushrooms)：** 如蘆薈、金盞花、特定藥用蘑菇等。
    *   **酒精 (Alcohol)：** 從發酵的植物或化學合成獲得。
    *   **鹽 (Salt)：** 某些配方需要。
*   **製作流程：** 通常需要基礎工作台或簡易化學台。將採集到的藥材與酒精、水等混合。
*   **進階提示：** 優先學習辨識有藥用價值的植物，它們在野外隨處可見，是您初期治療感染的重要保障。

#### 3. 急救包/醫療箱 (First Aid Kits/Medkits)

*   **用途：** 快速回復大量生命值，治療多種負面狀態，是應對高強度戰鬥或危險探索的必備品。
*   **核心材料：**
    *   **繃帶/紗布 (Bandages/Gauze)：** 大量需求。
    *   **消毒劑 (Disinfectant)：** 確保傷口清潔。
    *   **止痛藥 (Painkillers)：** 緩解疼痛，消除負面效果。
    *   **抗生素 (Antibiotics)：** 治療感染。
    *   **塑膠/金屬 (Plastic/Metal)：** 用於製作容器或注射器。
*   **製作流程：** 需要中級或高級化學工作台。配方通常較為複雜，涉及多種初級醫療品的組合。
*   **進階提示：** 製作急救包需要較多資源和時間，建議在基地穩定後，有計劃地批量生產，並妥善儲存。

#### 4. 解毒劑 (Antidotes)

*   **用途：** 治療中毒狀態，這在許多遊戲中是致命的威脅。
*   **核心材料：**
    *   **稀有藥草/真菌 (Rare Herbs/Fungi)：** 通常生長在危險區域或特定生態環境中。
    *   **化學溶劑 (Chemical Solvents)：** 透過化學合成或從廢棄物中提煉。
    *   **血液/腺體 (Blood/Glands)：** 來自特定危險生物。
*   **製作流程：** 幾乎都要求在高級化學工作台或實驗室進行，且可能需要專門的技能。
*   **進階提示：** 在前往已知有毒生物或環境的區域前，務必預先製作並攜帶足夠的解毒劑。

#### 5. 夾板/石膏 (Splints/Casts)

*   **用途：** 治療骨折，防止骨折惡化，恢復行動能力。
*   **核心材料：**
    *   **木材/金屬條 (Wood/Metal Scraps)：** 作為支撐。
    *   **布料/繩索 (Cloth/Rope)：** 用於固定。
    *   **樹脂/石灰 (Resin/Lime)：** 製作石膏的粘合劑。
*   **製作流程：** 可以在基礎工作台完成，但高級石膏可能需要化學台。
*   **進階提示：** 骨折會嚴重影響行動，因此夾板是您探索高風險區域時的必備應急品。

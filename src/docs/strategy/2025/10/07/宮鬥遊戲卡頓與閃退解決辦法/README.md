#### 2.1 基礎排除法：快速修復常見問題

1.  **重啟遊戲與裝置：** 這是最簡單卻往往最有效的方法。重啟可以清除暫存記憶體，關閉異常進程，讓系統重新載入。
2.  **檢查網路連線：** 確保您的Wi-Fi訊號穩定，或行動數據訊號良好。嘗試切換網路類型（例如從Wi-Fi切換到行動數據，反之亦然），或重啟路由器。
3.  **關閉背景應用程式：** 在進入遊戲前，手動關閉所有不必要的背景應用程式，釋放記憶體與處理器資源。
4.  **清理遊戲快取：**
    *   **Android：** 進入「設定」>「應用程式」> 找到您的宮鬥遊戲 >「儲存空間」>「清除快取」。**注意：不要點擊「清除資料」，否則可能需要重新下載遊戲數據。**
    *   **iOS：** iOS系統沒有直接清除單一應用程式快取的功能。您可以嘗試「卸載App」（保留文件與數據）再重新安裝，或直接刪除遊戲並重新下載。
5.  **檢查儲存空間：** 確保您的裝置至少有3-5GB的可用儲存空間。刪除不必要的照片、影片或應用程式。

1.  **更新遊戲至最新版本：** 遊戲開發商會持續發布更新，修復Bug並優化效能。2025年的遊戲版本通常會針對新硬體和作業系統進行優化。
2.  **更新作業系統：** 確保您的手機或平板電腦運行的是最新版本的Android或iOS。系統更新通常包含效能改進和Bug修復。
3.  **調整遊戲內畫質設定：**
    *   進入遊戲設定，將「畫質」、「解析度」、「特效」等選項調低。
    *   關閉不必要的動畫或粒子效果。
    *   對於部分宮鬥遊戲，可能提供「省電模式」或「流暢模式」，請優先選擇。
4.  **裝置效能模式設定（僅限部分Android裝置）：**
    *   部分Android手機提供「遊戲模式」或「高效能模式」。在系統設定中啟用這些模式，可以優先分配資源給遊戲。
    *   關閉「省電模式」，因為它會限制CPU頻率，導致遊戲卡頓。
5.  **檢查裝置過熱情況：**
    *   避免在充電時玩遊戲。
    *   取下手機殼，讓裝置更好地散熱。
    *   使用散熱風扇或將裝置放置在通風良好的環境。
    *   如果裝置持續過熱，可能是硬體老化或電池問題，建議送修。

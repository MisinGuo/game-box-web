理解不同承傷類型，是做出明智選擇的第一步：

*   **高血量/重甲型（如超巡、重巡）**：依賴高生命值和重型裝甲硬吃傷害。她們通常擁有較高的防禦屬性，能有效抵禦物理傷害，但面對穿甲彈或無視防禦的攻擊時可能較為脆弱。

*   **自我恢復型（如部分重巡、輕巡）**：透過技能在戰鬥中恢復自身血量。這類艦船能延長戰鬥續航，但通常需要一定的時間才能發揮最大效益。

顯示設定是影響遊戲效能與視覺體驗最直接的環節。

*   **解析度（Resolution）：**
    *   **建議：** 盡可能使用顯示器原生解析度。若硬體不足，可適當降低解析度或使用FSR/DLSS等升頻技術。2025年，DLSS 3.5和FSR 3.0等技術已相當成熟，能以更低的渲染解析度輸出接近原生畫質的影像，大幅提升幀數。
*   **幀率上限（FPS Limit）：**
    *   **建議：** 若顯示器支援高刷新率（如144Hz、240Hz），且硬體能達到，則解除幀率限制或設定為顯示器刷新率。若幀數不穩定，可設定一個略低於平均幀數的上限，以保持遊戲體驗的穩定性。
*   **垂直同步（V-Sync）：**
    *   **建議：** 若有畫面撕裂問題且幀數穩定高於顯示器刷新率，可開啟。但V-Sync會增加輸入延遲。2025年，G-Sync和FreeSync等自適應同步技術是更好的選擇，它們能有效消除撕裂同時保持低延遲。
*   **紋理品質（Texture Quality）：**
    *   **建議：** 若顯存（VRAM）充足，可設定為高。紋理品質對視覺影響大，但對幀數影響相對較小，主要消耗顯存。
*   **陰影品質（Shadow Quality）：**
    *   **建議：** 陰影是效能殺手。適當降低陰影品質（從極高降到高或中）通常能獲得顯著的幀數提升，而視覺差異不一定很大。2025年的軟體陰影技術優化，讓中等品質也能有不錯的效果。
*   **抗鋸齒（Anti-Aliasing）：**
    *   **建議：** MSAA、SSAA等傳統抗鋸齒技術對效能影響較大。2025年，TAA（Temporal Anti-Aliasing）或DLSS/FSR內建的抗鋸齒效果更為主流且高效。若追求極致幀數，可選擇FXAA或直接關閉。
*   **環境光遮蔽（Ambient Occlusion - AO）：**
    *   **建議：** AO能讓畫面更有立體感，但對效能有一定影響。可從最高降至中等，或選用SSAO等效能較低的選項。
*   **光線追蹤（Ray Tracing - RT）：**
    *   **建議：** 2025年，光線追蹤技術已更加普及，但仍是硬體資源消耗大戶。若顯卡支援且性能強勁，可開啟以體驗更真實的光影效果；否則，建議關閉或調低相關設定，並搭配DLSS/FSR使用。

要有效打斷BOSS技能，首先必須理解遊戲中的控制效果如何作用於敵方單位。明日方舟的「技能中斷」通常是指透過施加特定的控制效果，使正在讀條、蓄力或釋放技能的敵方單位暫停或取消其動作。

#### 1. 中斷的種類與效果

*   **眩暈 (Stun)：** 最常見且直接的中斷方式。被眩暈的單位會停止一切動作，包括技能讀條。
*   **束縛 (Bind)：** 使敵方單位無法移動，但通常不影響其攻擊或技能釋放。然而，對於需要移動後才能釋放的技能，束縛能間接起到中斷作用。
*   **凍結 (Freeze)：** 類似眩暈，使敵方單位停止一切動作，但通常附帶額外傷害或易傷效果。
*   **睡眠 (Sleep)：** 使敵方單位停止一切動作，但受到傷害後會立即醒來。需要精準的時機配合。
*   **沉默 (Silence)：** 使敵方單位無法使用其特殊能力或技能。對於依賴特殊技能的BOSS，沉默是極為有效的控制。
*   **位移 (Force Retreat/Shift)：** 透過推拉效果改變敵方位置，對於需要特定站位或讀條的BOSS技能，能有效打斷其施放流程。

#### 2. BOSS技能的特性：可中斷與不可中斷

並非所有BOSS技能都能被中斷。通常，帶有明顯讀條、蓄力動畫或前搖的技能，其可中斷性較高。而一些被動觸發、瞬間生效或具備「不可阻擋」特性的技能，則往往無法被常規手段中斷。博士們在攻略新關卡時，應仔細觀察BOSS的技能提示與動作模式，判斷其是否具備中斷點。

*   **睡眠 + 爆發：** 讓BOSS進入睡眠，待隊伍準備好爆發後，再解除睡眠並集火。

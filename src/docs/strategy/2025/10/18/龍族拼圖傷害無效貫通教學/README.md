傷害無效貫通主要有兩種發動方式：**覺醒技能發動** 和 **主動技能發動**。

#### 1. 覺醒技能發動（3x3 方塊消除）

這是最常見也是最重要的DVP發動方式。當隊伍中的寵物擁有「傷害無效貫通」覺醒技能時，你需要透過**消除特定屬性的9顆寶珠，排列成3x3的正方形**來發動。

*   **發動條件：** 消除9顆相同屬性寶珠，形成一個3x3的方塊。
*   **效果：** 擁有DVP覺醒的寵物，其攻擊力將無視敵人的傷害無效盾。通常，一個DVP覺醒會提供2.5倍的傷害加成，多個DVP覺醒則會以乘法疊加，提供極高的單體爆發。
*   **實戰技巧：**
    *   **寶珠供給：** 隊伍中必須有穩定的寶珠供給能力。多數情況下，這意味著你需要攜帶轉珠技能或生成指定屬性寶珠的寵物。
    *   **盤面規劃：** 在進入有傷害無效盾的關卡前，預先規劃盤面，盡量將目標屬性寶珠集中。
    *   **Combo數：** 發動DVP會佔用9顆寶珠，導致Combo數減少。因此，隊伍中需要有額外的Combo加成（如隊長技能、覺醒技能）來彌補。
    *   **練習：** 3x3方塊消除需要一定的熟練度。建議在練習模式或簡單關卡中多加練習，熟悉手感。

#### 2. 主動技能發動（貫通技能）

部分寵物擁有「無視傷害無效盾」的主動技能。這類技能通常會在發動後的特定回合內，使隊伍所有攻擊無視傷害無效盾，或者對敵人造成固定傷害並無視傷害無效盾。

*   **發動條件：** 啟動寵物的主動技能。
*   **效果：** 技能持續期間，所有傷害無視無效盾。
*   **實戰技巧：**
    *   **技能冷卻：** 貫通技能的CD通常較長，需要合理規劃使用時機。
    *   **覆蓋率：** 在面對連續的傷害無效盾敵人時，確保技能能夠覆蓋所有需要貫通的樓層。
    *   **輔助搭配：** 搭配加速或減CD的技能，可以提高貫通技能的覆蓋率。
    *   **2025年趨勢：** 隨著遊戲難度提升，單純的貫通技能已較少見，更多是與增傷、轉珠等效果結合，成為多功能技能。

在2025年的PAD環境中，優秀的DVP寵物是隊伍構成的基石。以下是幾種值得關注的DVP寵物類型和選擇方向：

#### 1. 核心DVP打手（多DVP覺醒）

這些寵物通常擁有2個或更多的「傷害無效貫通」覺醒，是隊伍主要的貫通輸出點。搭配潛在覺醒的「傷害上限突破」和「屬性殺手」，能打出天文數字般的傷害。

*   **選擇標準：** 高攻擊力、多DVP覺醒、主動技能能提供增傷或轉珠。
*   **2025年趨勢：** 許多新推出的強勢寵物會自帶3個甚至4個DVP覺醒，或透過超覺醒獲得額外的DVP，極大提升了單體爆發能力。同時，雙變身隊長也流行將DVP覺醒作為變身後的強力輸出手段。

#### 2. 寶珠供給者與轉珠寵

沒有足夠的寶珠，DVP覺醒就無法發動。因此，能穩定提供9顆指定屬性寶珠的轉珠寵至關重要。

*   **選擇標準：** 技能CD短、能生成大量指定屬性寶珠、全版轉珠或版面調整。
*   **2025年趨勢：** 能夠生成「指定屬性+心珠」的轉珠寵最為泛用，既能確保DVP發動，又能回復血量。此外，一些CD極短的轉珠寵（如2CD或3CD）也成為主流，確保每回合都有寶珠可用。

#### 3. 隊長技能輔助

雖然隊長技能通常不直接提供DVP，但優秀的隊長技能能為DVP提供極大的便利。

*   **選擇標準：** 提供高倍率攻擊加成、高血量/回復加成、固定追打或追加Combo、以及**易於發動的寶珠限制**（例如：指定屬性4顆以上消除、L字消除等，這些條件與DVP的3x3消除不衝突甚至能互補）。
*   **2025年趨勢：** 雙變身隊長依然是主流，變身後提供極高的基礎倍率和多功能性。能夠提供額外Combo數的隊長技能，對於DVP隊伍尤其重要，能有效彌補3x3消除造成的Combo數損失。

#### 4. 輔助貫通寵（主動技能貫通）

當覺醒DVP難以穩定發動，或需要應對多層傷害無效盾時，具備主動技能貫通的寵物就能派上用場。

*   **選擇標準：** 技能CD短、持續回合數長、能與其他增傷技能疊加。
*   **2025年趨勢：** 純粹的貫通技能較少，更多是結合了傷害吸收無效、屬性吸收無效、攻擊力提升等多種效果的複合技能。

要最大化DVP的效益，需要考慮多方面因素。

#### 1. 隊伍編成核心思路

*   **穩定的寶珠供給：** 至少2-3個轉珠寵，確保每回合或每隔一回合能有足夠的寶珠發動DVP。
*   **核心DVP打手：** 至少1-2隻擁有大量DVP覺醒的寵物，作為主要輸出。
*   **Combo數加成：** 透過隊長技能或覺醒技能（如Combo強化），彌補3x3消除造成的Combo數損失，確保傷害倍率。
*   **其他輔助覺醒：** 傷害吸收無效、屬性吸收無效、操作時間延長、封條回復等，都是高難度關卡必備。
*   **潛在覺醒：** 核心DVP打手務必裝備「傷害上限突破」潛在覺醒，將單隻寵物的傷害上限從21億提升至42億，最大化DVP的爆發力。同時，根據關卡需求，可搭配屬性殺手潛覺。

#### 2. 實戰操作策略

*   **觀察敵方技能：** 在進入關卡前或進入新樓層時，務必確認敵人是否有傷害無效盾。
*   **預留寶珠：** 在沒有傷害無效盾的樓層，盡量不要一次性消除太多指定屬性寶珠，為下一層做準備。
*   **先轉珠後消除：** 在需要發動DVP的樓層，先使用轉珠技能，將盤面調整為有利於3x3消除的狀態，再進行消除。
*   **多重機制應對：**
    *   **DVP + 傷害吸收：** 先使用傷害吸收無效技能，再發動DVP。
    *   **DVP + 屬性吸收：** 先使用屬性吸收無效技能，再發動DVP。
    *   **DVP + 根性：** 在發動DVP的同時，確保能觸發隊長技能的「固定追打」或「追加攻擊」，避免敵人殘血根性復活。
*   **操作時間：** DVP操作需要較長的時間來排珠，確保隊伍有足夠的「操作時間延長」覺醒或徽章。

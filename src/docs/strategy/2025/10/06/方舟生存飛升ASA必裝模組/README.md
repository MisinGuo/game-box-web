ASA的模組安裝流程相較於舊版《方舟：生存進化》（ASE）有所簡化，主要透過遊戲內的Mod管理介面進行。

1.  **啟動遊戲**：進入《方舟：生存飛升》。
2.  **進入主選單**：在主選單界面，您會看到一個「Mod」或「模組」選項。
3.  **瀏覽與訂閱**：點擊進入模組頁面，您可以瀏覽CurseForge上的所有ASA模組。找到您想安裝的模組後，點擊「訂閱」或「安裝」。
4.  **下載與啟用**：遊戲會自動下載模組。下載完成後，您需要將模組從「可用模組」列表移動到「已啟用模組」列表。
5.  **調整模組載入順序（重要！）**：某些模組之間可能存在兼容性問題或特定的載入順序要求。通常，**大型內容模組（如新增生物）應優先載入，而QoL或建造模組則可放在後面。** 如果遇到遊戲崩潰或異常，嘗試調整模組順序。
6.  **重啟遊戲/伺服器**：完成模組配置後，務必重啟遊戲或伺服器，讓模組生效。

**重要提示：**
*   **兼容性**：在安裝新模組前，務必查看其頁面說明，確認是否與當前ASA版本兼容。
*   **伺服器管理**：如果您在私人伺服器上遊玩，所有玩家都必須安裝伺服器上啟用的相同模組，且模組版本需一致。伺服器管理員需要將模組ID添加到伺服器啟動參數中。
*   **備份存檔**：在嘗試大量新模組之前，養成備份遊戲存檔的好習慣，以防萬一。

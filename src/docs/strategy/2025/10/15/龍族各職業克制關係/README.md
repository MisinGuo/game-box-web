*   **近戰物理輸出 (如：狂戰士、劍士)：** 高爆發或高持續傷害，通常具備一定的控制或生存能力，依賴貼身戰鬥。
*   **遠程物理輸出 (如：弓箭手、槍手)：** 遠距離持續傷害，擅長風箏和走位，生存能力相對較弱。
*   **近戰防禦/坦克 (如：守護者、聖騎士)：** 高血量、高防禦，具備強大的減傷和控制技能，是團隊的堅實屏障。

*   **輔助/治療 (如：牧師、吟遊詩人)：** 提供治療、增益或減益效果，是團隊續航和戰術執行的關鍵。
*   **敏捷/刺客 (如：影舞者、潛行者)：** 高機動性、高單體爆發，擅長突襲和收割殘血，通常有潛行或位移技能。

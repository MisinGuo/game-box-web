### 步驟一：開啟你的收藏之路

1.  **開啟介面：** 按下鍵盤上的預設快捷鍵（通常是 `[ESC]` 選單內）或直接點擊遊戲介面下方的選單，找到「怪物收藏」圖示。
2.  **了解介面：** 你會看到不同大陸與區域的分類。點進去後，可以看到該區域需要收藏的怪物列表。灰色代表未登錄，亮起則代表已成功登錄。
3.  **確認目標：** 瀏覽一下，看看哪些收藏線路是你目前等級可以輕鬆完成的，並將其設定為初期目標。

### 步驟二：高效登錄怪物技巧

怪物的登錄完全是**機率性（RNG）**的，但你可以透過一些技巧來最大化效率：

*   **地圖選擇是關鍵：** 選擇怪物密集、地形平坦、重生速度快的地圖。一個好的掛機地圖能讓你在單位時間內擊殺更多怪物，從而增加登錄機會。
*   **善用輔助道具與系統：**
    *   **輪迴碑石：** 大幅增加地圖怪物重生數量與速度，是登錄怪物的最佳夥伴。
    *   **燃燒場地：** 燃燒越高的地圖，怪物重生速度越快。
    *   **怪物公園：** 部分在野外難以找到的怪物，可以在怪物公園的特定副本中找到。
*   **等級差距：** 根據玩家社群長期測試，角色等級與怪物等級差距過大（通常是超過40等）可能會降低登錄率，建議在合適的等級區間進行。

### 步驟三：派遣探險與領取獎勵

當你完成一整條橫向的怪物收藏時，就會解鎖一個探險欄位。

*   **派遣探險：** 點擊收藏介面下方的「去探險」按鈕，選擇你想派遣的隊伍和探險時間（1小時、3小時、6小時、24小時）。
*   **時間與獎勵：** 探險時間越長，獲得稀有獎勵箱（例如「相當不錯的探險箱」）的機率就越高。**24小時探險**是獲取可疑方塊和咒文痕跡CP值最高的選擇。
*   **領取獎勵：** 時間一到，記得回來領取獎勵箱，並馬上再次派遣，確保收益不中斷。

2.  檢查該MOD是否與你的遊戲版本或其他已安裝的MOD相容。
3.  閱讀MOD頁面的說明，確認是否有特殊的前置MOD需要安裝。
4.  如果問題無法解決，移除該MOD並恢復你的遊戲備份。

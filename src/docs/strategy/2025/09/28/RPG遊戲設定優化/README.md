*   **顯示模式 (Display Mode):** 永遠優先選擇 **全螢幕 (Fullscreen)** 模式。這能讓系統將最多資源分配給遊戲，減少輸入延遲。無邊框視窗模式雖然方便切換，但通常會犧牲些許效能。

### 2. 效能殺手級選項：優先調整的關鍵設定

以下幾個選項對幀數（FPS）影響巨大，是優化時首先要調整的目標。

*   **陰影品質 (Shadow Quality):**
    *   **影響：** 極高。即時運算的動態陰影需要大量GPU資源。
    *   **建議：** 從「高」降至「中」，通常能在不犧牲太多視覺效果的情況下，換取15-25%的效能提升。除非你的硬體頂尖，否則避免使用「極致」或「瘋狂」等級。

*   **環境光遮蔽 (Ambient Occlusion, AO):**
    *   **影響：** 高。它為物體接觸面增加細微的陰影，提升畫面的立體感。
    *   **建議：** 較新的HBAO+或更高效的演算法，在「中」等級就能提供很好的效果。如果幀數吃緊，這是可以考慮關閉的選項之一。

*   **反鋸齒 (Anti-Aliasing, AA):**
    *   **影響：** 中至高。用於平滑物體邊緣的鋸齒。
    *   **建議：** TAA (Temporal Anti-Aliasing) 是目前的主流，效能與效果平衡最佳。避免使用舊的、資源消耗極大的MSAA。如果你的解析度在2K或以上，可以嘗試將AA等級調低，因為高解析度本身就能減輕鋸齒感。

**Q1: 為什麼我把所有設定都調到最低，幀數還是很低？**
A: 這很可能是CPU瓶頸。某些RPG（特別是擁有大量NPC或複雜物理運算的開放世界遊戲）對CPU要求很高。降低陰影、LOD（細節層次）或NPC密度等與CPU相關的設定可能會有所幫助。

**Q2: 我應該鎖定幀數嗎？**
A: 如果你的幀數波動很大（例如在60-90之間跳動），將其鎖定在一個穩定的數值（如60 FPS）可以提供更平滑的體驗，並減少畫面撕裂。配合螢幕的G-Sync/FreeSync技術效果更佳。

**Q3: 紋理品質（Texture Quality）對效能影響大嗎？**
A: 紋理品質主要消耗的是顯示卡的視訊記憶體（VRAM），對幀數的直接影響相對較小。只要你的VRAM充足，可以盡量開高以獲得更清晰的材質細節。如果VRAM不足，則會導致遊戲卡頓。

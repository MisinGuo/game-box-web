儘管不同遊戲的系統名稱各異（如「分解重構」、「時裝回收」、「設計工坊」），其核心操作邏輯大同小異。掌握以下三個通用步驟，你就能在任何換裝遊戲中遊刃有餘。

#### 步驟一：找到分解/回收入口

分解功能的入口通常位於遊戲主界面的「工坊」、「設計」、「衣櫃」或相關的功能集合頁面中。常見的名稱包括：

*   **分解 / 重構**
*   **回收 / 時裝回收**
*   **資源轉換**

花點時間熟悉你所玩遊戲的介面，找到這個關鍵功能。

#### 步驟二：選擇要分解的服飾

這是最需要策略性思考的一步。遊戲通常會提供篩選器，讓你快速選定目標。

*   **批量選擇：** 優先使用「自動選擇低星級」或「選擇重複部件」功能，這能極大地提升效率。
*   **手動檢查：** 在批量選擇後，務必手動檢查一遍，**避免誤分解**了某些雖然星級低、但在特定關卡中有奇效的標籤服飾。
*   **鎖定珍品：** 大部分遊戲都提供「鎖定」功能。請務必將所有限定、絕版、或你特別珍愛的服飾鎖定，防止它們出現在分解列表中。

#### 步驟三：確認分解與材料獲取

選擇完畢後，系統會彈出一個確認視窗，詳細列出你將分解的服飾清單以及預計獲得的材料。**請務必仔細核對**，確認無誤後再點擊「分解」。一旦操作完成，通常是不可逆的。分解後，獲得的材料會自動存入你的背包。

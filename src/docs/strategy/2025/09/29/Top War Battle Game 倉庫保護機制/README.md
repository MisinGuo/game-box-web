《Top War》的倉庫保護機制，是一種內建的資源防禦系統。當您的基地遭受攻擊時，它會自動保護一部分存放在倉庫中的資源免於被掠奪。這意味著，即使您的防線被突破，您的所有資源也不會一次性被洗劫一空。

1.  **超出保護上限：** 任何超出倉庫保護上限的資源，在被攻擊時都可能被掠奪。這是玩家最容易忽略的盲點。

A: 延遲是雙方的事。即使你的網路完美，如果對手的網路環境差（例如使用Wi-Fi或在訊號不好的地區），或者你們之間的物理距離過遠（例如跨洲對戰），延遲依然會很高。這就是為什麼設定連線品質篩選如此重要。

以下將術語分為不同類別，方便玩家理解與記憶：

### 一、基礎操作與移動術語

這些是格鬥遊戲中最基本的行動，理解它們是構築戰術的第一步。

*   **立回（Neutral Game）**
    *   **定義：** 指雙方在未發動攻擊、保持一定距離時，透過移動、牽制、試探等方式，尋找進攻機會或壓制對手的階段。這是《拳皇》戰鬥的靈魂，考驗玩家的經驗與判斷。
    *   **應用：** 透過前後移動、小跳、短蹲等，控制距離，引誘對手出招或尋找破綻。
*   **步（Dash / Backdash）**
    *   **定義：** 快速向前衝刺（前衝）或向後撤退（後退）。
    *   **應用：** 前衝快速接近對手進行壓制或連段；後退拉開距離，躲避攻擊或重新調整立回。
*   **跳（Jump）**
    *   **定義：** 包括普通跳、大跳（長按上方向）、小跳（輕點上方向）。
    *   **應用：** 小跳是《拳皇》系列極具特色的進攻手段，能快速越過對手低段攻擊，進行中段壓制或連段起手。大跳則多用於跨越場地或躲避地面波。
*   **蹲（Crouch）**
    *   **定義：** 按下方向鍵使角色蹲下。
    *   **應用：** 躲避對手中段攻擊，同時也是發動下段攻擊或防禦下段攻擊的姿態。
*   **擋（Block / Guard）**
    *   **定義：** 防禦。分為站立防禦（擋上中段攻擊）和蹲下防禦（擋中下段攻擊）。
    *   **應用：** 在《拳皇》中，防禦是基本生存之道。熟悉對手攻擊的段位，適時切換站蹲防禦至關重要。

### 二、攻擊與連段術語

掌握這些術語，你將能理解如何有效打擊對手並造成大量傷害。

*   **目押（Link）**
    *   **定義：** 指在一個攻擊動作結束後，精準把握時機，在極短的硬直恢復時間內輸入下一個攻擊指令，使其成功連擊。通常沒有系統提示，完全依靠玩家的節奏感。
    *   **應用：** 許多角色的高傷害連段都包含目押，是區分高手與新手的關鍵技巧。
*   **取消（Cancel）**
    *   **定義：** 在一個招式尚未完全結束其硬直時，立即輸入另一個招式指令，使前一個招式被後一個招式中斷，形成流暢的連擊。
    *   **應用：** 《拳皇》中的取消系統非常豐富，例如普通技取消特殊技、特殊技取消必殺技、必殺技取消超必殺技等，是構築連段的核心。
*   **擇（Mix-up）**
    *   **定義：** 指讓對手面臨多種進攻選擇（例如上段/下段、投技/攻擊、左/右），迫使對手猜測並做出錯誤判斷，從而製造破綻。
    *   **應用：** 這是心理戰的重要一環，例如：小跳壓制（中段）與下段攻擊的「二擇」，讓對手難以防禦。
*   **壓制（Pressure）**
    *   **定義：** 透過連續的攻擊、投技威脅或有利的幀數，讓對手處於被動防禦狀態，難以反擊。
    *   **應用：** 在對手倒地後進行「起身壓制」，或在版邊將對手逼入絕境，都是常見的壓制手段。
*   **起攻（Initiate Attack）**
    *   **定義：** 指在立回階段，找到機會發動第一次攻擊，打破僵局。
    *   **應用：** 常見的起攻手段有小跳攻擊、前衝投、遠距離牽制技等。
*   **確反（Punish）**
    *   **定義：** 在對手出招後，利用其招式的硬直時間進行反擊，打出連段。通常指那些「確定能反擊」的招式。
    *   **應用：** 熟悉各角色招式的硬直幀數，是進行有效確反的基礎。成功的確反往往能帶來巨大的傷害。

### 三、系統與資源術語

理解這些遊戲系統，能讓你更有效地管理戰鬥資源，打出爆發性傷害。

*   **能量槽（Power Gauge）**
    *   **定義：** 位於螢幕下方的能量條，透過攻擊、防禦、被攻擊等方式積累，用於發動必殺技、超必殺技和MAX模式等。
    *   **應用：** 合理管理能量槽是《拳皇》戰術的重中之重，決定何時爆發、何時節省資源。
*   **MAX模式（MAX Mode）**
    *   **定義：** 《拳皇》系列的重要系統之一。在《拳皇XV》中，消耗兩格能量進入MAX模式，角色攻擊力提升，且部分招式可進行特殊取消（如MAX取消、CLIMAX取消），大幅提升連段潛力。
    *   **應用：** 通常用於連段的終結或在關鍵時刻爆發傷害，是反敗為勝的利器。
*   **SC (Super Cancel)**
    *   **定義：** 消耗能量，在特定必殺技的攻擊判定中，取消該必殺技的硬直，直接發動超必殺技。
    *   **應用：** 大幅延長連段，造成更高傷害，是《拳皇》經典的連段技巧。
*   **DC (Drive Cancel) / BC (Burn Cancel) / Shatter Strike / Rush / Climax Cancel (KOF XV)**
    *   **定義：** 這些是不同《拳皇》版本中類似「取消」機制的統稱。
        *   **KOF XIII 的 Drive Cancel (DC)：** 消耗Drive Gauge，在特殊技或必殺技中取消，連接另一個特殊技或必殺技。
        *   **KOF XIII 的 Burn Cancel (BC)：** 消耗一格能量，在特定必殺技中取消，連接另一個必殺技，通常用於延長連段。
        *   **KOF XV 的 Shatter Strike (SS)：** 消耗半格能量，可防禦反擊或用於連段中斷。
        *   **KOF XV 的 Rush：** 新手友好的自動連段系統。
        *   **KOF XV 的 Climax Cancel：** 在特定的超必殺技中取消，直接發動CLIMAX超必殺技。
    *   **應用：** 這些系統是各代《拳皇》構築華麗高傷連段的關鍵，玩家需根據當前遊戲版本來理解和運用。
*   **爆氣（MAX Mode Activation）**
    *   **定義：** 消耗能量進入MAX模式的動作本身。
    *   **應用：** 在《拳皇XIV》和《XV》中，爆氣後可進行特殊取消，是連段爆發的起點。

### 四、防禦與反擊術語

有效的防禦與反擊，是減少傷害、扭轉戰局的關鍵。

*   **防禦反擊（Guard Cancel）**
    *   **定義：** 在防禦狀態下，消耗能量發動的特殊反擊技，通常具有無敵幀或快速出招，能打斷對手的壓制。
    *   **應用：** 在《拳皇XV》中，稱為「防禦反擊攻擊」和「防禦反擊閃避」，是解圍和反擊的重要手段。
*   **拆投（Throw Tech / Escape Throw）**
    *   **定義：** 在對手發動投技的瞬間，輸入特定指令（通常是輕拳+輕腳），解除投技。
    *   **應用：** 預判對手投技，及時拆投，避免被連段或失去有利位置。
*   **無敵（Invincibility）**
    *   **定義：** 角色在某些招式（如部分必殺技、超必殺技的啟動幀）或狀態下，不會受到任何攻擊傷害。
    *   **應用：** 利用無敵幀躲避對手攻擊，或在對手起身時進行壓制。
*   **受身（Ukemi / Roll）**
    *   **定義：** 在被擊倒後，透過輸入特定指令快速起身，可向前或向後滾動，避免被對手持續壓制。
    *   **應用：** 這是《拳皇》重要的防禦機制，能有效打亂對手的起身壓制節奏。

### 五、進階概念與策略術語

這些術語關乎玩家的戰術思維和心理博弈。

*   **心理戰（Mind Games）**
    *   **定義：** 透過欺騙、預判、製造假象等方式，影響對手的判斷和決策。
    *   **應用：** 例如，反覆使用「擇」讓對手難以捉摸，或故意賣破綻引誘對手反擊。
*   **資源管理（Resource Management）**
    *   **定義：** 合理分配和利用能量槽、MAX模式等戰鬥資源。
    *   **應用：** 決定何時使用能量條打出高傷連段，何時保留能量進行防禦反擊或爆發。
*   **角色對策（Character Matchup）**
    *   **定義：** 針對不同角色特性，制定相應的戰術和打法。
    *   **應用：** 了解自己角色對陣特定對手角色的優劣勢，例如哪些招式能有效克制對手，或如何應對對手的核心壓制手段。

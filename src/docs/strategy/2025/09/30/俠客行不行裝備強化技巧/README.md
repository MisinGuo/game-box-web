這是最直接的屬性提升方式，消耗強化石和金幣。

*   **機制：** 消耗精煉石和金幣，隨機或固定提升裝備的額外屬性。

*   **2025年重點：** 部分高階裝備開放了「定向精煉」功能，可鎖定部分滿意屬性，大幅降低成本與風險。

*   **特性：** 寶石等級越高，提供的屬性加成越顯著。
*   **2025年重點：** 留意新增的「複合寶石」以及「寶石共鳴」系統，達到特定寶石組合或總等級可觸發額外增益。

#### 4. 升階與洗練（特殊材料）
*   **升階：** 將裝備從低品質提升至高品質（如藍裝升紫裝，紫裝升金裝），大幅提升基礎屬性和潛力。

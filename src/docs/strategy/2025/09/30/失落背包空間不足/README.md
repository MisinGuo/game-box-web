### 3. 裝備處理策略

裝備掉落是背包滿載的主要原因之一。

*   **低級裝備：** 大部分從野外或低級副本掉落的綠色、藍色品質裝備，除非有特殊屬性需求，否則直接分解獲得強化材料或銀幣。
*   **高級裝備：** 紫色、橙色品質的裝備，特別是首飾和能力石，需要仔細檢查其刻印、屬性、品質。

    *   **能力石：** 需仔細點擊雕刻，確認是否能刻出你需要的刻印組合。多餘的直接分解。
*   **分解 vs. 出售：**
    *   **分解：** 通常能獲得強化材料，這是提升裝備等級的必需品。
    *   **出售：** 低級裝備出售給NPC只能獲得少量銀幣。拍賣行出售則針對有市場價值的物品。新手階段，分解通常是更優的選擇。

### 4. 貨幣與貴重物品

*   **銀幣、金幣、藍色水晶、紅色水晶：** 這些貨幣通常會自動進入貨幣欄位，不佔用背包空間。
*   **特殊代幣：** 如島之心、巨人之心、偉大藝術品等收藏品，會自動進入收藏品欄位。

隨著遊戲理解的加深，進階玩家需要更精密的戰術來應對多變的戰場。

#### 3.1 進攻篇：打破防線的關鍵思維

成功的進攻需要針對敵方防禦隊伍的特性，選擇最有效的破解之道。

##### 3.1.1 隊伍構成原則與多樣化選擇

*   **知己知彼：** 觀察敵方防禦隊伍的符文、神器配置，判斷其核心機制（如高速爆發、反擊反傷、持續傷害等）。
*   **多樣化準備：** 準備至少3-4套不同類型的進攻隊伍，以應對不同防禦組合。
    *   **高速爆發隊：** 針對脆皮或無意志的防禦隊。
    *   **意志壓制隊：** 針對依賴控制的防禦隊，用意志符文抗住第一波控制。
    *   **持續傷害/消耗隊：** 針對高血量、高防禦的防禦隊。
    *   **反擊/反傷隊：** 利用敵方進攻特性，進行反擊。
*   **屬性克制：** 充分利用水克火、火克風、風克水的屬性優勢，降低戰損。

##### 3.1.2 常見防禦隊伍的破解之道

*   **高速先手壓制隊：**
    *   **特徵：** 隊伍中有高速拉條魔靈（如火鳳凰、風奧伯龍）或強力控制魔靈（如水奧庫魯斯）。
    *   **破解：** 使用更高速度的隊伍搶先手；或使用意志符文隊伍硬扛第一波，然後反擊；或使用復活魔靈（如風方舟）進行容錯。
*   **高韌性反擊隊：**
    *   **特徵：** 隊伍中有高血量、高防禦且帶有反擊或反傷效果的魔靈（如水伊芙、火吸血鬼）。
    *   **破解：** 使用無視防禦輸出魔靈（如風丑、火龍騎士）；或使用持續傷害（毒、炸彈）慢慢消耗；或使用禁強化、禁治療魔靈。
*   **保護核心輸出隊：**
    *   **特徵：** 隊伍中有強力輸出魔靈，並有保護或復活魔靈（如風仙人、光仙人）。
    *   **破解：** 優先擊殺保護魔靈；或使用單點爆發直接秒殺核心輸出；或使用禁復活技能。

##### 3.1.3 符文、神器與技能搭配的重要性

*   **符文：** 根據魔靈定位選擇符文套裝（暴走、意志、絕望、猛攻、吸血等），並確保副屬性達標。公會戰中，速度、暴擊傷害、抵抗、命中都極為關鍵。
*   **神器：** 2025年神器系統的影響力不容小覷。例如，增加特定技能傷害、減少技能冷卻、增加抵抗/命中等詞條，能顯著提升魔靈的戰鬥力。務必根據魔靈技能特性選擇合適的神器。
*   **技能搭配：** 隊伍中魔靈的技能應能形成良好互補，例如：拉條+群控+群傷、破甲+單點爆發、治療+免疫+反擊等。

#### 3.2 防禦篇：構築銅牆鐵壁的秘訣

優秀的防禦隊伍能有效消耗敵方進攻次數，為公會爭取勝利。

##### 3.2.1 防禦塔配置策略

*   **平衡性：** 盡量避免在單一防禦塔中放置過於相似的防禦隊伍，以防被同一套進攻隊伍輕鬆破解。
*   **屬性塔利用：** 將與屬性塔加成相符的魔靈放置在該塔，最大化加成效果。例如，水屬性加成塔放置水系防禦魔靈。
*   **核心魔靈保護：** 將最難被破解的防禦隊伍放置在較難進攻的塔位，或作為公會的「守門員」。

##### 3.2.2 防禦隊伍的選擇與克制關係

*   **多樣化組合：** 避免所有防禦隊伍都走純粹的「肉盾」或「高速」路線。
    *   **意志反擊隊：** 許多防禦魔靈會搭配意志符文，確保第一輪不被控制，然後利用反擊、反傷或強力技能進行反擊。
    *   **復活拖延隊：** 搭配復活魔靈（如風方舟、光奧庫魯斯），增加敵方擊殺難度。
    *   **控制反制隊：** 搭配能解除弱化、提供免疫的魔靈，或自身具有強力控制的魔靈。
*   **克制關係：** 考慮常見進攻隊伍的特性，選擇能對其造成困擾的魔靈。例如，針對高速隊，可放置高速高抵抗的魔靈；針對單點爆發隊，可放置高血量高防禦的魔靈。

##### 3.2.3 如何利用地形與屬性優勢

*   **地形：** 公會戰地圖的屬性塔加成是固定的，但每週的公會戰地圖可能略有不同。善用這些加成，將對應屬性的魔靈放入，能有效提升防禦強度。
*   **屬性：** 防禦隊伍應盡量避免被單一屬性克制。例如，一支純火隊容易被水隊伍針對，應考慮搭配其他屬性魔靈。

##### 3.2.4 防禦隊伍符文與神器優化

*   **符文選擇：** 防禦魔靈通常優先選擇意志、祝福、暴走、吸血等。高抵抗和高防禦/生命是關鍵。
*   **神器選擇：** 選擇能增加生存能力（如減少被暴擊傷害、增加抵抗）、反擊機率或特定技能效果的神器。

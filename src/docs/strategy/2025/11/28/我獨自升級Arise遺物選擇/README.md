遺物是《我獨自升級：Arise》中提升角色戰力的重要途徑，其提供的基礎屬性加成與強大的套裝效果，能直接影響你的輸出、生存與輔助能力。了解其核心機制，是做出明智選擇的第一步。

#### 1. 遺物類型與部位

每位角色可裝備六件遺物，分為：
*   **武器** (攻擊力、暴擊率等)
*   **項鍊** (攻擊力、生命值等)
*   **戒指** (攻擊力、暴擊傷害等)
*   **防具** (防禦力、生命值等)
*   **手套** (攻擊力、暴擊率等)
*   **鞋子** (防禦力、生命值、速度等)

不同部位的遺物，其主詞條（主要屬性）的種類有所差異，這是選擇時必須注意的。

#### 2. 套裝效果的重要性

遺物最大的價值在於其套裝效果。集齊2件、3件或4件同名遺物，將激活強大的額外增益。在2025年版本中，許多高難度內容都極度依賴這些套裝效果來提升總體戰力。

#### 3. 主詞條與副詞條選擇原則

*   **主詞條：** 這是遺物提供的主要屬性，通常數值較高。對於輸出角色，優先選擇攻擊力百分比、暴擊率、暴擊傷害；對於生存輔助角色，則優先生命百分比、防禦百分比。
*   **副詞條：** 遺物強化過程中隨機增加的次要屬性。理想的副詞條能進一步放大主詞條的效益。對於輸出角色，追求暴擊率、暴擊傷害、攻擊力百分比、穿透；對於生存輔助，則為生命百分比、防禦百分比、效果抵抗、速度。

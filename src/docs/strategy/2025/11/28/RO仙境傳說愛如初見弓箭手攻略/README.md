#### 1. AGI-DEX 普攻暴擊流 (俗稱「鳥獵」或「平射獵」)

這是弓箭手最經典也最受歡迎的流派之一，特別適合喜歡穩定高輸出、刷怪效率的玩家。

*   **DEX (靈巧)：** 優先點滿至90-100，提供高額ATK、命中率和攻速。
*   **AGI (敏捷)：** 點至90-100，確保高攻速（ASPD）和閃避率。高ASPD能最大化普攻輸出和觸發技能（如：獵鷹突擊）。
*   **LUK (幸運)：** 剩餘點數投資LUK，提升暴擊率和獵鷹突擊觸發機率。
*   **STR (力量)：** 少量投資，提升負重，方便攜帶更多藥水和戰利品。
*   **VIT (體質) / INT (智力)：** 幾乎不點，或依需求微量投資。

**2025年趨勢：** 由於裝備能提供大量攻速和暴擊率，DEX和AGI的基礎點數可以略微降低，轉而將更多點數投入LUK，以追求更高的暴擊傷害和獵鷹觸發率。部分玩家也會投入少量STR提升負重。

#### 2. DEX-INT 技能爆發流 (俗稱「陷阱獵」或「箭雨獵」)

此流派更側重於技能傷害，特別是在PVP或需要範圍輸出的場合表現優異。

*   **DEX (靈巧)：** 優先點滿，提供高額ATK、命中和技能詠唱速度。
*   **INT (智力)：** 大量投資，提升SP上限、SP恢復速度和技能傷害。
*   **AGI (敏捷)：** 少量投資，提升攻速，縮短技能後搖，或完全不點。
*   **LUK (幸運)：** 少量投資，提升暴擊率和獵鷹觸發率，或完全不點。
*   **VIT (體質)：** 依個人需求少量投資，提升生存能力。

**2025年趨勢：** 隨著新技能和裝備對技能傷害的加成，DEX-INT流派在特定副本和PVP中表現突出。但對裝備依賴度較高，且練級效率可能不如普攻流。

**小提示：** 建議新手玩家初期以AGI-DEX普攻流為主，練級效率高，裝備門檻相對較低。後期再根據個人喜好和資源轉型。

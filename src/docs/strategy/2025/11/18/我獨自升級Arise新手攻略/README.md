#### 2.2 武器系統深度解析：專武與通用武器

武器是提升獵人戰力的重要環節，分為「專屬武器」和「通用武器」。

*   **專屬武器：** 每個獵人都有其專屬武器，能大幅提升該獵人的特定技能效果或面板屬性，是提升戰力的首選。
*   **通用武器：** 適用於所有獵人，提供基礎屬性加成。在沒有專武的情況下，選擇與獵人屬性或技能契合的通用武器也是不錯的選擇。
*   **武器養成：**
    *   **升級：** 消耗金幣和武器強化材料提升基礎屬性。
    *   **突破：** 消耗相同的武器或特定材料，提升武器的等級上限和潛力。
    *   **詞條：** 武器會隨機生成詞條，優先選擇暴擊率、暴擊傷害、攻擊力加成等輸出型詞條。

神器類似於其他遊戲中的裝備，提供大量的屬性加成，並能組成套裝觸發額外效果。

*   **神器類型：** 通常分為頭盔、胸甲、手套、鞋子等部位，每個部位提供不同基礎屬性。
*   **套裝效果：** 收集相同套裝的神器（通常是2件套或4件套）能觸發強力的額外效果，例如提升攻擊力、暴擊率、生命值等。
*   **搭配原則：**
    *   **輸出型獵人：** 優先選擇提升攻擊、暴擊、暴傷的套裝。
    *   **輔助/坦克型獵人：** 選擇提升生命、防禦、治療效果或能量回復的套裝。

#### 2.4 成長系統：等級、突破、技能、能力值

除了裝備，獵人自身的成長也是戰力提升的關鍵。

*   **等級：** 消耗經驗值提升，解鎖新的技能或提升基礎屬性。
*   **突破：** 消耗特定材料，提升獵人的星級，大幅增加基礎屬性上限。

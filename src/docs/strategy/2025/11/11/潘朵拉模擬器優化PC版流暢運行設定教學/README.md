這是影響幀數最直接的選項。

*   **渲染器（Renderer/Backend）：**
    *   **首選Vulkan：** 如果您的顯示卡支援Vulkan API，強烈建議選擇Vulkan。它通常能提供比OpenGL更高的效能和更低的CPU開銷。
    *   **OpenGL：** 如果Vulkan不穩定或不支援，則選擇OpenGL。確保您的OpenGL版本至少為4.5。
    *   **Direct3D 11/12 (如果可用)：** 部分模擬器可能提供，效能通常介於Vulkan和OpenGL之間。
*   **內部解析度（Internal Resolution）：**
    *   **建議1x或2x：** 不要盲目追求高解析度。對於中低階PC，保持1x（原生解析度）能確保最佳流暢度。若硬體允許，可嘗試2x。超過2x對效能要求極高。
    *   **輸出解析度（Output Resolution）：** 保持與您的顯示器解析度一致即可。
*   **非等向性過濾（Anisotropic Filtering）：** 設定為「自動」或「2x/4x」。過高會增加GPU負擔。
*   **垂直同步（V-Sync）：**
    *   **開啟：** 如果您遇到畫面撕裂（screen tearing），請開啟V-Sync。這會將幀數鎖定在顯示器刷新率，但可能引入輕微輸入延遲。
    *   **關閉：** 如果您的幀數遠低於顯示器刷新率，或者您對輸入延遲敏感，可以嘗試關閉V-Sync以獲得更高幀數。
*   **Shader快取（Shader Cache）：** 務必「開啟」。這能將已編譯的Shader儲存起來，避免重複編譯，大幅減少載入時的卡頓。
*   **紋理過濾（Texture Filtering）：** 選擇「雙線性（Bilinear）」或「三線性（Trilinear）」。
*   **後處理效果（Post-processing Effects）：** 如FXAA、SMAA等抗鋸齒或色彩校正效果，建議「關閉」以提升效能。

    *   **速度駭客：** 某些模擬器提供，可能透過犧牲少量精確度來換取效能。請根據遊戲和模擬器版本選擇性開啟。

在組建隊伍前，首先要理解各角色的基本定位，並確保隊伍具備以下核心功能：

1.  **核心輸出 (DPS)：** 隊伍的主要傷害來源，負責快速擊敗敵人。可以是物理、魔法或真實傷害型。
2.  **堅實防禦 (Tank/副坦)：** 承受敵方主要傷害，保護後排輸出與輔助角色。具備高生命值、防禦力或減傷技能。
3.  **強力輔助 (Support)：** 提供增益效果（如攻擊力、防禦力提升）、治療、護盾，或削弱敵方（如減速、破防）。
4.  **戰場控制 (Crowd Control, CC)：** 透過暈眩、冰凍、沉默、擊退等技能，打斷敵方施法、限制其行動，為我方創造輸出機會。

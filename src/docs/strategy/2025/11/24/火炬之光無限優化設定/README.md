    *   **說明：** 遊戲畫面顯示的像素數量，直接影響清晰度。

*   **顯示模式 (Display Mode)：**
    *   **說明：** 全螢幕、視窗化、無邊界視窗。全螢幕模式通常性能最佳，因為它讓遊戲獨佔顯示資源。
    *   **推薦：** **全螢幕模式**。
*   **垂直同步 (V-Sync)：**
    *   **說明：** 同步遊戲幀率與顯示器刷新率，防止畫面撕裂，但可能增加輸入延遲。
    *   **推薦：** 如果畫面有撕裂感，可以開啟。若追求最低延遲且FPS遠高於刷新率，可關閉並搭配G-Sync/FreeSync技術。
*   **幀率限制 (FPS Limit)：**
    *   **說明：** 限制遊戲的最大幀率。
    *   **推薦：** 建議設定為顯示器刷新率的兩倍或三倍（例如144Hz顯示器設定288或432），或根據個人喜好設定一個穩定值。過高的FPS可能導致硬體過熱，過低則影響流暢度。
*   **材質品質 (Texture Quality)：**
    *   **說明：** 遊戲中物體表面細節的品質。對顯示卡記憶體（VRAM）需求較高。
    *   **推薦：**
        *   **高階顯卡 (8GB VRAM以上)：** 高/極高。
        *   **中階顯卡 (4-6GB VRAM)：** 中。
        *   **低階顯卡 (2-3GB VRAM)：** 低。
*   **陰影品質 (Shadow Quality)：**
    *   **說明：** 陰影的精細程度。對GPU和CPU都有較大負擔。
    *   **推薦：** **低**。這是最能提升幀率的選項之一，對遊戲體驗影響不大。
*   **特效品質 (Effect Quality)：**
    *   **說明：** 技能、爆炸等視覺效果的複雜度。在《火炬之光：無限》中，這對性能影響巨大，尤其是在技能滿天飛的戰鬥中。
    *   **推薦：** **低/中**。除非你的硬體非常強大，否則調低此選項能顯著改善卡頓。
*   **抗鋸齒 (Anti-aliasing)：**
    *   **說明：** 平滑物體邊緣的鋸齒感。常用的有FXAA、TAA等。
    *   **推薦：**
        *   **追求性能：** 關閉。
        *   **中高階顯卡：** FXAA或TAA。
*   **後處理 (Post-processing)：**
    *   **說明：** 景深、光暈、環境光遮蔽等後期渲染效果。
    *   **推薦：** **低**或關閉。這些效果通常會消耗較多性能。
*   **其他視覺選項 (Other Visuals)：**
    *   如**環境光遮蔽 (Ambient Occlusion)**、**反射 (Reflections)**等，通常建議調低或關閉以提升性能。

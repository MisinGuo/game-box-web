*   **核心理念**：吸收傷害、保護隊友、製造控場機會。
*   **優勢**：
    *   PVE副本開荒、高難度挑戰的首選，能有效吸引並承受BOSS傷害。

*   **2025年趨勢**：隨著怪物傷害與機制越來越複雜，純坦流的戰略價值不減反增，特別是在需要穩定控制與減傷的場合。

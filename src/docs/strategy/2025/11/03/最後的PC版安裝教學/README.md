*   **閱讀並接受許可協議**：仔細閱讀使用者授權協議 (EULA)，並勾選「我接受」以繼續。

#### 步驟 3：選擇安裝路徑與選項

*   **選擇安裝目錄**：
    *   **建議**：將遊戲安裝在非系統碟 (例如 D: 或 E: 槽) 的獨立資料夾中，這有助於系統碟的效能並方便管理。

    *   **創建新資料夾**：例如 `D:\Games\最後的PC版`。
*   **選擇組件 (若有)**：某些遊戲會提供額外組件或語言包的選擇，根據您的需求勾選。
*   **創建桌面捷徑 (建議)**：勾選「創建桌面捷徑」以便快速啟動遊戲。

#### 步驟 4：執行安裝

*   **開始安裝**：確認所有設定無誤後，點擊「安裝」或「下一步」。
*   **耐心等待**：安裝過程可能需要數分鐘到數小時不等，具體取決於遊戲大小和您的硬碟速度。請勿中斷此過程。
*   **安裝完成**：當進度條達到100%時，會彈出安裝完成的提示視窗。點擊「完成」或「退出」。

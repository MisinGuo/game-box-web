遊戲主要分為以下幾種卡池：

*   **新手卡池 (Newbie Banner):** 通常提供較優惠的首次十連抽保底SSR，是初期快速累積戰力的重要途徑。
*   **常駐卡池 (Standard Banner):** 包含所有非限定的常駐角色與裝備，機率相對分散。通常設有較高的保底計數。
*   **限定UP卡池 (Limited UP Banner):** 這是玩家最關注的卡池類型，會特定提升某個新角色或復刻角色的獲取機率。通常會設有獨立的保底計數，且保底出UP角色的機率較高。
*   **武器/裝備卡池 (Weapon/Equipment Banner):** 專門用於抽取強力武器或裝備，與角色卡池分開計算。

所有卡池都設有「保底機制」。一般而言，在達到一定抽數後，必定會獲得最高稀有度的角色或物品（例如SSR）。限定UP池通常還有「大保底」與「小保底」之分：小保底時有50%機率抽到當期UP角色，若未抽到，下次保底則必定為當期UP角色，這就是所謂的「大保底」。玩家務必密切關注遊戲內公告，確認當前版本的具體保底抽數與規則。

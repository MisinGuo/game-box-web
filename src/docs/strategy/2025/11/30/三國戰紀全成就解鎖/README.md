#### 2.1 劇情通關與難度挑戰成就

這類成就是最基礎的，通常要求玩家在不同難度下通關遊戲。

*   **成就範例：**
    *   「初出茅廬」：首次通關遊戲。
    *   「身經百戰」：以任一角色通關遊戲五次。
    *   「亂世梟雄」：在「困難」模式下通關遊戲。
    *   「天下歸心」：在「噩夢」模式下通關遊戲。

*   **解鎖策略：**
    1.  **熟悉關卡流程：** 首先選擇你最擅長的角色（如諸葛亮、馬超），熟悉每個關卡的敵人配置、寶箱位置和BOSS戰術。
    2.  **善用道具：** 尤其是「天書」類道具（火書、冰書、電書）和「隱身衣」、「黃金甲」等，在關鍵時刻能扭轉戰局。
    3.  **聯機合作：** 在高難度模式下，與信賴的隊友合作能大大降低難度。分配好角色，例如一人主攻，一人輔助控場。
    4.  **練習BOSS戰：** 每個BOSS都有其弱點和攻擊模式，多加練習，掌握無傷打法是高難度通關的必備技能。例如，許褚的衝撞可利用走位引導，呂布的飛戟可防禦或閃避。

#### 2.2 角色專精與技能運用成就

這些成就考驗玩家對特定角色的操作技巧和技能理解。

*   **成就範例：**
    *   「臥龍出山」：使用諸葛亮通關遊戲。
    *   「虎將無雙」：使用馬超在單一關卡中擊敗200名敵人。
    *   「連招達人」：使用黃忠在一次連擊中造成超過30次打擊。
    *   「巾幗英雄」：使用黃忠或馬超以外的女性角色通關遊戲。

*   **解鎖策略：**
    1.  **選擇主攻角色：** 專注於1-2個角色進行練習，深入了解其普通攻擊、必殺技、超必殺技的攻擊範圍、傷害和破綻。
    2.  **練習連段：** 掌握角色的基礎連段，並嘗試將必殺技融入連段中，以達到更高的傷害和連擊數。例如，諸葛亮的「萬箭齊發」接普通攻擊，或馬超的衝刺攻擊後接「野馬奔騰」。
    3.  **特定關卡刷怪：** 對於擊殺數成就，選擇敵人較多、刷新頻率較高的關卡進行刷怪。例如，第一關或第二關的初期。
    4.  **利用地形：** 某些角色的技能在特定地形下效果更佳，如將敵人逼至牆角進行無限連擊。

#### 2.3 道具收集與特殊擊殺成就

這類成就要求玩家細心探索，並在特定情況下完成擊殺。

*   **成就範例：**
    *   「寶藏獵人」：收集所有隱藏寶箱中的道具。
    *   「五行大師」：在單一遊戲中，使用過所有五種天書（火、冰、電、爆、隱）。
    *   「一擊必殺」：在不受到任何傷害的情況下擊敗一個BOSS。
    *   「斬草除根」：擊敗所有關卡中的小BOSS（如貂蟬、許褚等）。

*   **解鎖策略：**
    1.  **背板與地圖：** 熟悉每個關卡的隱藏通道和寶箱位置。許多寶箱需要用特定攻擊（如滑鏟、重擊）才能打開。
    2.  **規劃道具路線：** 在遊戲初期就規劃好道具收集路線，確保能拿到所有天書。有些天書是隨機掉落，有些則固定在寶箱中。
    3.  **BOSS無傷技巧：** 這需要大量的練習。觀察BOSS的攻擊前搖，利用防禦、閃避或無敵技能（如某些角色的超必殺技）來規避傷害。例如，對付夏侯淵，抓住他射箭後的硬直進行輸出。
    4.  **小BOSS不漏殺：** 確保每個關卡的小BOSS都被擊敗。某些小BOSS可能在分支路線上，需要選擇正確的道路。

#### 2.4 隱藏與特殊挑戰成就

這些成就是遊戲中最具挑戰性，也最能體現玩家實力的部分。

*   **成就範例：**
    *   「真三國無雙」：在不使用任何道具的情況下通關遊戲。
    *   「極限生存」：單人模式下，僅使用一條命通關遊戲。
    *   「時間競速者」：在指定時間內（如30分鐘）通關遊戲。
    *   「完美主義者」：在單一關卡中，達到最高分數並擊敗所有敵人。

*   **解鎖策略：**
    1.  **無道具通關：** 這要求玩家對角色操作、敵人配置和BOSS戰術有極致的理解。建議選擇攻擊力高、技能範圍廣的角色，如諸葛亮或張飛。
    2.  **一命通關：** 這是對玩家綜合實力的終極考驗。除了上述技巧外，更需要冷靜的心態和對失誤的零容忍。建議從簡單模式開始練習，逐步提升難度。
    3.  **競速通關：** 熟悉最短路線，跳過不必要的戰鬥，只擊敗關鍵敵人。選擇移動速度快、清怪效率高的角色。
    4.  **高分與完美擊殺：** 盡量使用連擊來擊敗敵人，收集所有寶箱和金幣。在擊敗BOSS時，嘗試觸發高分擊殺條件（如特定連段終結）。

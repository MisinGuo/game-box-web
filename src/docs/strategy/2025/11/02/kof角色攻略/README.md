*   **近距離角色**：善用快速突進技或跳入攻擊，迅速拉近距離展開壓制。

#### 3. 防禦與壓制：攻防一體

《KOF》的防禦系統強調「普防」與「緊急迴避」（Roll）。學會如何安全地防禦對手攻擊，並在適當時機進行反擊，是生存與反攻的基礎。

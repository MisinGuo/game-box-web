要理解連段，首先要熟悉一些核心術語：

*   **目押（Link）**：指在一個攻擊動作結束後，精準地在極短的「硬直恢復」時間內輸入下一個攻擊，形成無法防禦的連續技。這通常需要較高的輸入精度和節奏感。
*   **取消（Cancel）**：指在某個招式尚未完全結束其動畫時，立即輸入特定指令，中斷當前招式並施展出另一個招式。KOF中常見的有普通技取消特殊技、特殊技取消超必殺技等。

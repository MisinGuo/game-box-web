    *   **辅助/控场**：逐光者 (硬控) / 任何具备冻结或眩晕技能的英雄
    *   **替补**：艾特兰 (破防/单体输出) / 任何具备高治疗量的辅助英雄

    *   **卡尔加** (Kharall the Fierce)：减防、弱化、AOE输出，非常全面。

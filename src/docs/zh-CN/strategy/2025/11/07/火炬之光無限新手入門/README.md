### 装备获取与词缀理解：如何挑选第一套神装

装备是提升战力的关键。初期你无需追求极品，但要学会辨别装备的价值。

*   **词缀：** 装备上的附加属性。分为前缀和后缀。
    *   **攻击词缀：** 增加伤害、暴击率、暴击伤害、攻击速度等。
    *   **防御词缀：** 增加生命值、护甲、抗性（火焰、冰霜、闪电、混沌）、格挡、闪避等。
    *   **通用词缀：** 增加移动速度、技能等级等。
*   **新手挑选原则：**
    1.  **優先補足抗性：** 確保所有元素抗性（火、冰、電）至少達到75%上限，混沌抗性也盡量提升。這是生存的基石。
    2.  **堆叠生命值：** 生命值越高，生存能力越强。
    3.  **提升主要属性：** 根据你的职业和技能，提升力量、敏捷或智力。
    4.  **增加核心伤害：** 确保你的武器伤害足够，并寻找增加你主要伤害类型（物理、火焰、冰霜等）的词缀。

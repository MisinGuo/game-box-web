### 什么是新世界服羁绊效果？

羁绊效果是指特定英雄之间，或特定阵营、类型英雄组合后，能触发的额外能力加成。这些加成可以是攻击力、防御力、生命值提升，也可以是特殊技能效果强化、怒气回复加速，甚至影响敌方状态等。在2025年最新版本中，部分羁绊效果加入了「阶段性强化」机制，意味着随着羁绊英雄的星级、专武等级提升，羁绊效果也会同步增强。

### 羁绊触发条件与类型


*   **指定英雄羁绊：** 最常见的类型，需要特定两至三位英雄同时上阵才能触发。例如，刘备、关羽、张飞的「桃园结义」。
*   **阵营羁绊：** 队伍中拥有一定数量的同阵营英雄（魏、蜀、吴、群），即可触发全队属性加成。新世界服尤其强化了阵营混搭后的「协同羁绊」，鼓励玩家打破单一阵营的限制。
*   **职业羁绊：** 队伍中拥有一定数量的同职业英雄（如突击、防御、辅助、治疗），也能触发额外效果。
*   **专武羁绊：** 部分英雄的专属武器达到特定等级后，会与其他英雄产生新的羁绊效果，这是2025年版本中一个重要的战力增长点。

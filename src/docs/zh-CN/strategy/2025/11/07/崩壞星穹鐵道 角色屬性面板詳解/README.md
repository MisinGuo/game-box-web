### 1.1 基础属性：构筑角色肉体与力量

![游戏场景](https://oss.2025.11.5yxy5.com/images/2025/11/07/%E5%B4%A9%E5%A3%9E%E6%98%9F%E7%A9%B9%E9%90%B5%E9%81%93%20%E8%A7%92%E8%89%B2%E5%B1%AC%E6%80%A7%E9%9D%A2%E6%9D%BF%E8%A9%B3%E8%A7%A3/image_2_1762466268.jpg)

这些是角色最核心的数值，直接影响其生存能力与输出潜力。

*   **生命值 (HP)：** 衡量角色承受伤害的能力。 HP越高，角色在战斗中越不容易被击倒。对于存护（坦克）和丰饶（治疗）角色而言，高生命值是其站场的根本。
*   **攻击力 (ATK)：** 影响角色造成伤害的基础数值。攻击力越高，普通攻击、战技和终结技造成的伤害越高。巡猎、毁灭、智识等输出型命途的角色，对攻击力有着极高的需求。
*   **防御力 (DEF)：** 降低角色受到的物理与属性伤害。防御力越高，角色受到的伤害越少。存护角色通常拥有较高的防御力，以确保队伍的生存。
*   **速度 (SPD)：** 决定角色在战斗中的行动顺序与频率。速度越快，角色行动条推进越快，能更快地出手，甚至在一个回合内多次行动。在2025年的高速环境下，速度对于所有角色都至关重要，尤其是辅助和部分输出角色。

### 1.2 进阶属性：决定战术深度与爆发潜力


这些属性通常通过遗器、光锥、行迹等方式获得，能极大程度地提升角色的战斗表现。

*   **暴击率 (CRIT Rate)：** 角色造成暴击伤害的机率。暴击伤害通常是普通伤害的1.5倍（基础值），能显著提升输出上限。
*   **暴击伤害 (CRIT DMG)：** 角色造成暴击时，伤害提升的倍率。暴击伤害与暴击率相辅相成，是输出角色追求的核心属性。
*   **击破特攻 (Break Effect)：** 提升角色击破敌人韧性条时造成的伤害，并延长敌人被击破后的行动延迟。对于击破流队伍和部分依赖击破效果的同谐角色（如阮•梅）而言，击破特攻是核心词条。
*   **效果命中 (Effect Hit Rate)：** 提升角色施加负面效果（如冰冻、禁锢、裂伤等）的成功率。虚无命途角色（如银狼、卡芙卡）对此属性需求极高。
*   **效果抵抗 (Effect RES)：** 降低角色受到负面效果影响的机率。对于需要稳定输出的主C或关键辅助，适量的效果抵抗能提升其生存与功能性。
*   **治疗量加成 (Outgoing Healing Boost)：** 提升角色施放治疗技能时的治疗量。丰饶命途角色（如罗刹、藿藿）的核心属性。
*   **能量恢复效率 (Energy Regeneration Rate)：** 提升角色获得能量的速度。能量恢复效率越高，角色越能频繁地施放终结技，对于依赖终结技的角色（无论输出或辅助）都非常重要。

### 1.3 属性伤害加成与抗性：专精与对策


*   **属性伤害加成 (Elemental DMG Boost)：** 例如「虚数伤害提高」、「冰伤害提高」等。直接提升特定属性技能造成的伤害。输出角色应尽可能堆叠与自身属性相符的伤害加成。
*   **属性抗性 (Elemental RES)：** 降低角色受到特定属性伤害的数值。通常在面对特定敌人时进行策略性调整。

*   **副词条 (Substats)：** 遗器的随机加成，是决定遗器品质的关键。

    *   **輔助角色：** 速度、能量恢復效率、效果命中 (虛無)、生命%、防禦%。

光锥不仅提供基础属性，其技能更是对角色属性面板的强力补充。

*   **专属光锥：** 通常能最大化角色的潜力，提供最优的属性加成与技能效果。在2025年的版本中，部分专属光锥的泛用性也得到了提升。

### 2.3 行迹升级：稳步提升角色基础


行跡是角色自帶的技能樹，升級行跡不僅能提升戰技、終結技等技能等級，還能解鎖額外的屬性加成（如生命%、攻擊力%、暴擊率等）。

*   **优先级：** 核心输出技能 > 关键辅助技能 > 属性加成行迹。
*   **2025年建议：** 在资源允许的情况下，尽可能点满关键角色的所有属性加成行迹，这些稳定提升是战力不可或缺的一部分。

### 2.4 队伍配置与属性协同：1 1>2的法则


单个角色的属性再强，也需要与队伍协同才能发挥最大效用。

*   **速度轴：** 根据队伍需求调整角色速度，确保辅助先于输出行动，或特定角色能吃到增益。
*   **属性覆盖：** 确保队伍能有效应对不同弱点属性的敌人，并在击破后能最大化击破伤害。
*   **增益叠加：** 了解不同角色提供的增益（如攻击力提升、暴击率提升、减防等）如何叠加，避免溢出或浪费。

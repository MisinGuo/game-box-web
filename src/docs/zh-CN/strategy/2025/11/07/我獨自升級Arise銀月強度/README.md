*   **普通攻击：** 造成冰属性伤害。
*   **核心技能一 (冰冻之触)：** 向前方释放冰霜，对敌人造成伤害并有机会施加冰冻效果。这是银月主要的群体控制技能。
*   **核心技能二 (冰霜冲击)：** 快速向前突进，对路径上的敌人造成伤害。可用于追击或规避伤害。
*   **终极技能 (冰河时代)：** 召唤巨大的冰晶对范围内所有敌人造成毁灭性的冰属性伤害，并长时间冰冻敌人。是银月最强力的群体控制与爆发技能。
*   **被动技能：** 提升自身冰属性伤害，并在触发冰冻效果时有额外收益（例如减少技能冷却或回复能量）。

#### 2.2 技能升级优先级

在2025年的版本中，银月的技能升级建议优先级如下：

1.  **终极技能 (冰河时代)：** 优先将其升满，大幅提升爆发与控制能力。
2.  **核心技能一 (冰冻之触)：** 提升冰冻机率与伤害，强化群体控制。
3.  **被动技能：** 提升冰属性伤害与触发收益。
4.  **核心技能二 (冰霜冲击)：** 最后升级，主要用于位移和补足伤害。

#### 2.3 独特机制运用

银月的核心机制是「冰冻」状态。最大化冰冻的覆盖率和效果是关键。
*   **冰冻连携：** 在敌人被「冰冻之触」控制后，立即衔接其他输出技能，确保伤害命中。
*   **终极技能的战略性使用：** 「冰河时代」的冰冻范围大、时间长，应在敌人聚集或需要爆发输出时施放，为队友创造输出空间。
*   **冰霜冲击的灵活运用：** 不仅是位移，也可以用来调整位置，确保「冰冻之触」能命中更多目标。

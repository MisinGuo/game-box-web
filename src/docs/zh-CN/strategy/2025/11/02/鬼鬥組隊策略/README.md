一支成功的鬼斗队伍，必须包含明确的职责分工。 2025年版本，以下三大核心定位依然是组队的基石：

### 1. 核心输出（Damage Dealer, DD）


负责造成主要伤害，快速击败敌人。他们通常拥有高爆发或持续输出能力。

*   **特点：** 技能伤害高、攻击频率快、对装备要求较高。
*   **范例（假设角色）：**
    *   **「影刃」夜叉：** 高机动性近战刺客，擅长单点爆发，切入敌方后排。
    *   **「幽火」狐仙：** 远程法术输出，拥有范围伤害与控制能力，适合清场。

### 2. 生存辅助（Support/Tank, SUP/T）


提供保护、治疗、增益或控制效果，确保队伍的生存与输出环境。

*   **特点：** 血量防御高、拥有护盾/治疗/控制技能、能吸引敌方火力。
*   **范例（假设角色）：**
    *   **「铁壁」石巨人：** 拥有强大的减伤与嘲讽技能，是队伍最坚实的屏障。
    *   **「圣光」灵鸟：** 提供范围治疗与增益效果，能显著提升队伍续航力。

### 3. 功能型角色（Utility/Off-Tank, UT）


介于输出与辅助之间，能提供次要伤害、控制、破甲或战场分割等效果，弥补队伍短板。

*   **特点：** 技能多样化、能适应不同战局、通常具有一定的生存能力。
*   **范例（假设角色）：**
    *   **「锁链」怨灵：** 拥有强力的群体控制与位移技能，能打乱敌方阵型。
    *   **「毒心」蜘蛛精：** 持续性毒素伤害与减益效果，能削弱敌方整体战力。

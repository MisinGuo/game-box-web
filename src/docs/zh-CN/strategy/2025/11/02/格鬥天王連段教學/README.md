连段是指在格斗游戏中，角色在短时间内连续施展多个攻击，使对手无法防御或反击的一系列招式组合。在KOF中，连段不仅能造成可观的伤害，更是累积能量、创造压制机会、甚至完成逆转的关键。掌握连段，意味着你掌握了战斗的主动权。

### 基本操作术语解析


要理解连段，首先要熟悉一些核心术语：

*   **目押（Link）**：指在一个攻击动作结束后，精准地在极短的「硬直恢复」时间内输入下一个攻击，形成无法防御的连续技。这通常需要较高的输入精度和节奏感。
*   **取消（Cancel）**：指在某个招式尚未完全结束其动画时，立即输入特定指令，中断当前招式并施展出另一个招式。 KOF中常见的有普通技取消特殊技、特殊技取消超必杀技等。
*   **派生（Follow-up）**：某些特殊技或必杀技在输入后，可以根据不同的按键组合或时机，追加额外的攻击动作。
*   **Super Cancel (SC)**：在特定必杀技击中对手时，消耗能量槽将其取消为超必杀技。
*   **Max Cancel (MC)**：在特定超必杀技击中对手时，消耗能量槽将其取消为Climax Super Special Move (C.S.S.M.)。
*   **Quick Max (QM) / Max Mode (MM)**：进入Max Mode后，角色的攻击力提升，并且可以在特定招式之间进行多种取消，大幅拓展连段潜力。 2025年版本中，Max Mode的启动与应用策略仍是高端对局的核心。

### 目押连段的精髓与练习方法


目押是KOF连段的核心挑战之一。其精髓在于「节奏感」而非「速度」。

*   **视觉判断**：观察角色攻击动画的「收招」瞬间，这是输入下一个指令的最佳时机。
*   **听觉辅助**：利用攻击命中音效或角色的语音提示来辅助判断节奏。
*   **分段练习**：将复杂的目押连段拆解成2-3个小段落，每个段落单独练习直到熟练，再逐步组合。

取消技是KOF连段多样性的来源。

*   **Max Mode连段**：进入Max Mode后，许多普通技和特殊技之间的取消限制会放宽，允许更长的连段。掌握Max Mode下的「快速取消」和「多段取消」是高端玩家的标志。

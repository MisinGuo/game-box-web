### 3. 基础操作概览


《拳皇》系列的操作核心是方向键（或摇杆）与四个攻击按钮：轻拳（LP）、重拳（HP）、轻脚（LK）、重脚（HK）。

*   **方向键/摇杆：** 控制角色移动、跳跃、下蹲、防御。
    *   `←`：后退
    *   `→`：前进
    *   `↑`：跳跃
    *   `↓`：下蹲
    *   `↙`：下蹲防御
    *   `←`（长按）：站立防御
*   **攻击按钮：**
    *   **LP/LK：** 速度快，伤害低，适合牵制、确认连段起手。
    *   **HP/HK：** 速度慢，伤害高，常作为连段中段或收尾。
    *   **特殊操作：** KOF还有一些特殊操作，例如：
        *   `→   HK`：指令投（部分角色）
        *   `→   HP`：特殊攻击（部分角色）
        *   `LP   LK`：闪避（Dodge/Roll）
        *   `HP   HK`：吹飞攻击（Blowback Attack），可击飞对手，常用于连段收尾或解围。

### 1. 移动与防御：格斗游戏的生命线


*   **移动：** 除了前后移动，还有「冲刺」（`→→`）和「后跳」（`←←`）。冲刺能快速接近对手或拉近距离，后跳则能迅速脱离危险。
*   **跳跃：** KOF有三种跳跃：
    *   **小跳（Short Jump）：** 轻点`↑`、`↖`或`↗`，跳得低且快，是KOF进攻的核心。
    *   **中跳（Middle Jump）：** 轻点`↓`，然后快速点`↑`、`↖`或`↗`。
    *   **大跳（High Jump）：** 长按`↑`、`↖`或`↗`，跳得高且慢。
    *   **超大跳（Super High Jump）：** 长按`↓`，然后长按`↑`、`↖`或`↗`。
*   **防御：**
    *   **站立防御（Standing Guard）：** 按住`←`，防御站立攻击和跳跃攻击。
    *   **下蹲防御（Crouching Guard）：** 按住`↙`，防御下段攻击和站立攻击。
    *   **重点：** 必须根据对手的攻击方向选择正确的防御姿势。

### 2. 普通攻击与特殊技：连段的基石


*   **普通攻击：** LP、LK、HP、HK。它们是连段的起手和填充。
*   **特殊技（Command Normals）：** 结合方向键与攻击键的普通攻击，例如草薙京的「`→ LK`」。这些招式通常有特殊属性，如突进、多段攻击等。
*   **必杀技（Special Moves）：** 搓招输入（如波动拳的`↓↘→   P`）发动的招式。这是《拳皇》战斗的精髓，掌握角色的必杀技是入门的关键。训练模式中会显示所有必杀技指令。

### 3. 超必杀技与MAX Mode：逆转战局的关键


*   **超必杀技（Super Special Moves）：** 比必杀技更强大，需要消耗能量槽（Power Gauge）。通常指令更复杂，如两次波动拳输入 (`↓↘→↓↘→   P`)。
*   **MAX Mode：** 《KOF XV》的核心系统之一。消耗一条能量槽进入强化状态，期间可以发动EX必杀技（强化版必杀技），并能使用「MAX超必杀技」和「Climax超必杀技」。 MAX Mode是将伤害最大化和逆转战局的重要手段。
*   **Climax超必杀技：** 消耗三条能量槽，是角色最强大的招式，通常伴随华丽的动画。

### 4. 连段基础：从简单到复杂


*   **自动连段（Rush Combo）：** 《KOF XV》为新手设计的系统。连续按轻拳（LP）即可打出一套简单连段，最后可接必杀技或超必杀技。这是体验连段乐趣的最快方式。
*   **基础连段：** 学习每个角色的「轻攻击起手接必杀技」是最基本的连段。例如：`LP > LK > 必杀技`。
*   **连段扩展：** 熟悉后，可以尝试加入更多普通攻击、特殊技，并在MAX Mode下使用EX必杀技和超必杀技来延伸连段。

### 5. 队伍系统：3对3的策略


KOF是3对3的团队战。你需要选择三名角色组成队伍。角色的出场顺序也很重要，通常会将最熟悉或最强势的角色放在最后（Anchor），因为他将有更多的能量槽可以使用。考虑角色间的互补性（例如，一个擅长压制，一个擅长远程，一个擅长爆发）。

### 什么是潜能？主潜能与附加潜能

![游戏场景](https://oss.2025.11.5yxy5.com/images/2025/11/02/%E6%A5%93%E4%B9%8B%E8%B0%B7%E8%A3%9D%E5%82%99%E6%BD%9B%E8%83%BD/image_2_1762018578.jpg)

每件具有潜能的装备都包含两种类型的潜能：

*   **主潜能 (Main Potential)：** 这是装备的主要潜能，通常会提供大幅度的能力加成。主潜能的稀有度分为「稀有」、「罕见」、「史诗」和「传说」四个等级，等级越高，潜能数值越强，可拥有的潜能行数也越多（史诗以上为三行）。
*   **附加潜能 (Additional Potential)：** 简称「附加」，是主潜能之外的额外潜能，通常提供较小的能力加成，但累积起来也能带来显著提升。附加潜能的稀有度同样分为「稀有」、「罕见」、「史诗」和「传说」，且通常为三行。

### 潜能等级与稀有度：战力提升的阶梯


潜能的稀有度直接影响其提供的数值范围与潜能种类。在2025年的枫之谷环境中，传说等级的主潜能和附加潜能是追求极致战力的最终目标。

*   **稀有 (Rare)：** 最基础的潜能，通常提供单一属性或少量百分比。
*   **罕见 (Epic)：** 比稀有更强，开始出现一些实用的百分比潜能。
*   **史詩 (Unique)：** 進入中後期玩家的門檻，能洗出不錯的百分比潛能，如攻擊力/魔力%數。
*   **傳說 (Legendary)：** 頂級潛能，能洗出最高百分比的潛能，如21%攻擊力/魔力、40%無視防禦、20%爆擊傷害等，是PVE與PVP的最終追求。

*   **传说方块：** 有机率将装备潜能等级提升至传说，并重置潜能数值。这是将史诗装备提升至传说的主要方块。

### 附加潜能方块


*   **附加方块：** 将装备附加潜能等级提升至罕见，并重置附加潜能数值。
*   **闪耀附加方块：** 有较高机率将装备附加潜能等级提升至传说，并重置附加潜能数值。

### 潜能锁与潜能保护卷轴


*   **潜能锁：** 在使用方块时，可以锁定一条或两条满意的潜能，使其不被重置。但锁定潜能会增加方块消耗量。

觉醒系统主要围绕以下几个核心要素：

### 1. 灵魂精华 (Soul Essence)

这是进行觉醒的主要材料。灵魂精华分为不同稀有度（普通、稀有、史诗、传奇），对应不同稀有度的英雄觉醒。它们主要透过「灵魂石地下城」（Soulstone Dungeons）获得，该地下城会定期开放，并且有不同的难度等级。

### 2. 灵魂币 (Soul Coins)

灵魂币是觉醒所需的另一种货币，主要用于购买或升级灵魂特质。你可以透过出售多余的灵魂（例如，你已经拥有该英雄的觉醒，或是不需要的英雄灵魂）来获得灵魂币。

### 3. 灵魂 (Souls)

每个英雄都有其对应的「灵魂」。当你从灵魂石地下城或特定活动中获得英雄灵魂时，你可以将其应用到对应的英雄上，以提升其觉醒等级。灵魂也分为不同稀有度，并有1至6星的等级。星级越高，提供的属性加成和特质越强大。

这是觉醒系统的精髓。每个英雄在觉醒后会获得一个随机的灵魂特质。这些特质种类繁多，从单纯的属性增强到改变技能机制，应有尽有。你可以透过消耗灵魂币来重新抽取特质，直到获得最适合该英雄的特质。 2025年的趋势是，玩家会更倾向于为核心英雄抽取「神级」（Divine）或「不朽」（Immortal）等级的强力特质。

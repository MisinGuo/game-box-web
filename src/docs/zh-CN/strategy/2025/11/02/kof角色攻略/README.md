### 一、KOF核心战术基础：角色攻略前的通用法则

在深入探讨特定角色之前，理解《KOF》系列共通的基础战术是所有玩家的必修课。这些原则适用于几乎所有角色，并且是构筑高阶战术的基石。

#### 1. 能量槽（Meter）管理：战局逆转的关键

能量槽是《KOF》战斗的核心资源，合理分配能量用于EX技能、超必杀技（Super Special Move）、CLIMAX超必杀技，或是防御性技巧如防御反击（Guard Cancel）、破坏攻击（Shatter Strike）至关重要。
*   **攻击性运用**：在连段中加入EX技能可增加伤害或延长连段，超必杀技则是终结对手的利器。
*   **防御性运用**：防御反击能在防御硬直中迅速反击，破坏攻击则能有效瓦解对手攻势并创造反击机会。

#### 2. 走位与间距（Spacing）：格斗游戏的灵魂

精准的走位与间距控制能让您有效利用角色攻击范围，避免被对手压制，并为自身创造进攻机会。
*   **远距离角色**：利用长手长脚的普通技或飞行道具牵制对手，保持安全距离。
*   **近距离角色**：善用快速突进技或跳入攻击，迅速拉近距离展开压制。

#### 3. 防御与压制：攻防一体

《KOF》的防御系统强调「普防」与「紧急回避」（Roll）。学会如何安全地防御对手攻击，并在适当时机进行反击，是生存与反攻的基础。
*   **紧急回避**：向前后滚动，可躲避攻击并改变位置，但有破绽。
*   **破防与择（Mix-up）**：利用上段、下段、投掷攻击，或跳入攻击制造择（猜拳）局面，迫使对手犯错。

### 1. 变异的发生机率


每次交配，子代有2.5%的機率發生變異。如果發生變異，子代將在父母的基礎屬性上額外獲得2點隨機屬性，並有50%的機率伴隨顏色變異。

### 2. 变异点数与属性继承


*   **基础属性继承：** 子代会随机继承父母中较高的单项属性点数。例如，父亲近战伤害有40点，母亲有30点，子代会继承40点。
*   ** 变异点数： ** 每个变异都会赋予2点属性，这2点会随机分配给一种七种变异属性（生命、耐力、氧气、食物、体重、近战、移动速度）。这意味着您可能需要多个变体才能叠加到您想要的特定属性。
*   **移动速度例外：** 虽然移动速度可以获得变异点数，但除非特殊设定，否则玩家无法手动加点，且恐龙的基础移动速度通常无法提升。因此，移动速度变异通常被视为「废变异」。

### 3. 恐龙的变异计数器（Mutation Counter）


这是2025年高效变异策略的关键。每只恐龙都有两个变异计数器：
*   **母系变异计数器 (Maternal Mutations)**
*   **父系变异计数器 (Paternal Mutations)**

当子代发生变异时，其变异计数器会增加2点。同时，如果父母的任何一方的总变异计数器（母系   父系）达到或超过20，那么子代从该方获得变异的机率将大幅降低。这就是著名的「20/20规则」。

**核心理解：**
*   如果父母双方的总变异计数器都达到20或更高，子代将无法再获得新的变异。
*   如果只有一方的总变异计数器低于20，子代仍有机会从该方获得变异。

这意味着我们需要利用「干净」的恐龙（变异计数器为0/0）来进行变异叠加，以确保变异机率最大化。

铭刻系统主要围绕以下几个核心要素展开：

### 1. 铭刻部位与解锁条件


铭刻功能通常在角色达到一定等级后开放，并分布于多个装备部位，如武器、头盔、衣服、裤子、鞋子、项链、戒指等。每个部位可铭刻的属性类型可能有所不同，玩家需根据自身职业和战斗需求进行选择。

### 2. 铭刻石的种类与品质


铭刻石是铭刻系统的核心。它们分为不同种类，提供不同的属性加成，例如：
*   **攻击型铭刻石：** 物理攻击、法术攻击、暴击率、暴击伤害。

*   **功能型铭刻石：** 穿透、命中、闪避、韧性等。

铭刻石也存在品质差异，通常以颜色区分（如绿、蓝、紫、橙），品质越高，提供的属性加成越强。 2025年版本中，高阶铭刻石的获取难度有所提升，但其带来的战力增幅也更为显著。

### 3. 铭刻与升级


*   **铭刻：** 获得铭刻石后，可在指定介面将其铭刻到对应的装备部位上。铭刻成功后，装备将获得铭刻石提供的属性。
*   **升级：** 铭刻石可以通过消耗同种类或通用铭刻材料进行升级。升级会大幅提升铭刻石的属性加成。随着等级的提升，所需的材料和金币消耗也会增加。

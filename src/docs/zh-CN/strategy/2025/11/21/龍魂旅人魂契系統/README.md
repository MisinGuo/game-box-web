### 什麼是魂契？

![游戏场景](https://oss.2025.11.5yxy5.com/images/2025/11/21/%E9%BE%8D%E9%AD%82%E6%97%85%E4%BA%BA%E9%AD%82%E5%A5%91%E7%B3%BB%E7%B5%B1/image_2_1763668293.jpg)

魂契是寄宿著強大龍魂力量的特殊道具，每個魂契都帶有特定的屬性加成（如攻擊力、生命值、防禦、暴擊率等）和一個主動或被動技能。魂契有不同的稀有度（例如：普通、稀有、史詩、傳說），稀有度越高，其基礎屬性與技能效果通常越強大。

### 魂契的稀有度與品質


魂契的稀有度直接影響其潛力。除了稀有度，魂契還有「品質」之分，即使是相同稀有度的魂契，其初始屬性與成長上限也可能因品質不同而有差異。在2025年版本中，傳說級魂契依然是所有玩家追求的終極目標，而史詩級魂契則是過渡期或特定搭配的優秀選擇。

### 魂契技能與羈絆效果


*   **魂契技能：** 每個魂契的核心價值所在。這些技能涵蓋了輸出、防禦、輔助、控制等多種效果。例如，某些魂契能提升普攻傷害，有些則能在受傷時觸發護盾，或是為隊友提供增益。
*   **羈絆效果：** 當你為角色裝備特定組合的魂契時，會觸發額外的羈絆效果。這些效果通常比單一魂契技能更為強大，是構築完整戰術體系的關鍵。2025年版本中，官方持續新增與調整羈絆組合，密切關注遊戲內公告至關重要。

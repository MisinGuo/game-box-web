*   **星星数：** 击败对手防守队伍可获得星星。第一次击败获得2颗星，第二次击败获得1颗星。

*   **目标选择：** 选择攻击血量较高的对手会获得更多基础分数。合理分配成员攻击目标，是公会战术的核心一环。

3.  **稳定性与爆发力兼顾：** 进攻队伍应在保证一定稳定性的前提下，尽可能追求高效的爆发力，以减少战斗回合，降低翻车风险。

*   **一波流（Cleave）战术：高爆发秒杀：** 透过速度优势或无敌/免疫保护，让核心输出魔灵在第一时间打出爆炸伤害，迅速击溃对手。

    *   **常见组合：** 风德鲁伊、火吸血鬼、水仙人、暗狼人等搭配高耐久符文。
    *   **战术核心：** 队伍整体生存能力强，能应对对手的爆发，并在劣势中逐渐扳回。
*   **拆队（Specific Counter）战术：针对性击破：** 根据对手防守队的特定核心魔灵或机制，选择专门的反制魔灵。
    *   **常见组合：** 对付高血量魔灵使用无视防御（如风丑、水伊芙），对付反击魔灵使用多段攻击或无法反击的技能，对付复活魔灵使用禁止复活（如火吸血鬼）。
    *   **战术核心：** 精准打击对手弱点，以最小代价换取最大优势。

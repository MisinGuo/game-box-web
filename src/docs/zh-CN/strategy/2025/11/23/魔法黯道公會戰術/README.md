英雄初始只有四个战斗技能，但每个职业都有七个技能可供选择。透过公会，你可以：

**2025年最新策略洞察：** 在游戏初期，优先升级核心输出技能和关键的控制/治疗技能至2-3级，以应对早期地城的挑战。随着游戏进展，再逐步提升所有常用技能至满级，确保英雄在任何情况下都能发挥最大效能。

#### 2. 技能选择策略：核心技能与辅助技能

每个英雄职业都有其独特的技能组，但并非所有技能都适用于所有情况。
*   **核心战斗技能：** 这些是英雄在战斗中频繁使用的主要输出、治疗或控制技能。例如，强盗（Highwayman）的「双手射击」（Point Blank Shot）和「斩击」（Duelist's Advance）。

**E-E-A-T实践：** 选择技能时，需根据英雄在团队中的「角色定位」进行。一个前排输出英雄应优先选择高伤害技能，而辅助型英雄则应侧重于控制、治疗或增益/减益。

除了战斗技能，每个英雄还有四个营地技能。这些技能在营地休息时使用，用于治疗压力、解除疾病、提升士气或为下一场战斗做准备。

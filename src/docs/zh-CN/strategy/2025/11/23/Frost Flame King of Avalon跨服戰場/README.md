*   **兵种搭配与数量考量：**
    *   **多兵种混搭：** 避免单一兵种，建议步兵（前排抗伤）、弓兵（后排输出）、骑兵（快速突进）合理搭配。

    *   **防守反击：** 若处于劣势，可采取诱敌深入、集中兵力反击的策略。
    *   **撤退与重整：** 当兵力损失过大时，果断撤退重整，避免无谓的牺牲。

    *   **英雄走位：** 输出英雄保持安全距离，控制英雄伺机切入，治疗英雄紧随主力。

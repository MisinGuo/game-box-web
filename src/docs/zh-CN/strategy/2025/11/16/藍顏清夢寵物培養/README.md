每隻寵物都有其獨特的屬性（如生命、攻擊、防禦、速度）和資質。資質決定了寵物每級成長的屬性加成，是判斷寵物潛力的關鍵。在初期，應優先選擇：

寵物裝備能提供額外屬性加成。

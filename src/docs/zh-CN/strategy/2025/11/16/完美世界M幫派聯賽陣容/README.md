*   **定位：** 核心前排，團隊的肉盾與火力吸引者。
*   **優勢：** 變身後高生命、高防禦，群體嘲諷技能能有效保護隊友。
*   **關鍵：** 掌握嘲諷時機，合理開啟減傷技能，為隊友爭取輸出時間。魂石與精煉優先選擇生命、防禦、韌性。

### 2. 武俠 (副坦/輸出/控制)


*   **定位：** 多功能戰士，可作為副坦、近戰輸出或強力控制。
*   **優勢：** 技能多樣，控制能力強，爆發不俗，兼具一定的生存能力。
*   **關鍵：** 根據隊伍需求調整技能流派。作為副坦時，注意切入時機與控制鏈；作為輸出時，找準目標爆發。

### 3. 法師 (核心法術輸出/控制)


*   **定位：** 範圍法術輸出核心，具備強大爆發和控制。
*   **優勢：** 範圍傷害高，單體爆發強，冰凍、眩暈等控制技能實用。
*   **關鍵：** 保護自身安全，在安全位置最大化輸出，利用控制技能打斷敵方關鍵技能或保護自己。魂石與精煉優先選擇法術攻擊、穿透、暴擊。

### 4. 刺客 (單體物理輸出/收割)


*   **定位：** 針對敵方後排的物理爆發輸出，擅長收割殘血。
*   **優勢：** 隱身、高爆發、位移技能靈活，能快速切入敵方後排。
*   **關鍵：** 判斷最佳切入時機，精準鎖定敵方治療或脆皮輸出，一套技能帶走。避免被集火。魂石與精煉優先選擇物理攻擊、穿透、暴擊。

### 5. 妖精 (輔助/控制/召喚)


*   **定位：** 強力輔助與控制，召喚物能提供額外輸出或干擾。
*   **優勢：** 多種控制技能（變形、定身），驅散能力，召喚物能分散敵方火力。
*   **關鍵：** 靈活運用控制技能，保護隊友，干擾敵方。召喚物可作為探路或阻擋技能。

### 6. 羽靈 (核心治療/輔助)


*   **定位：** 團隊生命線，提供治療、增益與驅散。
*   **優勢：** 強大的單體與群體治療能力，能驅散負面狀態，提供增益。
*   **關鍵：** 保持自身存活，合理分配治療技能，預判敵方爆發並提前釋放減傷或群療。羽靈是敵方優先集火目標，務必注意走位與自保。魂石與精煉優先選擇治療量、生命、免傷、韌性。

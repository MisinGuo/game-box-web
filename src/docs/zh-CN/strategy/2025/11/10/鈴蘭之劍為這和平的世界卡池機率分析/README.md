### 核心卡池类型与保底机制

![游戏场景](https://oss.2025.11.5yxy5.com/images/2025/11/10/%E9%88%B4%E8%98%AD%E4%B9%8B%E5%8A%8D%E7%82%BA%E9%80%99%E5%92%8C%E5%B9%B3%E7%9A%84%E4%B8%96%E7%95%8C%E5%8D%A1%E6%B1%A0%E6%A9%9F%E7%8E%87%E5%88%86%E6%9E%90/image_2_1762712005.jpg)

游戏主要分为以下几种卡池：

*   **新手卡池 (Newbie Banner):** 通常提供较优惠的首次十连抽保底SSR，是初期快速累积战力的重要途径。
*   **常驻卡池 (Standard Banner):** 包含所有非限定的常驻角色与装备，机率相对分散。通常设有较高的保底计数。
*   **限定UP卡池 (Limited UP Banner):** 这是玩家最关注的卡池类型，会特定提升某个新角色或复刻角色的获取机率。通常会设有独立的保底计数，且保底出UP角色的机率较高。
*   **武器/装备卡池 (Weapon/Equipment Banner):** 专门用于抽取强力武器或装备，与角色卡池分开计算。

所有卡池都設有「保底機制」。一般而言，在達到一定抽數後，必定會獲得最高稀有度的角色或物品（例如SSR）。限定UP池通常還有「大保底」與「小保底」之分：小保底時有50%機率抽到當期UP角色，若未抽到，下次保底則必定為當期UP角色，這就是所謂的「大保底」。玩家務必密切關注遊戲內公告，確認當前版本的具體保底抽數與規則。

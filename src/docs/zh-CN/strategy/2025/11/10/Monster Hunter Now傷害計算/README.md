让我们逐一解析这些关键因子：

### 1. 武器基础攻击力 (Weapon Base Attack)


这是武器面板上显示的攻击力数值。不同武器类型、不同稀有度与强化等级，其基础攻击力差异巨大。它是所有伤害计算的起点。

### 2. 猎人额外攻击力 (Hunter Bonus Attack)


这部分主要来自于：
*   **防具技能：** 例如「攻击力UP」、「看破」等直接增加攻击力的技能。
*   **道具：** 如「鬼人药」、「鬼人药G」等消耗品。
*   **特殊效果：** 部分武器或魔物素材装备可能带有的特殊攻击加成。

**【2025年提示】** 随着新魔物与新装备的推出，部分技能的数值或触发条件可能有所调整，务必留意官方公告。

### 3. 锐度调节器


武器的斩味（如绿、蓝、白、紫）不仅影响弹刀机率，更直接影响物理伤害输出。斩味越高，伤害修正系数越大。

**【实战建议】** 尽量保持高斩味，尤其是在狩猎高星魔物时。部分武器技能如「研磨师」能有效维持斩味。

### 4. 动作值 (Motion Value)


这是《魔物猎人》系列的核心概念之一，代表着不同武器招式对总伤害的贡献比例。例如，大剑的蓄力斩动作值远高于普通挥砍。虽然《MHW Now》没有明确显示动作值，但玩家可以通过实战经验判断哪些招式输出效率更高。

**【玩家经验】** 熟悉你的武器所有招式，找出高动作值的输出手段，并在安全的时机使用。

### 5. 会心修正 (Critical Hit Modifier)


当攻击触发「会心」（暴击）时，伤害会额外增加。
*   **会心率：** 由武器自带、防具技能（如「看破」）提供。
*   **会心伤害倍率：** 基础为1.25倍。部分技能（如「超会心」）可以进一步提升此倍率。

**【配装考量】** 高会心率搭配高会心伤害倍率是物理输出的重要策略。

### 6. 属性修正 (Elemental Modifier)


属性伤害是《MHW Now》中不可忽视的一环。
*   **属性攻击力：** 武器自带的属性值，可通过防具技能（如「[属性]攻击强化」）提升。
*   **魔物属性肉质：** 每种魔物对不同属性有不同的吸收率（肉质），弱点属性会造成额外伤害。

**【狩猎策略】** 针对魔物的弱点属性选择武器，是提升伤害最直接有效的方法。

### 7. 部位肉质修正 (Part Hitzone Modifier)


魔物身上不同部位对物理伤害和属性伤害的吸收率不同。攻击弱点部位（通常会跳出黄色伤害数字）会造成显著更高的伤害。

**【核心技巧】** 瞄准魔物的弱点部位进行攻击，是所有猎人必须掌握的基础。

### 8. 技能加成 (Skill Bonuses)


除了上述直接影响攻击力的技能，还有许多技能能间接或直接提升伤害：
*   **弱点特效：** 攻击弱点时提升会心率或攻击力。
*   **连击：** 连续攻击命中时提升攻击力。
*   **锁定：** 方便瞄准弱点。
*   **SP技能强化：** 提升SP技能的伤害。

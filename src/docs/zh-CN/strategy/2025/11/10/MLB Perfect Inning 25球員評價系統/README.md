    *   **选球 (EYE)：** 影响获得保送的机率与对坏球的判断。

*   **投球属性：**
    *   **球速度 (VEL)：** 冲击快球速度。
    *   **控球 (CON)：** 影响投球的精准度，减少保送与失投。

    *   **臂力 (ARM)：** 影响传球的距离与速度。

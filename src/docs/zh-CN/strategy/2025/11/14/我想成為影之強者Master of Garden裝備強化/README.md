#### 装备的分类与稀有度

游戏中的装备主要分为武器、防具、饰品三大类，各自提供不同的属性加成。
*   **武器**：通常提升攻击力、暴击率等输出属性。
*   **防具**：主要增加生命值、防御力等生存属性。
*   **饰品**：提供多样化的特殊属性，如速度、效果命中或抵抗等。

每件装备都有其稀有度（如R、SR、SSR、UR等），稀有度越高，基础属性越强，潜在强化上限也越高，是决定其长期投资价值的首要考量。在2025年的版本中，游戏可能会持续推出更高稀有度的装备，务必保持关注。

#### 强化系统概述：从零开始的基础

装备强化是透过消耗特定材料来提升装备等级，进而增加其基础属性。
*   **强化材料**：主要包括各式「强化石」和「金币」。强化石有多种等级，对应不同稀有度或强化阶段的装备。金币则是所有强化的通用货币。
*   **强化等级与阶段**：装备通常有其最大强化等级，达到一定等级后，可能需要进行「突破」或「升阶」才能继续强化。这些进阶操作往往需要更稀有的材料，并且会大幅提升装备的潜力。
*   **属性成长**：每次强化都会按比例提升装备的基础属性。高稀有度装备在满级时，其总属性加成远超低稀有度装备。

4.  **根据角色定位精准投资**：
    *   **输出型角色**：优先强化武器（攻击力、暴击率）、提升伤害的饰品。
    *   **坦克型角色**：优先强化防具（防御力、生命值）、增加生存能力的饰品。
    *   **辅助型角色**：根据辅助类型强化，如提升治疗量的饰品、或增加速度以优先行动的饰品。

*   **错误**：将所有强化材料平均分配给所有装备，或盲目强化低稀有度装备。

*   **错误**：只追求最高稀有度装备，而忽略了当前可获得的较低稀有度但已强化的装备。

*   **错误**：长时间不关注游戏更新，导致强化策略过时，或错失新活动带来的优厚材料。

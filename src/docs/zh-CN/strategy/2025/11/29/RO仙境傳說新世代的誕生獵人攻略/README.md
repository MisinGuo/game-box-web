*   **核心：** 主点DEX，其次AGI。
*   **优势：** 技能伤害高，命中稳定，适合箭矢流猎人。
*   **缺点：** 攻速较慢，对装备攻速词条依赖高。

*   **核心：** 主点AGI，其次DEX。

### 4. LUK型（暴击流）


*   **核心：** 在DEX和AGI达到一定基础后，投入LUK提升暴击率和暴击伤害。
*   **优势：** 暴击伤害极高，视觉效果华丽。
*   **2025年应用：** 配合高暴击装备与卡片，在特定高难度副本中追求极致伤害。

**总结：** 2025年的猎人素质点法更倾向于**AGI-DEX混合**，并根据个人玩法偏好（鸟猎或箭矢流）进行微调。 LUK则作为后期提升伤害的选项。

#### 3.2.2 任务：「黑水沼泽」 (Lv. 35)

*   **任务目标：** 穿越黑水沼泽，寻找失踪的沼泽猎人。

        *   **阶段一：** 密切关注BOSS的抬手动作，及时躲避击飞和冲锋。利用位移技能保持安全距离。
        *   **阶段二：** 优先击杀召唤出的能量体，它们会为BOSS提供护盾。当BOSS施放全屏AOE时，寻找场景中的安全区域（若有）或开启无敌/高减伤技能。

要精通充能斧，必须深入理解其独特的能量管理系统。

### 1. 剑模式与斧模式的切换


*   **剑模式（Sword Mode）：** 你的主要积蓄瓶子模式。此模式下，你手持剑与盾，攻击速度快，机动性高，且能进行防御。
*   **斧模式（Axe Mode）：** 你的主要输出模式。此模式下，将剑与盾合体为一把巨斧，攻击力与攻击范围大幅提升，但移动速度较慢，且无法防御。
*   **变形斩：** 这是充能斧的精髓。剑模式下可变形为斧模式进行攻击，斧模式下亦可变形回剑模式。熟练运用变形斩能让你在攻防之间流畅切换，保持火力不坠。

### 2. 瓶子系统：充能与解放


充能斧的核心动力来源是「瓶子」（Phials）。

*   **积蓄瓶子：** 在剑模式下攻击魔物，剑的能量槽会逐渐积蓄。当能量槽变红时，表示已积蓄足够能量。
*   **充能瓶子：** 当能量槽变红后，使用「充能」动作（防御 特殊攻击）即可将能量转化为瓶子。瓶子数量会显示在萤幕左上角，最多可积蓄5个。
*   **解放瓶子：** 瓶子积蓄后，可以在斧模式下透过各种「属性解放斩」来释放，造成额外伤害。

### 3. 核心瓶子解放技


*   **属性解放斩（AED）：**
    *   **属性解放斩I：** 消耗1个瓶子，伤害较低，但发动速度快，可用于连段。
    *   **属性解放斩II：** 消耗所有瓶子，伤害较高，但发动较慢。
*   **超高出力属性解放斩（SAED）：**
    *   这是充能斧最具代表性的爆发技。消耗所有瓶子，将瓶子能量一次性全部释放，造成大范围、高额的瓶子爆炸伤害。
    *   **关键技巧：** SAED在发动时，若能精准命中魔物弱点，伤害极其可观。
*   **属性强化回旋斩（SR）：**
    *   在斧模式下，消耗所有瓶子，使斧头进入「强化状态」。强化后的斧头攻击力大幅提升，且每次攻击都会附带瓶子爆炸效果。
    *   **2025年策略：** 针对高血量、行动缓慢的魔物，SR的持续输出往往比SAED更稳定高效。

*   在发动SAED的过程中，可以透过向后推摇杆并按下特殊攻击键，将SAED「取消」为AED。
*   **应用场景：**

2.  **可靠的辅助/治疗 (Support/Healer)：**
    *   **核心职责：** 维持队伍血线，提供增益状态（Buff），解除负面状态（Debuff），有时也需控场。
    *   **推荐职业：** 大主教 (Arch Bishop)、吟游诗人/舞娘 (Minstrel/Wanderer)。

3.  **强力的输出 (Damage Dealer)：**
    *   **核心职责：** 在安全的前提下，对MVP造成最大化伤害。
    *   **推荐职业：** 基因学者 (Genetic)、游侠 (Ranger)、修罗 (Sura)、元素使 (Elemental Master)。
    *   **关键属性：** 物理/魔法攻击力、暴击率/爆伤、穿透、属性增伤。

**摇摆召唤运作原理：**

1.  **设置摇摆刻度：** 将两张「摇摆怪兽」分别放置在决斗场上的左右两个「摇摆区域」。每张摇摆怪兽都有两个数值，称为「摇摆刻度」。
2.  **执行摇摆召唤：** 一旦设置好两个摇摆刻度，玩家便可以在两个刻度值之间的等级范围内，从手牌或额外牌组（表侧表示的摇摆怪兽）特殊召唤任意数量的怪兽。
3.  **摇摆怪兽的特殊处理：** 当摇摆怪兽从场上被破坏送入墓地时，它们不会进入墓地，而是会表侧表示加入额外牌组。这使得它们可以在后续回合中再次被摇摆召唤。

### 1. 技能种类概览

![游戏场景](https://oss.2025.11.5yxy5.com/images/2025/11/11/%E5%8A%8D%E8%88%87%E9%81%A0%E5%BE%81%E5%95%9F%E7%A8%8B%E8%8B%B1%E9%9B%84%E6%8A%80%E8%83%BD/image_2_1762801938.png)

*   **主动技能（Active Skill）：** 英雄在战斗中会自动施放的技能，通常带有冷却时间。它们是英雄战斗风格的核心体现。
*   **终极技能（Ultimate Skill / 大招）：** 英雄积攒能量后手动或自动施放的强大技能，往往能扭转战局。合理的能量管理与大招施放时机至关重要。
*   **被动技能（Passive Skill）：** 无需施放，持续生效的技能。它们能为英雄提供常驻的增益、减益效果，或是在特定条件下触发。
*   **专属装备技能（Exclusive Equipment Skill）：** 英雄达到特定专属装备等级后解锁的额外被动或强化技能，能大幅提升英雄战力与功能性。在2025年的版本中，专属装备对英雄技能的强化作用更加显著，甚至能改变英雄的战术定位。

### 2. 技能关键词解析 (2025年常见)


理解技能描述中的关键词，能帮助您更快掌握技能效果：

*   **能量（Energy）：** 施放终极技能的资源。攻击、受击、击杀都能积攒。
*   **冷却时间（Cooldown）：** 技能施放后再次施放所需的时间。
*   **增益（Buff）：** 提升友方单位能力的正面效果（如攻击力提升、护盾）。
*   **减益（Debuff）：** 削弱敌方单位能力的负面效果（如攻击力降低、晕眩、中毒）。
*   **控制（Crowd Control, CC）：** 限制敌人行动的效果（如晕眩、冰冻、沉默、击退）。
*   **持续伤害（Damage Over Time, DoT）：** 在一段时间内持续造成伤害。
*   **爆发伤害（Burst Damage）：** 短时间内造成大量伤害。
*   **吸血（Lifesteal）：** 造成伤害时恢复自身生命值。
*   **抵抗（Resistance）：** 减少受到特定效果的机率或持续时间。

### 三、2025年最新公会战战术攻略


战术是公会战的核心，灵活多变的策略能让公会立于不败之地。

*   **快速突破：** 集中优势兵力，利用基因学者的「手推车攻击」或卢恩骑士的「龙之气息」等高爆发技能，快速突破敌方防线，直捣守护石碑或华丽金属。

*   **集火目标：** 指挥官需清晰指示集火目标，所有输出职业集中攻击同一目标，快速击杀。避免分散火力导致效率低下。

#### 3.2 防守策略：据点固守、反击、拖延

*   **据点固守：** 在守护石碑或华丽金属周围建立坚固防线，利用基因学者的「植物投掷」设置陷阱，或卢恩骑士的「龙之咆哮」击退敌人。大主教的「光耀之堂」和「圣母之祈福」是防守的关键。

*   **拖延战术：** 在兵力劣势或等待援军时，利用控场技能（如冰冻、石化）或走位拖延时间，为队友争取战术空间。

*   **诗人/舞娘/宫廷乐师/浪姬舞者：** 他们的合奏技能在公会战中能提供巨大的增益或减益效果，如「布莱奇之诗」减少技能延迟、「为您服务」提升攻速等，是战术的核心。
*   **教授/妖术师：** 「魔法省悟」与「地领」等技能能有效反制敌方魔法输出，或保护己方免受元素伤害。

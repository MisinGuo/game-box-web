### 优先强化哪些装备部位？


*   **武器 (Weapon)：** 提供大量攻击力，直接影响输出，永远是第一优先。
*   **饰品 (Accessory)：** 通常提供暴击、暴伤、命中等关键副属性，对输出和功能性英雄至关重要。
*   **防具 (Armor)：** 提供防御力与生命值，优先强化坦克英雄的防具，确保其生存能力。
*   **鞋子 (Boots)：** 提供速度，对需要抢先手或快速累积能量的英雄非常重要。

### 根据英雄定位选择强化策略


*   **输出型英雄 (C位)：** 优先强化武器、饰品，追求高攻击、高暴击、高暴伤，附魔也应围绕这些属性。
*   **坦克型英雄：** 优先强化防具、头盔、鞋子，追求高生命、高防御、韧性，附魔可考虑免伤、格挡。
*   **辅助型英雄：** 根据其功能性决定。例如，治疗型辅助可强化生命、治疗加成；控制型辅助可强化命中、速度。

*   **平均主义：** 许多新手会尝试平均强化所有英雄的装备，导致资源分散，没有一个英雄能真正发挥作用。
*   **盲目强化：** 不考虑装备的品质与潜力，将大量资源投入到低品质装备上。
*   **忽视附魔：** 只注重基础强化，却忽略了附魔带来的关键属性提升。
*   **不看套装效果：** 某些装备具有套装效果，盲目替换或强化单件可能失去套装增益。

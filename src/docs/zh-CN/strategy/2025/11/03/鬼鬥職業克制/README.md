冥术师是《鬼斗》中不可或缺的远程输出与控场角色，擅长范围伤害与限制敌人行动。

    *   **恶灵召唤者 (Fiend Summoner) (战线压制)：** 召唤者的多个召唤物能有效吸收冥术师的范围技能，并形成多线压制，限制其走位与输出空间。

    *   **魂武者 (Soul Warrior) (正面对决)：** 如果幽影刺客未能先手成功，在正面对决中，魂武者的持续输出和生存能力可能让刺客陷入被动。

镇魂师是团队的坚实后盾，拥有强大的防御、控制和辅助能力，是战场上的中流砥柱。

*   **克制对象：**
    *   **魂武者 (Soul Warrior)：** 镇魂师的控制技能可以打断魂武者的突进，高防御则能有效吸收其爆发伤害，保护后排。
    *   **幽影刺客 (Phantom Assassin)：** 镇魂师的范围侦测技能能揭露潜行中的刺客，并通过控制技能限制其行动，使其爆发无从发挥。

    *   **冥术师 (Underworld Mage) (持续消耗)：** 虽然镇魂师防御高，但冥术师的持续输出和穿透技能在长时间的战斗中会逐渐磨掉镇魂师的血量。
    *   **恶灵召唤者 (Fiend Summoner) (多线压力)：** 召唤者的大量召唤物能有效分散镇魂师的控制，并制造多线压力，使其难以兼顾。

**实战建议 (镇魂师)：**
专注于保护后排，利用控制技能限制敌方突进。在面对冥术师或召唤者时，尝试利用地形或配合队友进行反击。

恶灵召唤者擅长召唤多个小弟辅助作战，透过数量优势与持续伤害来压制敌人。

    *   **镇魂师 (Spirit Warden) (多线压力)：** 召唤者能通过多个召唤物制造多线压力，让镇魂师难以有效防御，同时对其造成持续伤害。
*   **被克制对象：**
    *   **魂武者 (Soul Warrior)：** 魂武者的高爆发和范围伤害能快速清理召唤物，并直接威胁到召唤者本体。
    *   **幽影刺客 (Phantom Assassin)：** 刺客能绕过召唤物的防线，直接切入后排击杀召唤者，使其战术核心崩溃。

**实战建议 (恶灵召唤者)：**
保持与敌人的距离，利用召唤物进行牵制和输出。警惕魂武者和幽影刺客的突进，适时使用控制技能保护自己。

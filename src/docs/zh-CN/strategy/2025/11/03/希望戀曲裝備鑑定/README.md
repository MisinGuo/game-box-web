*   **体力 (CON):** 提升生命值、防御力。适合所有职业，特别是坦克与生存型角色。
*   **智慧 (INT):** 提升魔法攻击力、魔法防御力。适合魔法输出职业（法师、祭司等）。

这些词条直接影响战斗表现：

*   **回避率 (Evasion):** 提升闪避敌人攻击的机率。
*   **暴击率 (Critical Rate):** 提升暴击的机率。
*   **暴击伤害 (Critical Damage):** 提升暴击时的伤害倍率。
*   **防御力 (Defense):** 减少受到的物理伤害。
*   **魔法防御力 (Magic Defense):** 减少受到的魔法伤害。
*   **HP/MP恢复量 (HP/MP Recovery):** 提升生命/魔法值的恢复速度。

*   **掉宝率增加 (Drop Rate Increase):** 刷宝必备，提升稀有物品掉落机率。
*   **技能等级提升 (Skill Level Up):** 提升特定技能的等级，直接强化技能效果。
*   **无视防御/魔法防御 (Ignore Defense/Magic Defense):** 极为稀有且强大的词条，能大幅提升伤害输出。

**价值判断原则：**

1.  **职业匹配度：** 词条是否符合你角色的职业特性和主要输出方式？

3.  **稀有度：** 稀有词条（如无视防御、对人伤害）通常价值更高。

要精通装备养成，首先必须理解「强化」与「升星」这两大核心机制的区别。

### 1. 装备强化（Enhancement）


*   **目的：** 提升装备的基础属性数值。
*   **材料：** 主要消耗「装备强化剂」（Equipment Enhancer）和金币。
*   **上限：** 每件装备都有其强化等级上限，通常与装备品质（颜色）相关。
*   **效果：** 每次强化都会增加装备的攻击、防御、生命等固定数值。
*   **风险：** 强化过程无失败率，但随着等级提升，所需材料和金币会大幅增加。

### 2. 装备升星（Star-Up）


*   **目的：** 解锁并提升装备的额外属性加成，并大幅提升基础属性。
*   **材料：** 主要消耗同名装备碎片或通用装备碎片。
*   **上限：** 每件装备最高可升至5星。
*   **效果：** 每次升星都会显著提升装备的百分比属性加成，并解锁新的星级属性。
*   **风险：** 升星通常需要稀有材料，且材料获取难度较高。

**总结：** 强化是「量」的提升，让装备更硬、更痛；升星是「质」的飞跃，解锁更强大的百分比加成和特殊能力。两者相辅相成，缺一不可。

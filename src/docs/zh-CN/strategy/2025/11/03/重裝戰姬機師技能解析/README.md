### 1. 技能类型与触发机制

*   **主动技能 (Active Skills)**：需要手动施放或达到特定条件后自动施放。通常具有强大的爆发伤害、控制效果或团队增益，是战斗中扭转局势的关键。
*   **被动技能 (Passive Skills)**：无需手动施放，在满足特定条件时自动生效。这些技能通常提供持续性的属性加成、伤害提升、减伤或特殊效果，是机师战力构成的重要基石。
*   **连携技能 (Link Skills)**：部分机师之间存在特殊的连携关系。当特定机师同时上阵并满足条件时，会触发额外的强力效果，极大提升队伍的整体战斗力。

### 2. 技能升级与强化


机师技能可以透过消耗资源进行升级，提升技能效果的数值（如伤害倍率、增益持续时间、减益效果强度等）。高阶技能往往需要稀有材料，因此合理规划资源，优先强化核心机师的核心技能至关重要。

### 3. 影响技能效果的关键因素


*   **机师属性**：机师的攻击、防御、生命等基础属性直接影响技能的最终效果。例如，攻击力高的机师其伤害技能会更为致命。
*   **机体搭配**：不同的机体类型（如重型、中型、轻型）与其自带的模组效果，能与机师技能产生化学反应。选择与机师技能定位相符的机体，能最大化技能效益。
*   **装备选择**：装备提供的额外属性加成（如技能冷却缩减、技能伤害加成、能量回复速度等）能显著提升技能的实战表现。

*   **输出-辅助-防御平衡**：一般而言，一个均衡的队伍应包含至少一名主要输出、一名辅助（增益/治疗/控场）和一名具备一定生存能力的机师。
*   **技能连携与互补**：寻找技能之间能够互相强化、互相弥补的机师。例如，控场技能后接范围爆发技能，或是能量回复技能搭配高耗能输出技能。

### 2. PVE (推图/活动) 技能选择与搭配


*   **推图效率**：优先选择具备范围伤害、快速清场能力的主动技能机师。搭配能提供能量回复或技能冷却缩减的辅助，确保技能循环流畅。
*   **活动挑战**：根据活动Boss的特性（如高护甲、免疫控制、特定属性弱点），选择具备对应输出类型或特殊效果的机师。例如，Boss有护盾时，选择具备破盾技能的机师。
*   **范例搭配**：
    *   **速刷队**：1-2名高爆发AOE输出   1名能量回复辅助   1名具备聚怪或增伤能力的机师。
    *   **Boss攻坚队**：1名高单体输出   1名强力控场   1名生存辅助   1名增伤辅助。

### 3. PVP (竞技场) 技能选择与搭配


*   **先手与爆发**：PVP强调快速击溃对手。选择开场就能施放强大控制或高爆发伤害技能的机师，争取先手优势。

*   **范例搭配**：
    *   **控场爆发流**：1名强力控场机师（如艾丽卡-时空冻结）  2名高爆发输出（如薇欧蕾特-星辰裁决、伊芙琳-量子崩解）  1名增伤辅助。
    *   **持久消耗流**：1名高生存坦克   1名持续治疗/护盾辅助   1-2名具备持续伤害或削弱能力的输出。

### 4. Boss战 (高难度挑战) 技能选择与搭配


*   **针对性输出**：分析Boss的弱点属性、防御类型和技能机制。选择能有效克制Boss的输出机师。
*   **生存与续航**：Boss战通常耗时较长且伤害极高。治疗、护盾、减伤等生存技能至关重要。
*   **打断与控制**：许多高难度Boss会有读条技能或狂暴阶段。具备打断或强力控制技能的机师能有效限制Boss的威胁。
*   **技能循环与施放时机**：在Boss战中，精准的技能施放时机是成功的关键。例如，在Boss施放高伤害技能前开启护盾，或在Boss防御降低时集中爆发。

*   **活动奖励**：参与游戏内限时活动或节日庆典，常有机会获得稀有食材。

1.  **选择食谱**：介面会显示你已解锁的所有食谱。新食谱通常透过任务、探索或提升料理等级解锁。
2.  **投入食材**：根据选定的食谱，将所需食材拖曳至对应的空位。请注意食材的数量和品质。
3.  **开始制作**：确认食材无误后，点击「制作」按钮。消耗一定的活力值和时间后，料理即告完成。

角色升星主要依赖以下几种材料：

1.  **幻影（Phantasma）：** 这是升星的核心材料。幻影本身就是一种特殊类型的角色，其主要用途是作为升星的「狗粮」。
    *   **白色幻影（Mega-Phantasma）：** 初始3星，可升至4星作为4星狗粮。
    *   **红色幻影（Giga-Phantasma）：** 初始4星，可升至5星作为5星狗粮。
    *   **金色幻影（Tera-Phantasma）：** 初始5星，可升至6星作为6星狗粮（通常不这样做，太浪费）。

2.  **狗粮角色：** 任何非幻影的普通角色，只要星级与等级符合要求，也能作为升星材料。通常玩家会使用低星级（1星、2星）角色来培养幻影或直接作为低星级升星的材料。
3.  **金币：** 每次升星都需要消耗一定量的金币。星级越高，消耗的金币越多。

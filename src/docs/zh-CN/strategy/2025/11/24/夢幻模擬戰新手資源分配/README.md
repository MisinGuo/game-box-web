金币是游戏中最基础也最万能的货币，贯穿游戏始终。

*   **极高优先级**：
    *   **英雄升级与精通**：这是最主要的金币开销，确保主力队伍等级与兵种精通跟上进度。

*   **低优先级**：

英雄升级的必需品。

*   **极高优先级**：
    *   **主力英雄**：集中资源优先将5-6位主力英雄提升至当前等级上限。
    *   **兵营英雄**：为了解锁练兵场兵种，部分英雄需要达到特定等级，也需适度投入。

*   **极高优先级**：
    *   **符文石**：只用于主力输出、核心辅助和坦克英雄的**关键职业精通**。例如，输出英雄的攻击、技巧线，坦克英雄的防御、生命线。每个英雄最多使用2-3颗符文石。
    *   **魔导石**：用于主力英雄的**最终转职**，解锁最强职业与技能。
*   **新手常见错误**：
    *   **随意使用**：将符文石用在非主力或过渡型英雄身上。

**2025年提示**：符文石和魔导石的稀有度是游戏中最高的，务必三思而后行。在决定使用前，务必查阅最新英雄评价和职业转职攻略，确保投资在当前版本强势且适合长期培养的英雄身上。

    *   **命运之扉**：每日完成主力英雄的命运之扉，获取碎片提升星级，同时也能解锁羁绊。
*   **避免**：
    *   **盲目提升所有英雄羁绊**：初期资源有限，集中投资主力英雄。

**2025年提示**：羁绊是性价比极高的投资，务必每日刷满主力英雄的命运之扉。

*   **极高优先级**：
    *   **主力英雄所带兵种**：优先提升主力输出、坦克和辅助英雄常规携带的兵种科技。例如，步兵、枪兵、飞兵、弓兵等。

**2025年提示**：练兵场的提升是一个漫长的过程，但收益巨大。每日的兄贵健身房是获取材料的主要途径，务必刷满。

*   **极高优先级**：
    *   **精选卡池**：集中圣魔券和圣魔晶抽取当前版本强势且适合新手的主力英雄。
    *   **新手池**：初期的新手精选池非常划算，务必完成。
*   **避免**：
    *   **分散投资**：切忌在多个卡池中浅尝辄止，容易导致两头空。

*   **保护后排：** 确保远程输出和辅助单位始终位于近战单位后方，避免被敌方突进单位直接攻击。

*   **关键控制：** 将眩晕、击退等控制技能保留给敌方关键单位（如敌方高输出、治疗单位或攻城单位）。

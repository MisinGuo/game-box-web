精炼传承是《RO仙境传说：守护永恒的爱》中一项极为重要的机制。

*   **传承规则：** 通常同类型装备（如：单手剑传单手剑、披风传披风）才能进行传承。传承后，旧装备的精炼等级会转移到新装备上，旧装备的精炼等级归零。
*   **传承时机：** 当你获得一件更稀有、属性更好的装备（例如：从蓝色装备升级到金色装备，或获得洞数更多的装备）时，是进行传承的最佳时机。
*   **节省成本：** 传承可以让你将辛苦精炼的成果延续下去，避免重复投入大量资源。

*   **武器：** 毫无疑问是首要精炼对象，直接影响输出能力。

*   **防具（盔甲、披风、鞋子）：** 提供防御、生命等生存属性，可根据自身坦度需求精炼。

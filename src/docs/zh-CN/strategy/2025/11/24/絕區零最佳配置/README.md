攻擊型代理人是隊伍的主要傷害來源，負責在短時間內對敵人造成大量傷害。選擇時需考量其爆發力、持續輸出能力及對不同屬性敵人的適應性。

    *   **增伤/减防辅助：** 提升主C的伤害，加速战斗进程。
    *   **能量恢复器：** 加速队友必杀技的充能，提高技能循环。
*   **配置重点：** 优先堆叠能量恢复效率、生命值（影响治疗量或护盾值）、效果命中（影响Debuff成功率），并选择能提供团队增益的装备。

防御/控制型代理人负责吸引敌人火力、打断敌人攻击或限制敌人行动，为输出型代理人创造安全的输出环境。

*   **推薦角色類型：**

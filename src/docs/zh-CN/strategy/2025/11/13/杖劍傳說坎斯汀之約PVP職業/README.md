### 2. 团队协作与沟通


在团队PVP中，有效的沟通是胜利的关键。
*   **集火目标**：明确告知队友集火哪个目标，优先击杀敌方威胁最大的单位。

### 4. 资源管理与爆发时机


PVP中的每一个技能、每一次普通攻击都应有其目的。
*   **技能冷却**：不要盲目交出所有关键技能，尤其是在没有明确目标或机会时。
*   **爆发时机**：在敌方关键减伤技能交出、或敌方被控制时，才是最佳的爆发时机。
*   **法力/能量**：合理分配资源，确保在关键时刻有足够的资源施放技能。

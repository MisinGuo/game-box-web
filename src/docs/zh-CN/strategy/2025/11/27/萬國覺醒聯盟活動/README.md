### 1. 圣所与祭坛争夺战


这是联盟间最直接的PVP冲突，也是展现联盟实力的最佳舞台。

*   **活动目标**：在指定时间内占领并守护地图上的圣所或祭坛。
*   **2025年策略要点**：
    *   **前置侦查与集结**：活动开始前，R4/R5应指派成员侦查目标点周边敌情，并提前发动集结。
    *   **兵力配置多元化**：集结队伍中应包含不同兵种，以应对敌方可能的多样化兵种搭配。

### 2. 联盟科技研究与捐献活动


联盟科技是联盟发展的命脉，捐献活动则是加速科技进程的有效方式。

*   **活动目标**：透过捐献资源或加速道具，提升联盟科技等级，同时获取个人贡献点与奖励。
*   **2025年策略要点**：
    *   **日常累积**：平时就应养成捐献习惯，尤其是在有捐献加成活动时，效益最大化。
    *   **善用资源箱**：储存部分资源箱，在捐献活动开启时一次性使用，冲刺排名。
    *   **钻石捐献**：在冲刺关键科技或个人排名时，适度使用钻石进行捐献，性价比通常较高。
    *   **联盟目标**：R5/R4应设定明确的科技研究目标，引导成员集中捐献。

    *   **侦查与反侦查**：派遣侦察兵巡逻周边，防止敌盟偷袭；同时也要注意隐藏自己的采集点。
    *   **利用联盟堡垒**：将资源点设在联盟堡垒覆盖范围内，可获得防御加成。

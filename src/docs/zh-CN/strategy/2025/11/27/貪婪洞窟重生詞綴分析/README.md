### 词缀的种类与稀有度

游戏中的词缀大致可分为以下几类：

*   **基础属性词缀：** 直接提升角色生命、攻击、防御、魔力等基本数值。
*   **元素属性词缀：** 增加角色对特定元素的攻击力或防御力，如火、水、光、暗等。

词缀本身也有稀有度之分，从普通、魔法、稀有、史诗到传说，稀有度越高，词缀的数值上限与潜力也越大。 2025年版本中，传说级词缀的影响力尤其显著，往往能颠覆现有流派。

### 词缀数值与成长性

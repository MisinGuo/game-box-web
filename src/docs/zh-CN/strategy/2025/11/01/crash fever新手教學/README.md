### 消除方块：攻击、回复与技能蓄力


*   **方块种类：** 蓝、绿、红、黄四种属性方块，以及爱心方块（回复）、Crash Panel（C珠，单位技能发动）、炸弹方块等特殊方块。
*   **消除机制：** 点击任一方块，与其相邻的同色方块会一同消除。消除的方块越多，造成的伤害或回复量越高。
*   **连锁消除（Chain）：** 透过一次点击消除大量方块，或引发连锁反应，能累积更高的伤害与Fever计量条。
*   **C珠（Crash Panel）生成：** 消除特定数量的方块会生成C珠，点击C珠即可发动单位技能。

### 单位技能与Fever模式：战斗的转捩点


1.  **单位技能（Skill）：** 每个单位都拥有独特的技能，分为主动技能（点击C珠发动）和被动技能（常驻或特定条件发动）。技能效果多样，包括单体/全体攻击、回复、辅助（提升攻击/防御）、妨碍（降低敌人能力）等。
2.  **Fever模式：** 透过消除方块累积Fever计量条。计量条满后即可发动Fever模式。
    *   **Fever期间：** 攻击力大幅提升，且消除方块不再受属性限制，可以随意点击任何方块进行消除。这是爆发伤害、扭转战局的关键时刻。
    *   **Fever结束：** 会根据Fever期间消除的方块数量造成额外伤害。
3.  **属性相克：** 蓝克红、红克绿、绿克黄、黄克蓝。无属性单位（如彩虹色）则不受相克影响。在组队和战斗中，利用属性优势能造成更多伤害。

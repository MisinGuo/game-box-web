### 1. 明确分工，各司其职

在游戏初期，与队友讨论并确定各自擅长的领域：
*   **采集者（Gatherer）**：专注于木材、矿石、纤维等基础资源的获取。
*   **建造者（Builder）**：负责基地选址、设计与施工，确保防御坚固。
*   **猎人/战士（Hunter/Warrior）**：负责食物来源、防御敌人，以及外出探索时的战斗。
*   **探险家（Explorer）**：负责绘制地图、发现新资源点与潜在威胁。
*   **农夫/厨师（Farmer/Cook）**：负责食物生产、烹饪，维持团队温饱。

### 2. 有效沟通，情报共享

无论是透过语音聊天（Discord、游戏内语音）还是文字讯息，保持即时且清晰的沟通是成功的关键。
*   **报告资源位置**：发现稀有资源立即告知队友。
*   **警示危险**：遭遇敌人或陷阱时，立刻预警。
*   **协调行动**：外出探索、攻击目标、撤退时，必须同步行动。
*   **定期会报**：每天或每周（游戏内时间）进行一次进度会报，讨论下一步计画。

### 3. 资源集中管理与分配

虽然每个人都能采集，但将大部分资源统一存放于共用储物空间，并由一人或两人负责管理，能避免混乱与浪费。
*   **建立共用仓库**：确保所有队友都能存取。
*   **设定优先级**：讨论哪些资源优先用于基地建设、哪些用于工具升级、哪些用于食物生产。
*   **避免独占**：切勿一人独占稀有资源，应以团队利益为重。

### 4. 基地选址与防御规划

基地是团队的生命线，其选址与防御至关重要。
*   **选址考量**：靠近关键资源（水、矿、森林）、地势险要（易守难攻）、视野开阔（早期预警）。
*   **多层防御**：围墙、陷阱、哨塔、照明设施缺一不可。
*   **紧急撤离点**：规划好在基地被攻破时的撤离路线和临时避难所。

### 5. 战斗协同与应变

在面对敌人时，团队合作远比单打独斗有效。
*   **火力集中**：优先集火威胁最大的目标。
*   **角色搭配**：近战肉盾在前吸引火力，远程输出在后安全输出，治疗辅助随时支援。
*   **战术撤退**：当情况不利时，果断协调撤退，保存有生力量。

### 6. 共同目标与长期规划

生存游戏往往是一个漫长的过程。与队友设定短期（例如：今晚盖好围墙）和长期目标（例如：建造大型农场、探索地图边界），能够让团队保持方向感和动力。

### 1. 資源獲取型技能：穩定手牌與展開


這類技能旨在提升手牌質量或增加場上資源，是許多牌組穩定展開的基石。

*   **抽牌感知系列（Draw Sense: High-Level / Low-Level / Spell / Trap 等）**：當生命值降低到一定程度時，能夠從牌組中抽取指定類型的卡片。這類技能在逆風時提供關鍵補牌，或在特定戰術中穩定檢索核心組件，例如「抽牌感知：高星」能穩定檢索儀式或融合素材。
*   **LP恢復技能（Life Point Boost α / β）**：雖然不直接影響手牌，但增加生命值上限能為一些需要付出生命值代價的卡片提供緩衝，或延長對局時間，為逆轉創造機會。
*   **特定卡片檢索技能（如：真紅眼融合、光道招來等）**：這些技能能直接從牌組中檢索或特召特定卡片，極大地提升了特定牌組的穩定性和爆發力。例如「真紅眼融合」直接將「融合」上手，讓「真紅眼」牌組的展開更為順暢。

### 2. 場地與環境控制型技能：改變戰局的舞台


改變場地魔法或提供特定效果，能為自身牌組提供優勢，同時限制對手。

*   **場地魔法技能（如：頂級抽牌、闇）**：直接將特定的場地魔法發動。例如「頂級抽牌」能為龍族、機械族怪獸提供攻擊力加成，是「藍眼白龍」等牌組的常駐選擇；「闇」則能強化惡魔族與魔法使族。
*   **技能抽牌（Skill Drain - 角色技能版，非卡片）**：某些角色技能可能帶有類似「技能抽牌」的效果，能限制怪獸效果的發動，對依賴怪獸效果的牌組造成巨大打擊。

### 3. 特殊戰術型技能：創造獨有勝利方程式


這些技能往往能催生出獨特的戰術或OTK（一回合殺）潛力。

*   **注定一抽（Destiny Draw）**：當生命值降低到一定程度時，可以選擇抽取牌組中任意一張卡片。這是最強大的資源獲取技能之一，能確保在關鍵時刻抽到制勝的卡片，適用於幾乎所有牌組，但需要精準的生命值管理。

*   **技能憑依（Skill Prisoner）**：在特定條件下，能夠發動一次性效果來應對對手的招數。這類技能往往能作為奇襲或解場手段，出其不意地打亂對手的節奏。

《铃兰之剑》的圣物主要分为多种套装，每套圣物拥有独特的2件套和4件套效果。理解这些效果是搭配圣物的首要步骤。

### 1. 主词条与副词条：属性加成的核心


*   **主词条（Primary Stat）**：每件圣物在特定部位会有固定的主词条类型，例如头部通常是生命值百分比，身体是防御力百分比，项链和戒指则可能出现攻击力百分比、暴击率、暴击伤害等关键输出词条。选择与角色定位匹配的主词条至关重要。
*   **副词条（Substats）**：每件圣物会随机生成数条副词条，并在强化时随机提升。副词条的种类繁多，包括攻击力、生命值、防御力、暴击率、暴击伤害、效果命中、效果抵抗、速度等。追求高数值和有效词条的副词条是圣物养成的终极目标。在2025年，对于输出角色，暴击率、暴击伤害、攻击力百分比仍是主流；辅助角色则更侧重速度、生命值、效果抵抗。

圣物的获取与强化是一个长期且需要策略的过程。

### 1. 圣物获取途径（2025年最新）


*   **圣物副本（秘境挑战）**：这是最主要的圣物来源。不同难度的副本掉落不同品质的圣物，高难度副本有机会掉落稀有词条的圣物。 2025年版本新增了一些限时圣物副本，掉落特定套装，务必把握。
*   **活动奖励**：游戏内定期举办的限时活动、签到奖励、通行证等，常会赠送圣物箱或圣物自选礼包，这是获取特定圣物的好机会。
*   **商店兑换**：部分商店（如竞技场商店、公会商店）可以用特定货币兑换圣物，虽然数量有限，但可以弥补急需的部位。
*   **世界首领/高难度挑战**：击败世界首领或完成高难度挑战，也有机会获得高品质圣物。

### 2. 圣物强化与洗练


*   **强化（Upgrade）**：消耗金币和强化材料（或低级圣物）提升圣物等级。每提升3级或5级（依圣物品质而定），会随机提升一条副词条或新增一条副词条。这是提升圣物属性最直接的方式。
*   **洗练（Reroll）**：消耗特定材料（如洗练石）随机重置圣物的一条副词条。这项功能在2025年版本中变得更加重要，因为它允许玩家在一定程度上「修正」圣物，将无用词条替换成更有效的词条，但成本较高，建议用于已经有优秀主词条和多条有效副词条的圣物。

**高效强化建议**：
*   优先强化主词条正确的圣物。
*   在强化过程中​​，如果副词条走向不理想（例如输出圣物一直强化到效果抵抗），考虑停止强化或进行洗练。
*   利用游戏内的「圣物预览」功能，查看潜在的副词条升级方向，避免资源浪费。

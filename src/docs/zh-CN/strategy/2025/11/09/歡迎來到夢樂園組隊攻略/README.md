### 一、队伍角色定位与基本配置原则

![游戏场景](https://oss.2025.11.5yxy5.com/images/2025/11/09/%E6%AD%A1%E8%BF%8E%E4%BE%86%E5%88%B0%E5%A4%A2%E6%A8%82%E5%9C%92%E7%B5%84%E9%9A%8A%E6%94%BB%E7%95%A5/image_2_1762627887.jpg)

在组建队伍前，首先要理解各角色的基本定位，并确保队伍具备以下核心功能：

1.  **核心输出 (DPS)：** 队伍的主要伤害来源，负责快速击败敌人。可以是物理、魔法或真实伤害型。
2.  **坚实防御 (Tank/副坦)：** 承受敌方主要伤害，保护后排输出与辅助角色。具备高生命值、防御力或减伤技能。
3.  **强力辅助 (Support)：** 提供增益效果（如攻击力、防御力提升）、治疗、护盾，或削弱敌方（如减速、破防）。
4.  **战场控制 (Crowd Control, CC)：** 透过晕眩、冰冻、沉默、击退等技能，打断敌方施法、限制其行动，为我方创造输出机会。

**2025年推荐基本配置：** 1坦   1辅助   1控制   2输出，或 1坦   2辅助   2输出 (视辅助功能而定)。

### 三、角色协同与技能搭配的艺术


单纯的角色堆叠并不能带来最强战力，真正的精髓在于技能的「协同效应」。

*   **增益叠加：** 例如，【战歌祭司】瑟琳娜的攻击增益搭配【虚空魔女】伊莎贝尔的爆发技能，能让伤害瞬间达到顶峰。
*   **控制链：** 【冰霜女皇】芙蕾雅的冰冻接上【秘术学者】奥萝拉的晕眩，可实现长时间的硬控，让敌人无法反击。
*   **破防与输出：** 【暗影游侠】罗宾的破防技能，能让所有物理、魔法输出角色打出更高伤害。
*   **护盾与治疗：** 【圣光使者】莉亚的群体治疗与【生命之泉】艾琳的单体护盾，能互补不足，应对不同伤害模式。

**2025年特别提醒：** 许多新角色或调整后的旧角色，其技能描述中会明确指出与特定类型队友的协同效果，务必仔细阅读。例如，某角色可能对「带有燃烧效果的敌人」造成额外伤害，这就意味着你需要搭配能施加燃烧的角色。

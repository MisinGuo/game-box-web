**副属性的种类与作用：**

*   **速度 (SPD)：** 决定式神在战斗中的行动顺序，是PVP和部分PVE玩法的核心。
*   **暴击率 (Crit Rate)：** 增加式神造成暴击伤害的机率，输出式神的基础。
*   **暴击伤害 (Crit DMG)：** 提升暴击时的伤害倍率，是输出式神爆发的关键。
*   **攻擊加成 (ATK %)/生命加成 (HP %)/防禦加成 (DEF %)：** 以百分比形式提升式神的攻擊力、生命值或防禦力。
*   **效果命中 (Effect HIT)：** 提高控制式神施加负面效果的成功率。
*   **效果抵抗 (Effect RES)：** 降低式神受到负面效果的机率。
*   **固定攻击/生命/防御值：** 直接增加固定数值，通常不如百分比加成。

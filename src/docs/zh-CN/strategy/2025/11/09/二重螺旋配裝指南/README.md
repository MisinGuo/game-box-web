盲目追求高稀有度装备是新手常犯的错误。真正的配装高手会根据以下原则进行选择：

### 1. 角色定位与技能特性

首先，明确你所操作角色的定位（输出、坦克、辅助、治疗）以及其核心技能的伤害类型（物理、魔法、特定元素）。例如，依赖暴击的输出角色应优先堆叠暴击率和暴击伤害。

### 2. 内容需求：PVE vs. PVP

不同的游戏内容对配装有不同要求。
*   **PVE（副本、Boss战）：** 更注重稳定输出、生存能力和续航。某些副本可能需要特定抗性或控制效果。
*   **PVP（竞技场、对战）：** 爆发伤害、控制效果、速度和反手能力更为重要。

### 3. 属性平衡与阈值

过度堆叠单一属性会导致边际效益递减。例如，暴击率达到一定阈值后，再提升暴击伤害会比继续堆暴击率更有价值。透过游戏内数据或社群分享的「属性阈值表」，可以更精准地分配属性。

### 4. 装备词条的优先级

在次要屬性上，通常優先選擇與角色核心屬性相關的百分比加成詞條，而非固定數值。例如，「攻擊力%」通常優於「固定攻擊力」。對於輸出角色，暴擊率、暴擊傷害、元素傷害加成是首選；對於坦克，生命值%、防禦力%和減傷更為關鍵。

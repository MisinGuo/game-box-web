「装备继承」是《枫之谷M》中将旧装备的星力强化等级与潜能设定，安全地转移到新装备上的重要功能。它允许玩家在不损失宝贵强化资源的前提下，更换更高阶的装备。

### 2.1 什么是装备继承？


装备继承是指将一件「A装备」的星力强化等级和潜能属性，完整地传递给另一件「B装备」。通常用于当你获得了一件相同部位但等级更高、素质更好的装备时，避免重新从零开始强化。

### 2.2 装备继承的适用情境与优势


*   **更换更高阶装备：** 当你从史诗装备升级到独特、传说，甚至是神话装备时，继承是最佳选择。
*   **保留高星力与潜能：** 你不需要重新冲星或洗潜能，大幅节省金钱与时间。
*   **降低强化风险：** 避免了从低星开始强化的失败机率与装备损毁风险。

### 2.3 2025年装备继承操作教学与注意事项


1.  **开启继承介面：** 前往装备强化NPC（如赫丽泰尔），选择「装备继承」。
2.  **选择材料装备（旧装备）：** 放入你已强化好的旧装备，这件装备的星力与潜能将被继承。
3.  **选择目标装备（新装备）：** 放入你想要继承的新装备。
4.  **确认消耗：** 继承会消耗一定的枫币、继承石（或相应材料），以及新装备的等级。请注意，新装备的等级会被重置为1级。
5.  **执行继承：** 确认无误后执行，新装备将获得旧装备的星力与潜能。

*   **装备稀有度限制：** 通常只能从低稀有度继承到高稀有度，或相同稀有度之间继承（例如：传说继承到传说）。跨越稀有度继承时，请务必确认游戏内说明。

3.  **选择目标装备（新装备）：** 放入你想要接收这些属性的新装备。
4.  **确认消耗：** 转移会消耗枫币、转移石（或相应材料）。请注意，转移后旧装备通常会消失或降级。
5.  **执行转移：** 确认无误后执行。

*   **魂之宝珠转移：** 魂之宝珠转移通常要求新旧装备的种类和部位相符（例如：皆为武器，且可镶嵌魂之宝珠）。部分高阶魂之宝珠的转移可能需要特定的转移石。

### 1. 区域划分：功能性分区


将城市划分为几个独立的功能区域，是提升效率的第一步：

*   **资源生产区：** 集中所有资源田（农田、伐木场、采石场、金矿）。

*   **防御区：** 城墙、箭塔、哨站。

### 2. 加成建筑邻近原则


游戏中许多建筑会对周围的特定建筑提供加成。例如，某些科技或统帅技能会增加相邻资源田的产量。确保这些具有加成效果的建筑（如学院、仓库、部分统帅雕像）能最大化地覆盖到其目标建筑。

*   **前期（T1-T3兵种阶段）：** 重点在于资源生产与科技研发，确保资源田数量充足且效率高。军事建筑保持最低限度以解锁兵种。
*   **中期（T4兵种阶段）：** 资源与军事并重。开始强化军事训练区，并为医院预留足够空间。
*   **后期（T5兵种阶段及征服赛季）：** 军事训练、医院容量和防御体系成为重中之重。资源生产效率依然重要，但可适当调整部分资源田为军事或医院。

联盟中心是城市防御的关键，它能提供联盟增援。务必将其放置在城墙内部，最好是城市中心位置，以获得最大程度的保护。城墙的设计应尽量包围所有重要建筑，形成坚固的防线。

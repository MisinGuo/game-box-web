*   **核心理念**：最大化攻击力、暴击率、暴击伤害，并辅以一定的攻速。

        *   **【战魂】**：直接提升攻击力。
        *   **【暴击大师】**：提升暴击率。
        *   **【弱点洞察】**：提升暴击伤害。
        *   **【破甲】**：降低敌人防御，间接提升输出。

    1.  优先点满**战魂**、**暴击大师**，确保基础输出与暴击机率。
    2.  接着点出**虹吸斩**与**暴怒斩击**，构筑输出循环。
    3.  补足**弱点洞察**和**破甲**，进一步放大伤害。

*   **装备、属性搭配建议**：
    *   **属性**：优先攻击力、暴击率、暴击伤害、穿透。

*   **适用场景与优缺点**：
    *   **优点**：清怪速度快，单体BOSS爆发高。
    *   **缺点**：对装备依赖度高，生存能力相对较弱，需要走位或吸血维持。

        *   **【冷却缩减】**：加快技能循环。
        *   **【穿透射击/法术穿透】**：无视部分敌人抗性。

    1.  优先点满**元素精通**、**法力恢复**，确保基础输出与续航。

    3.  补足**冷却缩减**、**闪现/疾跑**，提升战斗节奏与生存。
*   **装备、属性搭配建议**：
    *   **属性**：优先魔法攻击力、元素伤害、冷却缩减、魔力值。
    *   **装备**：选择带有高魔法攻击、元素伤害、冷却缩减词缀的法杖、饰品。
*   **适用场景与优缺点**：
    *   **优点**：控场能力强，范围输出优秀，战术选择多样。
    *   **缺点**：对操作要求较高，需要良好走位，爆发可能不如纯物理流。

### 1. 如何根据装备词缀调整技能


装备是BD的另一半。许多强力装备会提供「 X级技能」、「技能额外投射物」、「特定伤害类型增益」等词缀。

*   **优先级判断**：当你获得一件带有强力技能词缀的装备时，应评估它是否能与你当前的核心技能形成更好的协同。有时，一件关键装备足以让你调整整个技能链甚至转换流派。
*   **词缀与天赋的互补**：例如，如果装备提供了大量暴击率，你就可以在天赋上多投入暴击伤害，而非继续堆叠暴击率，达到效益最大化。
*   **抗性与防御**：别忘了装备提供的抗性与防御词缀。它们是确保生存的基石，技能搭配也应考虑到自身抗性是否足够，是否需要额外技能来弥补。

### 2. 避免技能点浪费：精准规划


《火炬之光：无限》的技能点是宝贵的。每一次点选都应该有明确的目的。

*   **预先规划**：在开始构筑BD前，最好能大致规划好主要技能与其辅助技能的搭配，以及天赋树的走向。
*   **避免重复增益**：如果某个增益（如增加暴击率）已经从天赋或装备中获得足够，就无需在辅助技能上重复选择，应选择其他能带来乘法效益的辅助。
*   **资源管理技能**：确保你有足够的魔力或能量来持续施放技能。适当点选回魔天赋或使用回魔技能/光环至关重要。

### 3. 警惕「万金油」陷阱：专精带来强度


虽然有些技能看起来很通用，但过于分散的技能点与搭配往往会导致「样样通样样松」。

*   **专精核心**：选择1-2个核心主动技能，并将所有资源（技能点、装备词缀、天赋）集中投入，使其发挥最大威力。
*   **避免多核BD**：新手尤其容易犯的错误是想同时强化多个主动技能。这会稀释你的资源，导致任何一个技能都无法达到顶尖强度。
*   **功能性技能的选择**：位移、诅咒、光环等功能性技能可以适当选择，但它们的辅助技能链不应占用过多资源，通常1-2个辅助技能足矣。

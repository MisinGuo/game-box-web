### 篮板类型与时机判断


游戏中的篮板主要分为几种类型，理解其特性是成功抢板的第一步：

*   **卡位篮板：** 在对手投篮后，迅速移动到篮筐下方有利位置，利用身体阻挡对手，为队友或自己创造抢板空间。这需要极佳的预判能力和卡位意识。

*   **空中篮板：** 在球还未落地前，预判球的轨迹并直接起跳将球抓下。这是最高效的抢板方式，但对时机的把握极为严苛。

**时机的重要性：预判球的落点**
篮板的精髓在于「预判」。每次投篮后，篮球的弹道和落点都是可以预测的。
*   **近距离投篮（如上篮、灌篮）：** 球多数会弹向篮筐两侧或前方。
*   **中距离投篮：** 球可能弹向篮筐后方或左右两侧。
*   **远距离投篮（三分球）：** 球的弹道较平，弹出后通常会飞得更远，落点可能在三分线附近。

观察投篮者的位置、出手角度、以及球的飞行速度，能帮助你更精准地判断落点，提前移动。

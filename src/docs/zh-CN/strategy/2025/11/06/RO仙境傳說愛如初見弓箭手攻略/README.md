### 1. 敏捷灵巧流（Agi-Dex）：普攻输出核心


这是最经典也最受欢迎的弓箭手流派，专注于提升自动攻击的效率与伤害。

*   **核心属性：**
    *   **敏捷（AGI）：** 优先点满，提供大量攻速（ASPD）与少许闪避。高攻速是普攻流的核心。
    *   **灵巧（DEX）：** 次要点满，提供大量物理攻击力（ATK）、命中与咏唱速度。 DEX是弓箭手伤害的基石。
    *   **幸运（LUK）：** 可适量投资，提供暴击率、暴击伤害与少许完全回避。在Agi-Dex成型后，补LUK能显著提升输出上限。
    *   **体质（VIT）：** 除非是特殊PVP需求，否则初期不建议点，以保证输出最大化。

*   **配点策略（以99级为例）：**
    *   **初期（1-40级）：** DEX优先点到40-50，确保命中与基础伤害，然后开始点AGI至30-40。
    *   **中期（41-70级）：** 继续提升AGI至70-80，DEX保持在60-70。此时攻速与伤害会有显著提升。
    *   **后期（71-99级）：** 将AGI点至90-99，DEX补至90-99。之后可将剩余点数投资于LUK，或少量VIT提升生存力。

### 2. 灵巧智慧流（Dex-Int）：技能陷阱辅助


此流派更侧重于技能伤害、陷阱控制与辅助能力，在部分PVP或特定副本中表现出色。

*   **核心属性：**
    *   **灵巧（DEX）：** 优先点满，提升技能伤害、命中与咏唱速度。
    *   **智慧（INT）：** 次要点满，提升SP上限、SP恢复速度、魔法攻击力（对部分技能有加成）以及陷阱伤害。
    *   **体质（VIT）：** 适量投资，提高血量与防御，增强生存能力，以便在战斗中稳定施放技能。

*   **配点策略（以99级为例）：**
    *   **初期（1-40级）：** DEX优先点到50-60，然后开始点INT至30-40。
    *   **中期（41-70级）：** 继续提升DEX至80-90，INT保持在50-60。
    *   **后期（71-99级）：** 将DEX点至99，INT补至80-90。剩余点数投资于VIT，提升坦度。

### 什麼是技能連結？主動與輔助技能的協奏

![游戏场景](https://oss.2025.11.5yxy5.com/images/2025/11/19/%E7%81%AB%E7%82%AC%E4%B9%8B%E5%85%89%E7%84%A1%E9%99%90%E6%8A%80%E8%83%BD%E9%80%A3%E7%B5%90/image_2_1763508880.png)

《火炬之光：無限》中的技能系統由兩大類寶石構成：

*   **主動技能寶石：** 這是你實際施放的技能，例如「寒冰射擊」、「火焰脈衝」或「狂猛斬擊」。它們決定了你的主要攻擊或防禦手段。
*   **輔助技能寶石：** 這些寶石本身不提供主動效果，但當它們與主動技能寶石連結時，會為主動技能提供各種加成效果，如增加傷害、改變技能範圍、觸發額外效果、降低魔力消耗等。

當一個主動技能寶石與一個或多個輔助技能寶石連結時，這些輔助技能的效果就會應用到主動技能上。一個主動技能最多可以連結五個輔助技能，形成一個完整的技能組。

### 連結槽位、稀有度與顏色匹配


技能寶石必須鑲嵌在裝備上的「連結槽位」中。這些槽位有以下特性：

*   **數量：** 每件裝備的槽位數量不同，從1個到6個不等。連結槽位必須是「連通」的，才能讓輔助技能生效。
*   **顏色：** 槽位有紅（力量）、綠（敏捷）、藍（智慧）三種顏色。技能寶石也有對應的顏色，必須鑲嵌在顏色匹配的槽位中。
*   **稀有度：** 輔助技能寶石的稀有度（魔法、稀有、傳奇）會影響其提供的加成效果強度。傳奇輔助技能通常具有獨特且強大的效果。

在2025年版本中，靈活運用裝備上的多連結槽位，並透過「工匠之錘」等貨幣調整槽位顏色，是構築複雜BD的關鍵。

### 連結效果與加成方式：倍率與詞綴


輔助技能對主動技能的加成方式主要有兩種：

1.  **倍率加成：** 大部分輔助技能會直接增加主動技能的傷害、攻擊速度、施法速度等，以百分比形式呈現。例如「額外物理傷害」會將其自身的物理傷害添加到連結的主動技能上，再受主動技能的傷害倍率影響。
2.  **條件觸發或改變機制：** 部分輔助技能會改變主動技能的施放方式（如「連鎖」、「散射」），或在特定條件下觸發額外效果（如「擊中附加詛咒」）。

理解這些加成如何疊加，是計算最終傷害與效果的基礎。例如，多個「增加傷害」詞綴通常是加法疊加，而「更多傷害」（More Damage）則是乘法疊加，後者對總傷害的提升更為顯著。

對於新手玩家而言，技能連結系統初期可能有些複雜。避免以下常見誤區，能讓你更快上手。

### 1. 盲目追求高傷害，忽略生存與回復


這是新手最常犯的錯誤。雖然高傷害能快速清怪，但在面對Boss或高難度異界時，一旦生存不足，再高的傷害也無法發揮。永遠記住，只有活著才能輸出。

### 2. 忽略技能標籤與連結匹配


每個技能寶石都有其「標籤」（例如：投射物、範圍、法術、攻擊、火焰、冰冷等）。輔助技能只會對具有匹配標籤的主動技能生效。例如，「投射物傷害」輔助無法加成「旋風斬」這個攻擊技能。仔細閱讀技能說明是關鍵。

### 3. 連結槽位分配不當


許多玩家會將所有輔助技能都連結到主要輸出技能上，導致其他功能性技能（如位移、光環、詛咒）沒有足夠的連結，無法發揮最大效果。合理的分配多個技能組的連結槽位，是高階BD的標誌。

### 4. 不了解魔力消耗與回復


強力的技能連結往往伴隨著高額的魔力消耗。如果魔力回復不足，你會發現技能放不出來，嚴重影響戰鬥節奏。應搭配「減少魔力消耗」輔助、魔力回復天賦、裝備詞綴或契約來解決。

### 2. 卡池类型与周期分析


*   **常驻卡池：** 包含基础灵犀，随时可抽，但通常不建议大量投入，除非是新手初期急需补充灵犀。
*   **限定活动卡池：** 游戏核心，围绕特定活动或节庆推出，包含限定灵犀，活动结束后通常会进入常驻池，但机率会大幅降低，甚至不再复刻。这是玩家们最需要关注的卡池类型。
*   **双人限定卡池：** 近年来，官方也频繁推出两位男主同池的限定卡池，这类卡池通常有独立的保底机制，且指定角色机率较低，抽取前务必评估自身资源与需求。

### 3. 抽卡道具与资源：钻石、抽卡券与活动代币


*   **北极星（钻石）：** 最主要的通用抽卡资源，可透过日常任务、活动奖励、储值获得。

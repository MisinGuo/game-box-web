*   **初期专精一种资源：** 新手时期，建议集中发展一种高产资源（如粮食或木材），透过公会互助或市场交易补足其他资源。

*   **步兵克制骑兵**
*   **骑兵克制弓兵**
*   **弓兵克制步兵**
*   **攻城器** 主要用于攻城，对其他兵种伤害较低，但对城墙和陷阱有额外伤害。

**新手训练建议：**

1.  **均衡发展：** 初期应均衡训练T1或T2士兵，确保各兵种数量相当，以应对不同战况。避免只训练单一兵种。
2.  **优先训练T2：** T2士兵是新手期性价比最高的选择，具备一定战力且训练成本相对较低。
3.  **少量攻城器：** 攻城器训练少量即可，主要用于攻打无兵的死城。在有兵防守的城堡战中，应避免大量使用。

*   **侦查是第一步：** 攻击前务必使用侦查，了解敌方兵力、英雄、陷阱和资源储备。

*   **加入集结：** 作为新手，最安全的作战方式是加入公会盟友的集结。在集结中，您的士兵会与盟友的士兵一同出征，分担风险。

*   **防御罩（盾牌）：** 最有效的防御手段，尤其在您不活跃或有大量资源时，务必开启。

*   **陷阱与城墙：** 升级城墙并建造陷阱，但对高阶玩家效果有限，主要用于消耗敌人T1/T2兵力。
*   **假集结：** 在自己城堡发起一个长时间的集结，让敌人误以为您有大量兵力在集结中，从而犹豫不决。

#### 2.2.4 英雄技能的精准运用

*   **出征英雄：** 选择对应兵种加成的军事英雄带队出征。
*   **内政英雄：** 在城堡中放置内政英雄，享受其资源生产、训练加速等加成。
*   **战斗技能：** 在战斗中，适时使用英雄的战斗技能，如治疗、眩晕、范围伤害等，能扭转战局。

### 2. 画质预设与细部调整


如果时间有限，可以先从游戏提供的「画质预设」入手。通常「中等」或「低」预设能大幅提升帧数。若要进一步精准优化，请参考以下细部设定：

*   **纹理品质 (Texture Quality)：** 影响游戏中物体表面的细致度。如果显示卡记忆体 (VRAM) 较少（如 4GB 以下），建议设为「中」或「低」。这是影响 VRAM 占用最主要的设定之一。
*   **阴影品质 (Shadow Quality)：** 对效能影响巨大。从「高」降至「中」或「低」能带来显著的帧数提升，且视觉差异相对较小。
*   **抗锯齿 (Anti-Aliasing)：** 消除画面边缘的锯齿感。 MSAA、TAA 等方法对效能消耗较大。若追求极致流畅，可关闭或选择效能较低的选项（如 FXAA）。若游戏支援 DLSS/FSR 等技术，请务必开启并设定为「效能」或「平衡」模式，这能同时提升帧数与画质。
*   **环境光遮蔽 (Ambient Occlusion, AO)：** 让物体接触面产生阴影，增加立体感。对效能影响较大，可设为「中」或「低」，甚至关闭。
*   **体积光/体积雾 (Volumetric Lighting/Fog)：** 营造光线穿透或雾气弥漫的效果，非常耗效能。建议设为「低」或关闭。
*   **反射品质 (Reflection Quality)：** 影响水面、金属等物体的反射效果。可适度降低。
*   **粒子效果 (Particle Effects)：** 爆炸、魔法等特效的细致度。在战斗密集时可能造成卡顿，可设为「中」或「低」。
*   **视野距离 (View Distance)：** 影响远处物体的显示细节。过高会增加 CPU 和 GPU 负担，可适度降低。
*   **垂直同步 (V-Sync)：** 锁定帧数与萤幕刷新率同步，可避免画面撕裂。但若帧数低于刷新率，可能导致输入延迟或帧数锁定在更低的倍数。建议若帧数稳定且高于刷新率，可开启；若帧数不稳，可关闭并搭配显示器 G-Sync/FreeSync 功能。

### 3. 帧数限制 (FPS Limit)


如果你的显示器刷新率是 60Hz，而游戏帧数经常飙到 100 ，可以考虑将帧数限制在 60 或 75 帧。这能减轻显示卡负担，降低温度，并让帧数输出更稳定，减少波动造成的卡顿感。

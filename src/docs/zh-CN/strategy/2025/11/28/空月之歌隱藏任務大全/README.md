*   **2025年更新：** 任务仅在游戏时间「清晨」且「有雾」的天气下才能触发。

1.  **发现线索：** 在迷雾沼泽入口处（座标：X:50, Y:550），你会看到一顶破旧的帐篷。靠近后，帐篷内会传出微弱的求救声。
2.  **进入迷雾：** 点击帐篷，选择「进入」。你将被传送到一个被浓雾笼罩的特殊区域。
3.  **追踪痕迹：** 在迷雾中，你需要按照地上的脚印和断裂的树枝痕迹前进。沿途会遇到一些幻象生物，它们不会攻击，但会阻碍视线。
4.  **寻找探险队员：** 最终你会在一个古老的祭坛旁发现三具探险队员的尸体。其中一具尸体的手中紧握着一个「神秘的日记本」。
5.  **解读日记：** 阅读日记本，里面记载了探险队员们在迷雾中遭遇的恐怖经历，以及他们发现的一个「远古符文」。
6.  **启动符文：** 根据日记本的提示，你需要将「精灵之尘」洒在祭坛中央的「远古符文」上。符文会发出绿色的光芒，迷雾逐渐散去。

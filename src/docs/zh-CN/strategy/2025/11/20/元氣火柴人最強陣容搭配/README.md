### 1. 角色替換與靈活變通


即使是「最強陣容」，也可能因個人角色池深度而有所調整。當你缺少特定核心角色時，可以考慮以下替換原則：

*   **坦克：** 尋找具備高生命、高防禦、帶有控制或護盾技能的角色。
*   **輸出：** 尋找同類型（物理/魔法）、同定位（單體/群體）且傷害接近的角色。
*   **輔助：** 尋找具備治療、護盾、增益或淨化能力的角色。
*   **控制：** 尋找帶有硬控（暈眩、沉默）的角色。

### 2. 裝備與技能搭配


每個火柴人角色的裝備選擇和技能升級都至關重要。

*   **裝備：** 根據角色定位選擇主屬性（例如：輸出選攻擊/暴擊，坦克選生命/防禦，輔助選治療量/冷卻縮減），並搭配合適的套裝效果。
*   **技能：** 優先升級核心輸出技能、關鍵控制技能和輔助治療/護盾技能。

### 3. 站位策略


合理的站位能最大化隊伍優勢，最小化敵方威脅。

*   **前排：** 坦克或半坦角色，承擔主要傷害。
*   **中排：** 近戰輸出、控制型輔助，能兼顧輸出與支援。
*   **後排：** 遠程輸出、治療型輔助，確保輸出環境和生存。
*   **針對性調整：** 根據敵方陣容，調整刺客的切入位置，或將脆皮角色放在不易被集火的位置。

### 4. 資源分配與角色培養


資源有限時，應優先投資於核心輸出和核心坦克，確保隊伍的進攻能力和生存能力。輔助角色在滿足基本功能後，再逐步提升。

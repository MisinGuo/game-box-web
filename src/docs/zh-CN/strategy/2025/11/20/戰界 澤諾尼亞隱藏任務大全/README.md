### 四、《戰界澤諾尼亞》隱藏任務詳盡攻略 (2025年最新整理)


以下是目前社群中整理出的主要隱藏任務列表，涵蓋了觸發條件、詳細步驟與最終獎勵。請注意，部分任務可能因遊戲版本更新而略有差異，本攻略將以2025年最新版本為準。

#### 1. 任務：失落的吟遊詩人

*   **觸發地點：** 艾爾文村（Elvin Village）東北方，被遺棄的磨坊。
*   **觸發條件：** 在遊戲時間的夜晚（20:00-04:00），進入磨坊，會發現一個奄奄一息的吟遊詩人。
*   **任務步驟：**
    1.  與吟遊詩人對話，他會請求你尋找他失落的豎琴。
    2.  豎琴被哥布林盜賊團偷走，藏在「寂靜森林」深處的哥布林營地寶箱中。
    3.  前往寂靜森林，擊敗營地內的「哥布林首領」，打開其身後的木箱獲得「破舊的豎琴」。
    4.  將豎琴帶回給吟遊詩人。他會因為豎琴損壞而請求你尋找「精靈之弦」。
    5.  「精靈之弦」可在「精靈之森」中心地帶，擊敗「魅惑妖精」後有低機率掉落。
    6.  將精靈之弦交給吟遊詩人，他會修復豎琴並為你演奏一曲。
*   **任務獎勵：**
    *   經驗值：5000 XP
    *   金幣：2000 G
    *   物品：「吟遊詩人的祝福」— 永久提升少量移動速度與暴擊率（被動效果）。
    *   稱號：「樂之守護者」

#### 2. 任務：被遺忘的歷史學者

*   **觸發地點：** 聖光城（Holy City）圖書館二樓，最深處書架後方。
*   **觸發條件：** 在圖書館二樓，點擊最深處的書架（無任何互動提示），連續點擊五次後，書架會移動，露出一個隱藏的房間。進入房間與「歷史學者艾倫」對話。
*   **任務步驟：**
    1.  艾倫會請求你幫他找回三本被盜的古籍：《黎明紀元》、《黑暗契約》、《諸神黃昏》。
    2.  《黎明紀元》：位於「沙海遺跡」底層，擊敗「沙蟲女王」後掉落。
    3.  《黑暗契約》：位於「詛咒沼澤」深處，一個被藤蔓纏繞的破舊祭壇上。需要先擊敗附近遊蕩的「沼澤守衛」。
    4.  《諸神黃昏》：位於「寒冰洞窟」最深處的冰晶寶箱中，寶箱需要「融冰火把」才能打開（融冰火把可在聖光城雜貨店購買）。
    5.  將三本古籍帶回給艾倫。
*   **任務獎勵：**
    *   經驗值：7500 XP
    *   金幣：3500 G
    *   物品：「學者之眼」— 永久提升少量智力與精神（被動效果）。
    *   稀有附魔石：隨機一個高級屬性附魔石。

#### 3. 任務：礦工的遺願

*   **觸發地點：** 灰燼礦坑（Ashen Mine）深處，一個被落石堵住的舊礦道。
*   **觸發條件：** 在舊礦道盡頭，調查地上一個閃爍的「破舊礦燈」，會觸發與「礦工幽靈」的對話。
*   **任務步驟：**
    1.  礦工幽靈會講述他被困的經歷，並請求你將他的「傳家寶」帶給他在「黑市」的家人。
    2.  傳家寶就在礦工幽靈身旁的一個小石堆中，拾取「生鏽的項鍊」。
    3.  前往黑市（位於聖光城下水道入口附近，需要特定時間或完成前置任務才能進入），找到NPC「神秘商人艾德」。
    4.  與艾德對話，選擇「關於礦工的遺願」，將生鏽的項鍊交給他。
    5.  艾德會給你一個「回報包裹」，讓你帶回給礦工幽靈。
    6.  返回灰燼礦坑，將包裹交給礦工幽靈，他會安息並消失。
*   **任務獎勵：**
    *   經驗值：6000 XP
    *   金幣：3000 G
    *   物品：「艾德的謝禮」— 內含大量金幣、稀有礦石與一個隨機史詩級材料。

#### 4. 任務：迷失的靈魂

*   **觸發地點：** 暮光森林（Twilight Forest）中央，一棵巨大的枯樹下。
*   **觸發條件：** 在枯樹下使用任何「治療藥水」或「復活卷軸」，會觸發「迷失的靈魂」現身。
*   **任務步驟：**
    1.  迷失的靈魂會低語著尋求安寧，請求你幫助他找回「聖潔之露」。
    2.  「聖潔之露」可在「月光湖畔」的「月光祭壇」處獲得。但祭壇被「暗影魔物」守護。
    3.  擊敗月光湖畔的「暗影魔物領主」，然後在月光祭壇上互動，即可收集到「聖潔之露」。
    4.  將聖潔之露帶回給迷失的靈魂，他會得到淨化並升天。
*   **任務獎勵：**
    *   經驗值：4500 XP
    *   金幣：2500 G
    *   物品：「靈魂結晶」— 可用於強化裝備，大幅提升成功率。
    *   永久增益：「靈魂庇護」— 提升對黑暗屬性傷害的抗性。

#### 5. 任務：守護者的考驗

*   **觸發地點：** 龍脊山脈（Dragon's Back Mountains）最高峰，一個被雷電環繞的祭壇。
*   **觸發條件：** 達到角色等級50級，並在遊戲時間的白天（08:00-18:00）觸摸祭壇。會召喚出「山脈守護者」。
*   **任務步驟：**
    1.  山脈守護者會對你發起挑戰，要求你證明自己的力量。
    2.  擊敗山脈守護者（此為一場高難度單人BOSS戰，建議準備充足）。
    3.  擊敗後，守護者會認可你的力量，並給予你獎勵。
*   **任務獎勵：**
    *   經驗值：10000 XP
    *   金幣：5000 G
    *   物品：「守護者之證」— 可用於兌換一件傳說級裝備（在聖光城裝備商人處兌換）。
    *   稱號：「山脈征服者」

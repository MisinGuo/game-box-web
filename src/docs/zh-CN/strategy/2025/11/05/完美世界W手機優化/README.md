游戏内设定是影响性能最直接的因素。以下是针对不同手机性能等级的建议：

### 1. 画质设定：平衡视觉与效能


*   **画面品质：**

### 2. 帧率设定：稳定是王道


*   **帧率上限：**
    *   **建议：** 设定为「60 FPS」。这是目前主流手机萤幕的刷新率，能提供最流畅的视觉体验。
    *   **若手机发热或卡顿：** 尝试降至「45 FPS」或「30 FPS」。虽然画面流畅度会有所下降，但能显著降低功耗与发热，提升稳定性。
    *   **注意：** 即使手机支援120Hz刷新率，游戏内若无对应选项，设定60 FPS即可。过高的帧率设定若手机无法稳定输出，反而会造成画面撕裂或不稳定。

### 3. 特效与细节设定：精打细算

    *   **核心优化点！ ** 在野外挂机或参与大型活动时，将「同屏人数」设定为「少量」或「极少」。这能大幅减少CPU与GPU的渲染压力。
*   **粒子效果、技能特效、阴影、反射：**
    *   **建议：** 除了「技能特效」可依个人喜好保留外，其他如「粒子效果」、「阴影」、「反射」等，建议全部关闭或设定为「低」。这些特效对性能影响巨大。
*   **抗锯齿：**
    *   **建议：** 关闭。抗锯齿能让画面边缘更平滑，但对性能消耗极大。在手机萤幕上，其视觉提升效果不如PC明显。

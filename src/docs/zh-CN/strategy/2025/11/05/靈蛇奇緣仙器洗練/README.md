### 洗练材料与消耗


进行仙器洗练主要需要以下材料：

*   **洗练石**：最基础的洗练材料，消耗量大。通常透过日常任务、副本掉落、活动奖励或商城购买获得。
*   **高级洗练石/锁定石**：用于锁定已洗出的满意属性，确保下次洗练时该属性不被改变。是精准洗练不可或缺的道具。
*   **金币/元宝**：每次洗练或锁定属性均需消耗一定量的金币或元宝。

### 仙器属性词条品质与类型


洗练出的属性词条通常分为不同品质（如绿、蓝、紫、橙、红），品质越高，属性加成效果越好。词条类型则包罗万象，大致可分为：

*   **攻击类**：物理攻击、法术攻击、暴击率、暴击伤害、穿透等。
*   **防御类**：物理防御、法术防御、生命值、免伤、韧性等。
*   **特殊类**：命中、闪避、控制抗性、技能冷却缩减、属性伤害加成等。

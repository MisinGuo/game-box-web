### 羁绊触发条件与类型：2025年最新概览


2025年《极速三国》的羁绊触发条件与类型更加丰富，主要分为以下几种：

*   **武将羁绊：** 最常见的类型，当特定武将同时上阵或持有时触发。例如「桃园结义」（刘备、关羽、张飞）或「五虎上将」。 2025年新增了一些跨阵营的历史羁绊，增加了搭配弹性。

*   **缘分羁绊：** 某些武将之间因历史典故或特殊关系而产生的羁绊，不一定需要同时上阵，可能仅需拥有即可激活。

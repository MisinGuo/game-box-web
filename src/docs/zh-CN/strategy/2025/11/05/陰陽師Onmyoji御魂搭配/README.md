### 1. 御魂位置与主属性

御魂分为六个位置（1号至6号），每个位置的主属性是固定的，但副属性则随机生成。

*   **1号位：攻击** (固定)
*   **2号位：速度、攻击加成、生命加成** (可选)
*   **3号位：防御** (固定)
*   **4号位：攻击加成、生命加成、防御加成、效果命中、效果抵抗** (可选)
*   **5号位：生命** (固定)
*   **6号位：暴击、暴击伤害、攻击加成、生命加成、防御加成** (可选)

**关键提示：** 2、4、6号位是御魂搭配的核心，尤其2号位的速度、4号位的效果命中/抵抗以及6号位的暴击/暴击伤害，直接影响式神的定位与输出能力。

### 2. 御魂套装效果


御魂套装分为两件套和四件套效果。

*   **兩件套效果：** 提供固定的屬性加成，如攻擊加成15%、暴擊率15%、生命加成15%等。通常用於湊散件或強化特定屬性。

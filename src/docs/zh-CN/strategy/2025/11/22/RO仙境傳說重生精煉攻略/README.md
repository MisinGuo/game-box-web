### Q1：我應該先精煉武器還是防具？


A1：通常建議優先精煉「武器」。武器精煉帶來的攻擊力提升最直接且顯著，能大幅提高打怪效率和輸出能力。之後再根據需求精煉防具和飾品。

### Q2：我沒有很多Zeny，怎麼開始精煉？


A2：從低成本的裝備開始，例如便宜的武器或防具，先衝到安全精煉等級+4。累積一些經驗和Zeny後，再考慮衝刺更高級的裝備。參與遊戲內的賺錢活動、打寶販賣材料也是累積資本的好方法。

A3：不一定。在安全精煉等級（通常+4）以下失敗，只會精煉等級歸零。超過安全等級後，失敗才有「機率」毀損。使用高密度材料可以保證失敗時裝備不毀損，僅精煉等級歸零。

*   **高密度材料：** 成功率更高，且在失敗時「保證」裝備不會毀損，只會精煉等級歸零。是衝刺高精煉裝備的首選。

### Q5：精煉有什麼「吉時」或「秘訣」嗎？


A5：遊戲內精煉的成功率是基於機率計算，沒有官方證實的「吉時」或「秘訣」。所謂的「墊裝」、「人少時段」等更多是玩家的心理安慰或經驗累積。最實際的秘訣還是：準備充足的材料、控制風險、量力而為。

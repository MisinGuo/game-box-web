### 1. 进入洗练介面


通常在主城NPC处（如铁匠铺、附魔师）或背包介面中找到「装备洗练」选项。选择你想要洗练的装备。

### 2. 选择洗练模式


《龙魂旅人》通常提供两种洗练模式：

*   **普通洗练：** 消耗基础洗练石和金币，随机重置装备上所有或部分可洗练属性。此模式成本较低，但随机性高。
*   **精准洗练（或锁定洗练）：** 消耗更高级的洗练材料（如「龙魂精华」、「锁定符文」）和更多金币。此模式允许你锁定部分满意的属性，只重置未锁定的属性，大幅降低随机性，但成本也更高。 **在2025年的版本中，精准洗练是中后期玩家打造毕业装备的必经之路。 **

### 3. 属性词条类型与品质


每件装备的可洗练属性槽位数量固定，但能洗练出的属性类型和数值范围则由装备的「品阶」和「稀有度」决定。

*   **主属性与次属性：** 某些装备部位（如武器、胸甲）会有固定的主属性，而洗练主要针对次级属性。
*   **属性池：** 每种装备类型都有其专属的属性池。例如，法师的法杖不会洗出战士的「重击伤害」。
*   **数值范围：** 即使是相同的属性，其数值也会在一个范围内浮动。洗练的目标就是洗出接近或达到上限的「极品词条」。

### 4. 洗练材料与获取途径


*   **基础洗练石：** 最常见的材料，透过日常任务、副本掉落、商店购买获得。
*   **龙魂精华：** 用于精准洗练或提升洗练成功率，通常来自高难度副本、世界BOSS、活动奖励或商城限定礼包。
*   **锁定符文：** 锁定属性专用，获取难度较高，多数来自PVP奖励、特殊活动或限时商城。
*   **金币：** 洗练操作的基础消耗。

**2025年提示：** 随着游戏经济的发展，金币在洗练中的消耗比例可能有所调整。同时，官方在2025年可能推出新的活动机制，提供更多获取稀有洗练材料的机会，请密切关注游戏内公告。

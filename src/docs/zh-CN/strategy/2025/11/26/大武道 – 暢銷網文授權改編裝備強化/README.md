### 1. 装备强化种类与效果

《大武道》的装备强化主要分为以下几种，每种都有其独特的战力提升路径：

*   **基础强化（等级提升）**：消耗「强化石」与「金币」，直接提升装备的基本属性（攻击、防御、生命等）。这是最直接且最基础的强化方式，等级越高，属性加成越显著。
*   **精炼（品质提升）**：消耗「精炼石」与「金币」，提升装备的品质等级，通常会解锁或增强装备的附加属性，甚至提升套装效果。精炼是后期追求极致战力的关键。
*   **镶嵌（宝石镶嵌）**：在装备的宝石孔中镶嵌不同种类的宝石，赋予装备额外的属性加成，如暴击、命中、闪避、属性攻击等。宝石可升级，且不同宝石组合能产生协同效应。
*   **突破（阶级跃升）**：当装备达到一定强化等级或品质后，可进行突破。突破通常需要稀有材料，成功后会大幅提升装备属性上限，并可能改变装备外观或解锁特殊技能。

*   **强化石**：用于基础强化，分为不同等级，对应不同装备等级。
*   **精炼石**：用于精炼，同样有等级之分。
*   **宝石**：各类属性宝石，通过副本、活动、商城获取。
*   **突破材料**：通常是稀有副本掉落、活动奖励或合成获得的专属材料。

*   **误区一：盲目追求单件装备的极限强化**
    *   **问题**：将所有资源投入到一件装备上，导致其他装备属性过低，整体战力不均衡。

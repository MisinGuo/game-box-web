### 什么是属性相克？

*   **地属性 (Earth)**
*   **圣属性 (Holy)**
*   **暗属性 (Dark)**
*   **毒属性 (Poison)**

### 相克倍率与计算方式


当属性相克关系成立时，伤害会依据倍率进行增减。通常的倍率为：

*   **克制屬性：**+造成+**150%**+ 傷害+(即額外50%傷害)。
*   **被克制屬性：**+造成+**50 %**+傷害+(即減少50%傷害)。
*   **無關屬性：**+造成+**100%**+傷害。

值得注意的是，部分技能或装备可能会调整这些倍率，但基础逻辑不变。精确的伤害计算还需考虑攻击力、防御力、减伤、增伤等因素，但属性倍率是其中一个最显著的乘法因子。

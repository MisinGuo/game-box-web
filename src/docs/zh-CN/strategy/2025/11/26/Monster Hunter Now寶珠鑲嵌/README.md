### 什么是宝珠？

![游戏场景](https://oss.2025.11.5yxy5.com/images/2025/11/26/Monster%20Hunter%20Now%E5%AF%B6%E7%8F%A0%E9%91%B2%E5%B5%8C/image_2_1764102530.jpg)

宝珠是蕴含特定技能点数的道具，每个宝珠都对应着游戏中的某项技能（例如：攻击、看破、弱点特效等）。宝珠通常有不同的稀有度与等级，稀有度越高、等级越高的宝珠，提供的技能点数也越多。

### 宝珠的种类与稀有度


宝珠依据其提供的技能种类繁多，从基础的攻击、防御，到进阶的属性强化、异常状态耐性，应有尽有。它们的稀有度通常分为R1到R5，稀有度越高（如R5），获取难度越大，但提供的技能点数或特殊效果也越强。

### 镶嵌系统运作原理


每件装备（武器、头盔、胸甲、腕甲、腰甲、腿甲）都可能带有数量不等的「宝珠孔位」。这些孔位有大小之分，通常以数字（如1、2、3）表示其可容纳宝珠的等级。例如，一个「等级2」的孔位可以镶嵌等级1或等级2的宝珠。当镶嵌的宝珠累积足够的技能点数时，对应的技能便会被激活，提升猎人的作战能力。

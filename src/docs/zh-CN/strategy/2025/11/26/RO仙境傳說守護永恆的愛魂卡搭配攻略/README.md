### 1. 确立角色定位与主要属性

*   **物理输出 (ADL/爆击流/技能流)：** 优先选择增加物理攻击、爆击、攻速、穿透、无视防御的卡片。

### 2. PVE、PVP、GVG场景区分搭配

*   **PVE (日常/副本/MVP)：**
    *   **日常挂机：** 以效率为主，选择增加对应种族/属性魔物伤害、范围伤害、吸血、减少消耗的卡片。
    *   **副本/MVP：** 针对性最强，研究BOSS的种族、属性、技能，选择能克制或大幅提升生存/输出的卡片。例如，对恶魔种族BOSS使用恶魔种族加伤卡。
*   **PVP (竞技场/战场)：**
    *   **生存与控制：** 增加生命、防御、免伤、抗异常状态、减少对方移速/攻速的卡片。
    *   **爆发与穿透：** 针对玩家的减伤与防御，选择增加穿透、无视防御、爆击伤害的卡片，追求瞬间击杀。
*   **GVG (公会战)：**
    *   **团队协作：** 除了个人战力，也要考虑如何支援队友。例如，坦克需要更强的生存能力吸引火力，输出需要更高的范围伤害或单点爆发。

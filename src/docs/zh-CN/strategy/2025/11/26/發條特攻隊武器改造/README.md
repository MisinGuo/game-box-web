### 1. 核心模组（Core Modules）


这是武器改造的基石，通常决定武器的主要属性加成，例如基础伤害、射速、暴击率等。 2025年新增的「原型核心模组」拥有更强大的基础属性，但通常伴随着特定的负面效果，需要玩家权衡利弊。

*   **实战建议**：优先选择与武器类型相符，并能放大其优势的核心模组。例如，对于射速快的冲锋枪，可选择提升暴击伤害或元素触发机率的核心。

### 2. 辅助晶片（Auxiliary Chips）


辅助晶片提供多样化的次级属性加成，如装填速度、稳定性、弹匣容量、后座力控制或特殊效果（如穿甲、吸血）。 2025年版本引入了「联动晶片」，当特定晶片组合装备时，可触发额外的隐藏效果。

*   **实战建议**：联动晶片是2025年版本的一大亮点。在规划改造时，应考虑如何搭配晶片以激活这些强大的联动效果，这往往能带来超越单一晶片加成的效益。

### 3. 能量导管（Energy Conduits）


能量导管主要影响武器的能量消耗、技能冷却缩减或特殊能量回复速度。对于依赖技能或能量攻击的武器来说，导管的选择至关重要。 2025年更新后，部分导管现在能提供「超载」效果，短时间内极大提升武器性能。

*   **实战建议**：如果你偏好频繁使用武器技能，或武器本身有高能量消耗，投资于能量效率导管是明智之举。超载导管则适合爆发型打法。

### 4. 外壳强化（Casing Reinforcements）


外壳强化偏向防御或功能性，如提升武器耐久度、降低受损机率、增加携弹量，甚至提供微量的玩家护甲加成。 2025年的「反应式外壳」能在受到特定伤害时提供短暂的抗性。

*   **实战建议**：外壳强化通常是最后考虑的选项，但在高难度任务中，其提供的生存能力或功能性不容小觑。例如，在持久战中增加弹药携带量可省去频繁补给的麻烦。

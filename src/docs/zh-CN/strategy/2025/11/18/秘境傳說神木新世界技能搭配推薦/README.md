#### 1.1 核心技能類型與機制

遊戲中的技能大致可分為以下幾類：

*   **主動技能：** 需手動施放，是戰鬥中的主要輸出與控制手段。它們通常有冷卻時間和魔力消耗。
*   **被動技能：** 學習後自動生效，提供常駐的屬性加成或特殊效果，是提升角色基礎能力的關鍵。
*   **輔助技能：** 主要用於增益隊友、減益敵人或提供治療與護盾，在團隊戰中不可或缺。
*   **職業專屬奧義：** 各職業的終極技能，通常擁有華麗的特效與毀滅性的威力，是扭轉戰局的殺手鐧。

#### 3.1 裝備與技能的協同效應

2025年版本中，許多傳說級裝備和套裝效果都與特定技能或技能類型產生強大的協同效應。例如，某些裝備會「增加XX技能的傷害15%」或「施放XX技能後，下一次攻擊造成額外傷害」。在選擇技能搭配時，務必檢視你當前的核心裝備，並圍繞其特效來優化你的技能樹，這樣才能發揮出1+1>2的效果。

#### 3.2 夥伴/寵物技能搭配考量

《秘境傳說：神木新世界》的夥伴與寵物系統在2025年得到了強化，它們的技能不再是單純的輔助，而是能夠與玩家技能形成戰術配合。例如，如果你的寵物有範圍聚怪技能，你的法師就可以搭配高範圍爆發技能，實現快速清場；如果你的夥伴能提供護盾或治療，你的輸出職業就能更放心地進行輸出。

#### 3.3 針對不同副本與BOSS的靈活調整

沒有一套技能搭配是萬能的。資深玩家的經驗告訴我們，針對不同的副本機制和BOSS特性，靈活調整技能是通關的關鍵。例如：

*   **多目標BOSS戰：** 優先選擇帶有範圍傷害或能快速轉火的技能。
*   **單體高壓BOSS戰：** 側重於高單體爆發、減傷或驅散技能。

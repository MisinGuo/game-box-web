### 途徑一：基礎強化（升級與進階）


這是最直接的強化方式，透過消耗金幣和強化石，提升裝備的基礎數值。

*   **操作方式：** 進入裝備介面，選擇欲強化的裝備，點擊「強化」按鈕。
*   **材料消耗：** 主要為金幣和各類強化石。強化石通常分為初級、中級、高級，對應不同品質和等級的裝備。
*   **策略建議：**
    *   **優先級：** 優先強化主力輸出武將的武器，其次是防禦型武將的防具和生命裝備。
    *   **平均提升：** 在遊戲前期，建議主力隊伍的裝備保持相對平均的強化等級，以確保整體戰力穩定。進入中後期，可將資源集中於核心武將的關鍵裝備。
    *   **適時替換：** 當獲得更高品質的裝備時，務必及時替換並將舊裝備的強化等級「傳承」給新裝備（若有此功能），避免重複投入資源。

### 途徑二：寶石鑲嵌與洗練


寶石鑲嵌能為裝備額外增加屬性，而洗練則能讓裝備獲得隨機的附加屬性，是打造特色裝備的重要環節。

*   **寶石鑲嵌：**
    *   **寶石種類：** 攻擊、防禦、生命、暴擊、抗暴、命中、閃避等各類寶石，提供不同的屬性加成。
    *   **鑲嵌策略：** 根據武將定位選擇。例如，輸出型武將優先鑲嵌攻擊、暴擊寶石；防禦型武將則選擇生命、防禦、抗暴寶石。
    *   **寶石升級：** 寶石本身也可以透過消耗同類寶石或專屬材料進行升級，提升其加成效果。
*   **裝備洗練：**
    *   **機制：** 消耗洗練石或鑽石，隨機刷新裝備的附加屬性。
    *   **追求目標：** 針對性地洗練出與武將技能、陣容搭配最契合的屬性。例如，為具備追擊能力的武將洗練「增傷」或「速度」屬性。
    *   **鎖定屬性：** 通常遊戲會提供鎖定功能，讓玩家可以保留滿意的屬性，只刷新其他屬性，但會增加消耗。

### 途徑三：精煉與突破（進階強化）


這兩種是提升裝備上限與質變的關鍵系統，通常在遊戲中後期才開放或變得重要。

*   **裝備精煉：**
    *   **效果：** 進一步提升裝備的基礎屬性百分比，或解鎖額外被動效果。
    *   **材料：** 消耗精煉石和大量金幣，精煉石通常較為稀有。
    *   **策略：** 精煉的成本較高，建議優先精煉主力武將的橙色及紅色品質裝備。
*   **裝備突破：**
    *   **效果：** 突破會大幅提升裝備的星級或品質上限，並帶來巨大的屬性飛躍，甚至改變裝備外觀。
    *   **材料：** 消耗專屬的突破石、同名裝備碎片或特定稀有材料。
    *   **策略：** 突破是遊戲中後期提升戰力的重要手段。由於材料獲取難度高，應集中資源優先突破核心武將的核心裝備（如專屬武器、套裝）。

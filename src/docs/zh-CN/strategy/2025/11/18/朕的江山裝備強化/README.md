### 1. 等級強化（升級）

這是最基礎的強化方式，消耗銅幣和強化石提升裝備的基礎屬性（攻擊、防禦、兵力等）。
*   **關鍵點：** 強化等級上限受裝備品質和武將等級限制。優先強化主力武將的核心裝備，確保資源不浪費。

### 2. 進階（突破）

當裝備達到一定強化等級後，可消耗特定材料進行進階，提升裝備品質（例如從藍色進階到紫色），並解鎖更高的強化上限與潛在屬性。

### 3. 精煉（洗練）

精煉功能允許玩家消耗精煉石或元寶，隨機改變裝備的附加屬性。這是在基礎屬性之外，進一步客製化裝備以符合武將特性的重要手段。
*   **關鍵點：** 追求與武將技能、定位高度契合的屬性，例如輸出武將追求暴擊、增傷，肉盾武將追求減傷、免傷。

### 4. 寶石鑲嵌

裝備上通常設有寶石孔位，可鑲嵌不同種類的寶石以獲得額外屬性加成。寶石可升級，且不同顏色寶石提供不同屬性。
*   **關鍵點：** 根據武將類型選擇合適的寶石。例如，物理輸出武將鑲嵌物攻寶石，法術武將鑲嵌法攻寶石，坦克武將鑲嵌生命、防禦寶石。高階寶石合成不易，初期可先以低階寶石填滿孔位。

### 5. 覺醒與銘刻（2025年新增或優化機制）

根據2025年最新版本資訊，部分稀有裝備引入了「覺醒」或「銘刻」系統。

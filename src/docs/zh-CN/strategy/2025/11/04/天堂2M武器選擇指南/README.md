### 强化系统：稳定提升基础属性


*   **概念：** 使用「强化卷轴」提升武器的强化等级（ 1、 2...），每次成功强化都能显著提升武器的基础攻击力与命中。
*   **风险：** 强化等级越高，失败机率越大，失败可能导致武器损毁或强化等级下降。
*   **策略：**
    *   **安全强化：** 在达到一定等级前（例如 6或 7），失败不会损毁武器。
    *   **祝福卷轴：** 使用祝福的强化卷轴可提高成功率或在失败时保护武器。
    *   **稀有度：** 高稀有度武器的强化收益更大，但强化成本也更高。

### 属性附魔：针对性强化输出


*   **概念：** 为武器附上火、水、风、地四种属性之一。攻击时，若目标为相克属性，将造成额外伤害。

*   **劣势：** 附魔后无法更改，除非使用「属性变更卷轴」。
*   **策略：**

    *   **PVP：** 考虑当前伺服器中主流职业的属性抗性，或选择通用性较强的属性。

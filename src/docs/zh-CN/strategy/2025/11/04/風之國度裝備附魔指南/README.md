附魔效果通常分为不同的等级与品质，等级越高、品质越好的附魔，所提供的属性加成也越显著。

*   **强化石/精华：** 用于提升附魔的成功率或保底失败不掉级。
*   **游戏币（银贝/金贝）：** 每次附魔操作都需要消耗一定量的游戏币。
*   **特殊材料：** 部分稀有或高阶附魔可能需要特定副本掉落的稀有材料。

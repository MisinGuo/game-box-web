#### 理解装备基础属性与套装效果

所有装备都包含主属性与副属性，并可能形成套装效果。
*   **主属性 (Main Stat)：** 武器固定为攻击力，头盔固定为生命力，铠甲固定为防御力。项链、戒指、鞋子的主属性则有多种选择，是装备搭配的核心考量点。例如，攻击型英雄的项链常选攻击百分比，鞋子常选速度。

*   **套装效果 (Set Effect)：** 穿戴指定数量（通常为2件或4件）的同类型装备，可激活额外加成。常见套装包括：
    *   **攻擊套 (Attack Set)：** 攻擊力+35% (4件)
    *   **速度套 (Speed Set)：** 速度+25% (4件)
    *   **暴擊套 (Critical Set)：** 暴擊率+12% (2件)
    *   **生命套 (Health Set)：** 生命力+18% (2件)
    *   **防禦套 (Defense Set)：** 防禦力+20% (2件)
    *   **命中套 (Hit Set)：** 命中率+20% (2件)
    *   **抵抗套 (Resist Set)：** 效果抵抗+20% (2件)
    *   **反擊套 (Counter Set)：** 被攻擊時30%機率反擊 (4件)
    *   **憤怒套 (Rage Set)：** 對精英/Boss傷害+30% (4件，PVE特化)
    *   **貫穿套 (Penetration Set)：** 忽略防禦力+15% (2件，PVP爆發特化)
    *   **免疫套 (Immunity Set)：** 开局获得1回合免疫，防止被控 (2件，PVP常见)

#### 优先级：主属性与副属性的抉择

在《第七史诗》的装备选择中，主属性决定了装备的基本方向，而副属性则决定了装备的上限与精准度。
*   **主属性优先：** 对于项链、戒指、鞋子，务必根据英雄需求选择正确的主属性。例如，输出型英雄追求攻击百分比、暴击伤害百分比、速度；辅助型英雄追求生命百分比、防御百分比、效果抵抗百分比、速度。
*   **副属性洗练：** 即使主属性正确，如果副属性不佳，这件装备的潜力也会大打折扣。理想的副属性应与英雄定位高度契合，例如输出英雄偏好暴击率、暴击伤害、速度、攻击力；坦克或辅助英雄偏好生命力、防御力、效果抵抗、速度。强化装备时，副属性「歪」到不必要的属性是常见现象，这就需要透过装备洗练石来调整。

#### PVE（讨伐、深渊、迷宫）装备推荐

PVE内容更注重稳定输出、生存能力与特定机制应对。
*   **讨伐队伍：**
    *   **输出位：** 通常追求高攻击、高暴击伤害。愤怒套搭配贯穿套是讨伐Boss的顶级选择，其次是攻击套 暴击套。主属性多为攻击百分比项链、暴击伤害百分比戒指、速度或攻击百分比鞋子。
    *   **破甲位/辅助位：** 速度套 命中套或生命套 抵抗套，确保能先手施加减益效果或提供稳定增益。
*   **深渊/迷宫：**
    *   **生存优先：** 由于战线较长，生命套、防御套、抵抗套的组合更受欢迎。免疫套也能提供开局保护。
    *   **控制与续航：** 确保辅助英雄的速度与效果命中，能稳定控制敌人或提供治疗。

#### PVP（竞技场、RTA）装备策略

PVP强调速度压制、爆发输出、控制反制与生存反打。
*   **先手速度队：**
    *   **拉条手/控制手：** 极致的速度套 免疫套或命中套。主属性鞋子必选速度，副属性全力堆叠速度。这是2025年PVP环境中依然主流的策略。
    *   **爆發輸出：** 速度套+暴擊套/貫穿套。確保在拉條後能立即打出高額傷害。暴擊率接近100%，暴擊傷害越高越好。
*   **反击/坦克队：**
    *   **反击套：** 适用于特定英雄，如查尔斯、暗帝等。搭配免疫套增加生存力。
    *   **生命套/防御套 抵抗套/免疫套：** 为坦克型英雄提供高额坦度与抵抗，抵抗控制效果。主属性多选生命百分比、防御百分比。
*   **贯穿套的崛起：** 随着英雄防御力普遍提升，贯穿套在PVP中对高防御目标能打出更可观的伤害，是2025年输出型英雄的热门搭配。

#### 装备洗练：追求完美副属性

洗练石能让装备的副属性数值达到最大值，甚至调整不理想的副属性。
*   **稀有洗练石：** 优先用于最高品质（红装、T70以上）且副属性完美的装备。
*   **属性调整：** 针对副属性中「歪」掉的属性（例如输出装备强化到效果抵抗），可以使用洗练石将其调整为更适合的属性，但这需要消耗大量资源，需谨慎评估。
*   **速度洗练石：** 对于PVP中的速度装备尤其重要，能将速度副属性洗练到最大值，微小的速度差距都可能决定胜负。

随着游戏内容更新，一些副本的难度设计更具挑战性。

*   **难点一：敌方高额范围伤害**
    *   **应对**：强化前排美人的防御与生命值，或携带具有范围治疗、护盾的美人。同时，尽快击杀敌方范围输出单位。

    *   **应对**：优先集火敌方治疗单位。若无法集火，则考虑携带具有禁疗效果的美人，或提升我方整体输出，形成压倒性优势。

### 二转流程通用步骤


二转的流程通常比一转更为复杂，需要更多的跑图、战斗与解谜。

3.  **触发转职任务**：与该职业的转职NPC对话，领取二转任务。
4.  **完成多阶段试炼**：二转任务通常包含数个环节，考验玩家的综合能力：
    *   **笔试/知识问答**：考验对职业背景、技能的理解。

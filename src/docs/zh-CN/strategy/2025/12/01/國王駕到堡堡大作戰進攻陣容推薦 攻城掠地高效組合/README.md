### 1. 「铁壁推进」重装碾压流


*   **核心思路**：利用高生命值与防御力的单位吸引火力，后排输出稳定推进，逐步蚕食敌方防线。此阵容对付单点高输出防御塔或密集型防御效果显著。
*   **推荐组合**：
    *   **前排肉盾**：2-3队「圣殿骑士」或「巨型石魔」（依据对方防御类型选择，石魔对物理防御高，骑士对魔法防御高）。
    *   **中排辅助/控制**：1队「战地牧师」（提供治疗与增益）  1队「寒冰法师」（范围减速与控制）。
    *   **后排核心输出**：2队「帝国弓手」或「弩炮部队」（弓手对轻甲目标，弩炮对建筑与重甲目标有奇效）。
    *   **英雄选择**：搭配具有范围增益或强大单体控制能力的英雄，如「狮心王」（提升周围单位防御）或「暗影刺客」（切入后排干扰）。
*   **攻城技巧**：
    1.  **侦察**：优先侦察敌方防御塔的攻击类型与范围。
    2.  **部署**：将肉盾单位部署在主要火力点前方，吸引仇恨。
    3.  **推进**：待肉盾站稳脚跟后，迅速部署辅助与输出单位，确保其在肉盾保护下输出。
    4.  **技能**：适时使用牧师的治疗技能，或寒冰法师的范围控制，延缓敌方反击。

### 2. 「疾风突袭」绕后歼灭流


*   **核心思路**：利用高机动性单位绕过前排防御，直捣敌方后排核心输出或英雄，造成混乱并迅速瓦解防线。适用于敌方后排防御薄弱或布局分散的情况。
*   **推荐组合**：
    *   **先锋突击**：2队「疾风游侠」或「暗影刺客」（高机动性，可快速切入后排）。
    *   **中路牵制**：1队「狂暴战士」（吸引中路火力，为绕后单位争取时间）。
    *   **远程支援**：2队「火炮部队」或「精灵射手」（提供远程火力支援，清理残余或压制敌方增援）。
    *   **英雄选择**：选择具备位移技能或范围爆发伤害的英雄，如「风暴之刃」（快速突进，范围斩杀）或「火焰术士」（远程高爆发）。
*   **攻城技巧**：
    1.  **侦察**：找出敌方防御薄弱的侧面或后方，以及关键的后排输出单位。
    2.  **牵制**：先部署狂暴战士等单位在中路吸引火力。
    3.  **突袭**：在敌方火力被牵制时，从侧翼或后方部署疾风游侠/暗影刺客，迅速集火敌方后排。
    4.  **协同**：远程部队紧随其后，清理被突袭单位瓦解后的残余​​防御。

### 3. 「空中打击」制空权压制流


*   **核心思路**：利用飞行单位规避地面防御，直接攻击核心建筑或防御塔，达到快速破坏的目的。对地面防御强大但对空能力不足的堡垒尤其有效。
*   **推荐组合**：
    *   **主力空军**：3-4队「皇家飞龙」或「狮鹫骑士」（高生命值与伤害，优先攻击防御塔）。
    *   **空中辅助/清场**：1队「精灵龙」（范围伤害，清理小兵或低血量建筑）。
    *   **地面牵制**：1队「哥布林工兵」（部署在地面吸引少量火力或清理外围建筑）。
    *   **英雄选择**：搭配具有对空增益或范围伤害的英雄，如「天空领主」（提升空军攻击力）或「雷霆法师」（范围闪电链）。
*   **攻城技巧**：
    1.  **侦察**：确认敌方对空防御塔（如箭塔、法师塔）的位置与数量。
    2.  **部署**：将主力空军集中部署，避开密集对空火力，直奔核心防御塔。
    3.  **牵制**：适时部署地面单位吸引部分火力，为空军争取更多输出时间。
    4.  **集火**：利用空军的机动性，集中火力快速摧毁一至两座关键对空防御塔，打开突破口。

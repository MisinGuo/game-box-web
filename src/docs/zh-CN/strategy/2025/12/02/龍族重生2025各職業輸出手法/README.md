    2.  **起手爆发：** 使用高伤害突进技能（如「冲锋斩」）接近目标，立即接续主要爆发技能（如「旋风斩」或「毁灭打击」），并在期间穿插怒气生成技能（如「英勇打击」）。

### 法师 (Mage)：元素掌控的远程炮台


法师在2025年依然是远程输出的代表，其强大的元素魔法能够对敌人造成毁灭性打击。

    *   **PVE：** 寻找Boss的易伤期或短暂停顿期，打出最大化爆发。在多目标战斗中，可利用AOE技能进行补充伤害。
    *   **PVP：** 精准判断进场时机，一套控制加爆发迅速击杀关键目标。利用「消失」技能脱离战斗或重置潜入，再次寻找机会。

2025年的圣骑士不仅是坚不可摧的防线，其惩戒专精在输出方面也表现出色，攻守兼备的特性使其在团队中拥有独特价值。

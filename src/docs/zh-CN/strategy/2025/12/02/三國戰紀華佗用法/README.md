*   **特性：** 华佗向前推出一道气波，能击飞敌人并造成多段伤害。
*   **用法：**
    *   **起手与连招：** 是华佗最常用的起手招式之一，击飞效果能为后续连招创造机会。

*   **特性：** 华佗召唤一只仙鹤向前冲刺，对路径上的敌人造成伤害。部分版本可能带有冰冻或击晕效果。

    *   **控场：** 若带有冰冻效果，是极佳的控场技能，能锁定高威胁敌人。
    *   **连招衔接：** 在仙人指路击飞后，可接华佗援护继续输出。

### 4. 元气斩 (↑↓A)


*   **特性：** 华佗原地跳起，挥舞拂尘向下攻击，造成范围伤害并击倒敌人。
*   **用法：**
    *   **解围：** 被敌人包围时的有效解围技，能将周围敌人击倒争取喘息空间。
    *   **对空：** 对于空中敌人有不错的判定。
    *   **连招终结：** 在一套连招的末尾使用，能确保敌人倒地，防止反击。

### 5. 气功波 (↓↑A)


*   **特性：** 华佗发射一道缓慢移动的气功波，持续对触碰到的敌人造成伤害。
*   **用法：**
    *   **压制与区域封锁：** 气功波的持续伤害和缓慢移动特性，使其成为优秀的区域压制技能，可以封锁敌人的走位。

    *   **掩护：** 在队友复活或进行其他操作时，气功波能提供一定的掩护。

### 6.功法回春（↓↘→B/必杀技）

    *   **能量管理：** 由于消耗能量较大，需谨慎使用，确保在最关键时刻发挥作用。

### 7.毒雾形成（↓↙←B/必杀技）


*   **特性：** 华佗释放一片毒雾，对进入范围的敌人造成持续毒素伤害。

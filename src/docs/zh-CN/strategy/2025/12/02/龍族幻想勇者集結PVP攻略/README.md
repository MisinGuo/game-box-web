进入战场后，有效的战术执行是将准备转化为胜利的关键。

勇者集结是团队战，有效的沟通至关重要。

*   **技能配合：** 坦克开团、控制技能衔接、输出集火、辅助保护，环环相扣。

#### 2. 开局布局与据点争夺：先发制人

*   **快速占点：** 开局后，根据地图与队伍配置，迅速分配队员前往关键据点进行占领。
*   **侦查与反侦查：** 派出机动性强的职业进行侦查，了解敌方动向，避免被偷袭。
*   **小规模遭遇战：** 在据点争夺中，注意利用地形优势，优先集火敌方威胁大的输出或治疗。

#### 3. 战斗中策略：应变与集火

*   **集火目标：** 这是PVP中最关键的战术。队伍应集中火力攻击敌方同一目标，优先击杀敌方输出或治疗。
    *   **优先级：** 敌方治疗 > 核心输出 > 强力控制 > 坦克。
    *   **标记：** 使用游戏内标记功能，快速指示集火目标。
*   **控制链与免控利用：**
    *   **控制链：** 队友之间衔接控制技能，让敌方目标长时间无法行动，为击杀创造条件。
    *   **免控技能：** 善用自身的免控技能，在关键时刻解除控制或免疫控制，反打一波。
*   **走位与站位：**
    *   **输出：** 保持安全距离，避免被敌方控制或集火。
    *   **坦克：** 站在队伍最前方，阻挡敌方进攻，保护后排。
    *   **辅助：** 紧跟输出或坦克，提供及时支援。
*   **技能CD管理：** 了解自己与队友的爆发技能、控制技能的冷却时间，合理规划下一波进攻或防守。
*   **利用地图机制：** 某些地图可能会有增益Buff、传送点或可破坏的障碍物，善加利用能扭转战局。

#### 4. 逆风局应对：永不放弃

*   **调整战术：** 如果开局不利，不要僵化。分析失利原因，是集火不够？还是站位分散？
*   **防守反击：** 劣势时可选择抱团防守，利用地形优势进行反击，等待敌方失误。
*   **偷点或偷袭：** 在敌方大部队集中时，派出高机动性职业去偷袭无人防守的据点，牵制敌方兵力。
*   **等待关键技能：** 在劣势时，等待队伍中的大招或关键控制技能CD转好，再发起一波反攻。

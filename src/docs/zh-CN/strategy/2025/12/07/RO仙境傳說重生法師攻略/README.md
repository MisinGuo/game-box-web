#### 1. 纯智力/范围输出流（AoE DPS）

这是最常见也最有效率的练功与PVE流派，以最大化魔法伤害为目标。

*   **魔法师（1转）技能推荐：**
    *   **火箭术 (10)、冰箭术 (10)、雷击术 (10)：** 初期单体输出与属性克制。

    *   **雷鸣术 (10)：** 高伤害单体技能，打王主力。
    *   **禅心 (5)：** 提升SP恢复，长期练功续航力保证。
    *   **圣灵召唤 (1)：** 前置技能或应急。
    *   **冰冻术 (5)：** 强力单体控场，PVE/PVP皆有用。
    *   **冰刃之墙 (1)：** 前置技能。
*   **巫师（2转）技能推荐：**
    *   **暴风雪 (10)：** 法师的招牌范围技，练功刷怪核心。
    *   **崩裂术 (10)：** 高伤害范围技，与暴风雪搭配使用。
    *   **陨石术 (10)：** 另一高伤害范围技，但咏唱较长，可依需求点满。
    *   **怒雷强击 (10)：** 强力范围雷属性魔法，针对水属性敌人效果极佳。
    *   **泥沼术 (5)：** 降低敌人移动速度与敏捷，PVP/PVE实用控场。
    *   **能量外套 (5)：** 提升物理防御，增加生存能力。
    *   **吸魂术 (1)：** 前置或应急。

#### 2. 冰雷控场流（Crowd Control

此流派更侧重于利用冰冻、石化等控场技能限制敌人，并在关键时刻爆发输出，适合PVP与高难度副本。

*   **魔法师（1转）技能推荐：**
    *   **冰冻术 (10)：** 核心控场技能，PVP必备。
    *   **石化术 (10)：** 另一强大控场，尤其对高防敌人有效。
    *   **火焰之壁 (10)：** 同样重要。

*   **巫师（2转）技能推荐：**
    *   **暴风雪 (10)：** 依然是优秀的范围控场与输出。
    *   **怒雷强击 (10)：** 针对性爆发输出。
    *   **泥沼术 (5)：** 强化控场。

### PVE与PVP实战技巧：精通法师的战场艺术


#### 1. PVE副本与BOSS战策略

*   **拉怪与站位：** 利用火焰之壁、冰冻术控场，将怪物集中后，再施放暴风雪、崩裂术等范围技能。保持与怪物的安全距离。
*   **属性克制：** 了解BOSS或小怪的属性，使用克制属性的魔法能造成更多伤害。
*   **走位与吟唱：** 法师技能咏唱时间较长，需预判敌人行动，在安全位置咏唱。利用「Shift 点击」或设定热键，精准施放技能。
*   **辅助技能：** 能量外套在危急时刻能提供宝贵的物理减伤。禅心确保SP续航。

#### 2. PVP竞技场与GVG团战战术

*   **控场优先：** 在PVP中，冰冻术、石化术、泥沼术是法师的生命线。优先控制敌方高威胁目标（如刺客、猎人、其他法师）。
*   **范围压制：** 在GVG中，暴风雪、崩裂术能有效压制敌方阵型，制造混乱并输出大量伤害。
*   **走位与保护：** 法师是集火目标，务必保持与队友的距离，并利用地形或队友的保护。一旦被近身，立即使用冰冻或石化反制。
*   **技能连招：**
    *   **单体爆发：** 冰冻术 -> 雷鸣术/崩裂术
    *   **范围压制：** 泥沼术 -> 暴风雪 -> 崩裂术
*   **装备切换：** 准备一套PVP专用的减伤装备或抗性装备，能大幅提升生存率。

*   **技能描述：** 霜焰龙向前方扇形区域喷射冰火混合能量，对范围内所有敌军造成大量伤害，并有机率施加「冻结」或「灼烧」效果。

*   **技能描述：** 霜焰龙的鳞片被冰霜能量强化，提升自身与所带领部队的防御力与生命值上限。

*   **技能描述：** 霜焰龙体内蕴含炽热能量，提升自身与所带领部队的攻击力与部队攻击速度。

#### 4. 被动技能：冰火共鸣 (Elemental Resonance)

*   **技能描述：** 当霜焰龙的主动技能触发冻结或灼烧效果时，有一定机率对目标周围的敌人造成额外伤害或控场效果。

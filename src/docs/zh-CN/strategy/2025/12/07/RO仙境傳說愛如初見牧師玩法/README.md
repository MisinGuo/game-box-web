牧师的配点主要围绕「智力（INT）」、「体质（VIT）」和「灵巧（DEX）」三大核心属性。根据不同的玩法需求，可以分为以下主流配点方案：

### 1. 全辅助型（Full Support, FS）牧师：PVE副本与GVG首选


这是最常见也是最受欢迎的牧师配点，旨在最大化治疗量、Buff持续时间与生存能力。

*   **INT (智力)：** 100-120 (优先点满)。智力直接影响魔法攻击力、治愈量、最大SP和SP恢复速度。满INT是FS牧师的基石。
*   **VIT (体质)：** 80-100 (次要点高)。体质提升最大HP、物理防御、魔法防御、异常状态抗性。高VIT能让牧师在战场上站得更稳。
*   **DEX (灵巧)：** 40-60 (确保咏唱速度)。灵巧减少技能咏唱时间，对于频繁释放技能的牧师至关重要。
*   **AGI (敏捷)、STR (力量)、LUK (幸运)：** 通常不点或微点，作为剩余点数的补充。

**配点顺序建议：**
1.  初期：INT优先点至60-80，确保基础治疗与SP循环。
2.  中期：补足VIT至40-60，提升生存能力；DEX点至30-40，改善咏唱手感。
3.  后期：将INT点满至100-120，VIT点至80-100，DEX补足至40-60。

### 2. 战斗型（Battle Priest, BP）牧师：练功与部分PVP


如果你喜欢单人练功或在PVP中发挥更多进攻作用，BP牧师是个有趣的选择。但请注意，在《爱如初见》中，BP牧师的效率通常不如纯输出职业，且对装备要求较高。

*   **INT (智力)：** 80-100 (提供基础魔攻与治愈)。
*   **DEX (灵巧)：** 60-80 (提升咏唱速度与命中)。
*   **VIT (体质)：** 30-50 (保持一定生存能力)。
*   **STR (力量)：** 根据武器类型适当点数，用于提升物理攻击力（如果使用钝器）。
*   **LUK (幸运)：** 部分BP会点LUK提升暴击率，但效益通常不高。

**配点顺序建议：**
1.  初期：INT与DEX均衡提升，确保技能伤害与咏唱。
2.  中期：根据主要输出技能（如神圣之光）的属性，选择性提升INT或DEX。
3.  后期：补足VIT，并根据装备与打法调整其他属性。

**总结：** 2025年版本，全辅助型牧师依然是主流且最推荐的选择。战斗型牧师更适合有特定目标或追求独特玩法的玩家。

    *   **Buster卡 (红卡)：** 暴击星生成率最低，主要用于高伤害输出。
*   **Overkill（鞭尸）：** 当敌人HP归零后，继续对其进行攻击（鞭尸），会显著提升NP和暴击星的获取量。这是高效产星的技巧之一。

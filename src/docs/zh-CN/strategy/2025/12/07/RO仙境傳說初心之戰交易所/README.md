### 物品稀有度与价格波动：供需法则的影响


交易所的物价波动，核心在于供需法则。

*   **低需求、高供给：** 价格下跌。例如，大量玩家挂机产出的普通材料。
2025年，官方活动、版本更新、新职业开放等，都会剧烈影响特定物品的供需关系，进而引发价格波动。

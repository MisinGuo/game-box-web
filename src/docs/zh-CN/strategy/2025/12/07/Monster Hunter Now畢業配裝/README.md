### 1. 大剑 (Great Sword)


大剑的毕业配装核心是「集中」与「攻击守势」。

*   **推荐武器：**
    *   **火属性：** 飞雷龙大剑 (高雷属)、火龙大剑 (高火属)
    *   **水属性：** 泥鱼龙大剑 (高水属)
    *   **冰属性：** 冰牙龙大剑 (高冰属)
    *   **龙属性：** 恐暴龙大剑 (高龙属) - *若2025年已推出*
    *   **无属性：** 角龙大剑 (高物理面板，配合无属性强化)
*   **关键技能：** 集中 (加速蓄力)、攻击守势 (格挡后增伤)、弱点特效 (提升会心伤害)、看破 (提升会心率)。
*   **配装思路：** 优先堆叠「集中」至Lv3，其次是「攻击守势」与对应的「元素攻击强化」或「无属性强化」。防具选择上，可考虑火龙头、角龙腰等提供关键技能的部位。

### 2.长剑


太刀以连段流畅、气刃斩爆发著称，毕业配装注重「看破」、「弱点特效」与「攻击」或「匠」（若未来加入）。

*   **推荐武器：**
    *   **火属性：** 火龙太刀
    *   **水属性：** 泥鱼龙太刀
    *   **雷属性：** 飞雷龙太刀
    *   **冰属性：** 冰牙龙太刀
    *   **龙属性：** 恐暴龙太刀
*   **关键技能：** 看破 (提升会心率)、弱点特效 (提升会心伤害)、攻击 (提升基础攻击力)、回避距离UP (增加机动性)。
*   **配装思路：** 由于太刀本身具有较高的机动性，在保证输出技能的同时，可适当加入「回避距离UP」提升狩猎体验。火龙套装的「看破」与「弱点特效」是太刀的绝佳选择。

### 3. 片手剑 (Sword


片手剑攻守兼备，可灵活使用道具，毕业配装强调「看破」、「弱点特效」与「攻击守势」。

*   **推荐武器：** 各属性武器皆有优秀表现，例如火龙片手、泥鱼龙片手等。
*   **关键技能：** 看破、弱点特效、攻击守势、连击 (提升伤害) 或 属性攻击强化。
*   **配装思路：** 片手剑可以频繁格挡，因此「攻击守势」的收益极高。搭配「连击」或「属性攻击强化」能最大化其输出。

### 4. 双剑 (Dual Blades)


双剑是元素伤害的极致代表，毕业配装以「属性攻击强化」为核心。

*   **推荐武器：**
    *   **火属性：** 火龙双剑

    *   **龙属性：** 恐暴龙双剑
*   **关键技能：** 各属性攻击强化 (堆满)、看破、弱点特效。
*   **配装思路：** 双剑的毕业配装几乎是为各元素量身打造。每种元素都需要一套专属的**顶级装备**，将对应的「属性攻击强化」堆叠至最高等级。例如，火龙头、手、腰等部位提供火属性强化，搭配其他提供看破或弱点特效的部件。

### 5. 轻弩 (Light Bowgun)


轻弩的毕业配装取决于弹种，通常以「反动减轻」、「装填速度」和「属性攻击强化」为核心。

*   **推荐武器：**
    *   **贯通弹/散弹：** 角龙轻弩 (高物理)

*   **关键技能：** 反动减轻、装填速度 (确保射击流畅)、各属性攻击强化、看破。
*   **配装思路：** 轻弩的配装非常依赖特定武器的弹种。例如，使用属性弹的轻弩需要将对应的「属性攻击强化」堆满。而使用贯通弹或散弹的轻弩则更注重「攻击」和「弱点特效」。

### 6. 弓


弓箭的毕业配装以「集中」与「属性攻击强化」为核心，并搭配「看破」或「弱点特效」。

    *   **火属性：** 火龙弓

    *   **冰属性：** 冰牙龙弓
    *   **龙属性：** 恐暴龙弓
*   **关键技能：** 集中 (加速蓄力)、各属性攻击强化、看破、弱点特效。
*   **配装思路：** 弓箭的毕业配装与双剑类似，也是为各元素量身打造。堆叠「集中」至Lv3是基本，然后将对应的「属性攻击强化」堆满。

### 7. 锤 (Hammer)


锤子的毕业配装围绕着「集中」和「攻击守势」展开，以蓄力攻击为主要输出手段。

*   **推荐武器：**
    *   **无属性：** 角龙锤 (高物理)
    *   **火属性：** 火龙锤
    *   **雷属性：** 飞雷龙锤
*   **关键技能：** 集中、攻击守势、弱点特效、攻击。
*   **配装思路：** 锤子需要快速蓄力并精准打击弱点，因此「集中」和「弱点特效」至关重要。 「攻击守势」在格挡后提供可观的增伤。

### 8. 长枪 (Lance)


长枪以精准防御反击为核心，毕业配装注重「防御性能」、「攻击守势」和「看破」。

*   **推荐武器：** 各属性长枪，如火龙长枪、泥鱼龙长枪。
*   **关键技能：** 防御性能 (提升格挡效果)、攻击守势 (格挡后增伤)、看破、弱点特效。
*   **配装思路：** 长枪的生存能力极强，毕业配装在保证「防御性能」的基础上，最大化「攻击守势」的收益，并辅以输出技能。

### 9. 铳枪 (Gunlance) - *2025年假想新增武器*


若铳枪在2025年加入MHN，其毕业配装将围绕「炮术」和「防御性能」展开。

*   **推荐武器：** 根据炮击类型（通常、放射、扩散）选择。
*   **关键技能：** 炮术 (提升炮击伤害)、防御性能、攻击守势。
*   **配装思路：** 铳枪的炮击伤害固定，不受攻击力影响，因此「炮术」是核心。同时，其防御能力也需强化。

### 10. 斩击斧 (Switch Axe) - *2025年假想新增武器*


若斩击斧在2025年加入MHN，其毕业配装将注重「集中」或「属性解放」以及「看破」。

*   **推荐武器：** 根据瓶种类型（强击、强属、麻痹等）选择。
*   **关键技能：** 集中 (加速变形斩斧槽回复)、属性解放 (提升属性瓶效果)、看破、弱点特效。
*   **配装思路：** 斩击斧需要在斧模式和剑模式间灵活切换，并利用剑模式的瓶效果。

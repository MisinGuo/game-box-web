一个均衡的队伍是通关的基础。您的阵营战场队伍应至少包含：
*   **控场角（Crowd Control, CC）**：负责晕眩、冰冻、恐惧或减少速度，有效限制敌方行动。
*   **治疗角（Healer）/护盾角（Shielder）**：提供队伍续航能力，抵御伤害。

### 3. 速度调校（Speed Tuning）


在阵营战场中，英雄的速度顺序至关重要。
*   控场角应优先于输出角行动，确保敌人被控制后再进行输出。
*   治疗角和护盾角应在敌人造成主要伤害之前或之后及时提供支援。
*   复活角通常需要较高的速度，以便在队友倒下后能尽快行动。

### 4. 针对性战术：Boss关卡应对

*   **阻挡复活（Block Revive）**：某些Boss会复活小怪，阻挡复活技能能有效简化战斗。

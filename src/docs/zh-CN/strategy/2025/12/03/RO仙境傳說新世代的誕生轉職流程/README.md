*   **Base等级：** 达到Lv.10。
*   **任务前置：** 通常无特殊前置任务，只需完成部分引导任务。

### 一转任务步骤：快速上手指南


1.  **前往普隆德拉（Prontera）中央广场：** 大部分一转职业的导师都位于此处或附近。
2.  **寻找职业导师NPC：**

    *   **服事：** 普隆德拉教堂（左下角）
    *   **商人：** 普隆德拉中央广场下方

3.  **对话并接受转职任务：** 导师会给予您一个简单的任务，通常是击败特定魔物、收集物品或进行知识测验。
4.  **完成任务：** 按照任务指示完成后，返回导师处。
5.  **确认转职：** 导师会恭喜您成功转职，并解锁新的技能树。

*   **技能点：** 优先点满该职业的核心输出技能或辅助技能，以提高练功效率。

*   **组队支援：** 若遇到难以单独击败的魔物，大胆向公会或世界频道求助。

*   **骑士 vs. 十字军：** 骑士偏向输出与范围攻击，十字军则更偏向防御、辅助与单体高伤。

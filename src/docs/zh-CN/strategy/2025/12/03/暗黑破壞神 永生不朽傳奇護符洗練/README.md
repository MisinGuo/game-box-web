这是最常用且最有效的策略。当你洗出一个你非常满意的核心技能时，立即使用白金币将其锁定。然后继续洗练剩余的未锁定词缀。

*   **目标设定：** 在洗练前，明确你这次洗练的「最低可接受标准」是什么（例如，洗出3个核心技能）。

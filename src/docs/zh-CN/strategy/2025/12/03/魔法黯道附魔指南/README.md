附魔的来源多样，主要包括：
*   **地牢掉落：** 在各种地牢中击败敌人、开启宝箱或完成任务，都有机会获得不同稀有度的附魔。

*   **任务奖励：** 完成特定城镇事件或远征任务，有时会获得独特的附魔作为奖励。
*   **DLC专属：** 如《血色庭院》和《狂热农庄》等DLC会引入大量独特的附魔，为游戏策略带来更多变数。

### 2. 附魔的稀有度与等级


附魔根据其稀有度分为多个等级，通常稀有度越高，提供的加成越强大，但同时也可能伴随更显著的负面效果。常见稀有度包括：
*   **普通 (Common)**
*   **不常见 (Uncommon)**
*   **稀有 (Rare)**
*   **非常稀有 (Very Rare)**
*   **遗物 (Ancestral)**：通常是游戏中最顶级的附魔，提供强大增益，且负面效果较小或可控。
*   **英雄专属 (Class Specific/Set Trinkets)**：某些附魔专属于特定英雄职业，或是一套组合（如DLC中的套装附魔），能提供独特的强化。

### 3. 理解附魔的「双面刃」特性


《魔法黯道》的附魔设计充满了「风险与回报」的哲学。许多强大的附魔在提供巨大增益的同时，也会带来显著的负面效果（如增加压力、降低抗性、减少闪避等）。资深玩家的策略核心，便在于如何权衡这些利弊，将负面影响降至最低，甚至转化为可控的战术优势。

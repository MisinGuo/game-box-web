### 1. 宝石的种类与基础概念

![游戏场景](https://oss.2025.11.5yxy5.com/images/2025/12/03/DoE%E6%B5%81%E4%BA%A1%E5%9C%B0%E7%89%A2%E5%AF%B6%E7%9F%B3%E9%91%B2%E5%B5%8C/image_2_1764714791.png)

《DoE流亡地牢》的宝石主要分为以下几类：

*   **主动技能宝石 (Active Skill Gems)**：提供新的主动技能，如火球术、旋风斩、召唤骷髅等。它们是你BD的核心输出或辅助手段。
*   **辅助技能宝石 (Support Gems)**：无法单独使用，必须连结到主动技能宝石上，以强化或改变主动技能的效果。例如，增加伤害、扩大范围、降低消耗等。
*   **光环/诅咒宝石 (Aura/Curse Gems)**：提供持续性的增益或对敌人施加减益效果。
*   **特殊/传奇宝石 (Special/Unique Gems)**：拥有独特且强大的效果，往往能成为BD的核心。

每颗宝石都有其特定的颜色（红、绿、蓝）和属性需求（力量、敏捷、智慧），这与装备上的插槽颜色和连结数息息相关。

### 2. 镶嵌与连结机制


*   **插槽 (Sockets)**：装备上会有不同颜色的插槽，宝石必须嵌入颜色相符的插槽。白色插槽则能兼容任何颜色的宝石。
*   **连结 (Links)**：相邻的插槽之间可能存在连结，这允许辅助技能宝石对连结的主动技能宝石生效。连结数越多，一个主动技能可以获得的辅助效果就越多，潜力也越大。
*   **宝石等级与经验**：宝石会随着你击败敌人而获得经验并升级，提升其效果。部分宝石在达到特定等级后，还能解锁更强大的「觉醒」版本。

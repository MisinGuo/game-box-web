### 什么是公会战？目标、规则与奖励

公会战每周进行数次，目标是击败对手公会的防守队伍，夺取其据点。每个玩家需要配置两组防守队伍（每组3只魔灵），并使用两组进攻队伍来挑战对手的防守。胜利将获得公会点数、公会魔法石等奖励，这些资源对于公会商店的购买和魔灵培养至关重要。

### 胜败判定与积分机制


*   **胜利条件：** 成功击败敌方防守队伍，即可占领该据点并获得胜利点数。
*   **积分计算：** 每个据点有不同的血量，击败防守队伍会扣除据点血量。据点血量归零后，该据点即被占领。公会战结束时，占领据点多、剩余据点血量高的公会获胜。个人胜负场次和击杀数也会影响个人贡献度与奖励。

*   **智力（INT）：** 显著提升陷阱的基础伤害。
*   **灵巧（DEX）：** 影响技能咏唱速度与部分陷阱伤害。

*   **自动触发：** 敌人进入陷阱范围后自动触发。
*   **手动触发：** 使用「引爆陷阱」技能，引爆范围内的所有陷阱。手动引爆的伤害通常更高。

2025年陷阱游侠的属性点分配，应以最大化陷阱伤害和生存能力为核心。以下为建议优先级：

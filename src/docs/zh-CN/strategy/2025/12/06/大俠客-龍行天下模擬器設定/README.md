*   **核心原则：** 分配的CPU核心和RAM应根据你的电脑实际配置来决定，但通常建议分配电脑总资源的一半或稍多。

#### 渲染模式选择：OpenGL vs. DirectX

*   **OpenGL：** 大多数游戏的预设推荐，兼容性广，效能稳定。
*   **DirectX：** 在某些特定游戏或显卡配置下，可能提供更好的渲染效果或更高的帧率。
*   **2025年建议：** 对于《大侠客-龙行天下》，**优先选择 OpenGL 模式**。如果遇到画面显示异常或明显卡顿，可以尝试切换到 DirectX 模式并重启模拟器。
*   **设定位置：** 通常在模拟器的「设定」->「效能设定」或「图形渲染模式」中。

#### FPS帧率设定：高帧率模式的优势

*   **核心原则：** 高帧率能带来更流畅的游戏体验，尤其在战斗和移动时感受明显。

*   **设定位置：** 通常在模拟器的「设定」->「游戏设定」或「帧率设定」中。务必同时在游戏内设定中开启「高帧率模式」或「极致画质」选项。

### 宝石共鸣的运作机制：深入解析

要有效提升宝石共鸣，首先需理解其背后的运作原理：

1.  **传奇宝石的星级与等级：**
    *   **星级（Stars）：** 传奇宝石分为1星、2星、5星。星级越高，其基础属性与共鸣潜力越大。 5星宝石在满级后能提供最高的共鸣值。
    *   **等级（Rank）：** 每个传奇宝石都有自己的等级，从1级开始，透过消耗「宝石能量」（Gem Power）和「同名宝石副本」来提升。宝石等级越高，其提供的共鸣值也越高。

2.  **装备觉醒（Awakening）：**
    *   当你的传奇宝石达到10级时，可以将其镶嵌在主要装备（主手武器、副手武器、头盔、胸甲、肩膀、裤子）上，并使用「曙光回响」（Dawning Echo）进行觉醒。
    *   觉醒后的装备会获得一个额外的宝石插槽，并大幅提升镶嵌宝石的共鸣值。这是一个巨大的飞跃，也是后期共鸣提升的必经之路。

3.  **共鸣层级：**
    *   你的总宝石共鸣值会达到不同的层级（例如1000、2000、3000等），每个层级都会提供额外的全装备属性加成。这些加成是乘法计算，效果非常显著。

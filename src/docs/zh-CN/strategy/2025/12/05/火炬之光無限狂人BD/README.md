### 三、 狂人BD通用进阶技巧与实战策略


无论选择哪个流派，掌握以下通用技巧，将让你的狂人BD更上一层楼。

#### 3.1 怒气管理与爆发时机

怒气是狂人的生命线。务必确保在战斗中能持续获得怒气，并在关键时刻（如Boss血量进入斩杀线、大量精英怪聚集）开启「狂暴」状态，搭配「狂怒」技能，打出毁灭性爆发。利用「战吼」技能也能有效聚怪并提供怒气。

#### 3.2 生存能力强化：防御机制与药剂运用

狂人虽输出爆炸，但作为近战，生存压力较大。

*   **防御机制：** 堆叠足够的生命值、护甲、闪避，并确保元素抗性满值。利用天赋树中的物理减伤、生命偷取、护盾恢复等节点。
*   **药剂运用：** 携带生命药剂、护甲药剂或闪避药剂，并搭配能解除异常状态（冰冻、燃烧、中毒）的药剂。善用药剂的自动触发功能，保持战斗续航。

#### 3.3 拓荒与后期刷图建议

*   **拓荒期：** 优先选择清图效率高的旋风斩流派，快速推进剧情。装备以基础属性好、生命抗性高的为主，无需过度追求顶级词缀。
*   **后期刷图：** 根据地图词缀调整部分技能或装备，例如面对高物理减免的敌人，可能需要更多物理穿透。

#### 3.4 面对不同Boss的应对策略

*   **高频率攻击Boss：** 提升护甲与物理减伤，利用位移技能躲避关键技能。
*   **元素伤害Boss：** 确保对应元素抗性满值，使用抗性药剂。
*   **需要走位的Boss：** 善用跃击/冲击等位移技能，保持输出节奏的同时规避伤害。

    *   **A:** 通常在活动期间，登入游戏后会有提示视窗让你指定一个符合资格的新角色。只要角色等级仍在指定范围内（通常是10等以下），都还有机会。

    *   **A:** 大部分奖励，特别是装备和消耗品，都是角色限定且不可交易的。但部分奖励（如宠物）可能是帐号内可转移，请以游戏内说明为准。

世界王是《天命之子》中定期举办的全服讨伐活动。玩家需要在限定时间内，组织一支由20位角色（Child）组成的庞大队伍，对抗拥有巨量生命的Boss，并根据造成的总伤害量获取排名与奖励。

其核心机制主要围绕以下几点：

*   **20人战队系统：** 与常规的5人队伍不同，WB考验的是玩家的角色池深度与对角色技能的理解。如何从上百位角色中挑选出最佳的20人，是高分的基础。

*   **属性克制与Boss特性：** 每个WB都有特定的属性（如光、暗、木、火、水）与独特的技能机制（如石化、中毒、回复无效等）。针对性的队伍配置是攻略的必要前提。

### **城墙与防御建筑的优先级**


城墙的等级至关重要，它直接决定了你的基地在被攻击时的燃烧速度与自动灭火上限。在2025年，我们建议的升级优先级如下：

1.  **司令部 (Headquarters):** 永远是第一优先，解锁一切。
2.  **城墙 (Wall):** 提升耐久与防御值，是拖延时间的关键。
3.  **瞭望塔 (Watchtower):** 提供敌军来袭的详细情报，让你做出正确应对。
4.  **英雄营地 (Hero Precinct):** 决定了你的防守英雄数量与阵型。
5.  **医院 (Hospital):** 尽可能提升容量，伤兵比死兵好。

### **资源建筑的内外布局策略**


传统观念是将资源建筑外置，作为缓冲带。但在目前版本，这种布局容易让敌人无损赚取击杀分数。

*   **推荐策略：** 将资源建筑紧贴城墙内侧放置，形成第二道屏障。将兵营、靶场等非核心军事建筑放在最外围，吸收第一波伤害。这样可以迫使敌方部队在攻击你的防御塔和核心部队前，浪费更多时间清理这些「路障」。

### **主流防御阵型分析**


*   **引导阵型 (Funneling Layout):** 在基地一侧或角落留出一个明显的缺口，周围布满防御塔和陷阱。这种阵型能将敌军引导至你的「击杀区」，集中火力快速消灭。
*   **角落龟缩阵型 (Corner Defense):** 将司令部和核心建筑置于地图的一个角落，用其他建筑层层包裹。这种布局能最大化利用地图边界，减少被攻击的面，但缺点是被突破后容易全军覆没。

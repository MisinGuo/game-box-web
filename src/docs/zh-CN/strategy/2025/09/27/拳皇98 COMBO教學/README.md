### 什么是「取消」（Cancel）？

![游戏场景](https://oss.2025.10.5yxy5.com/strategy/2025/09/27/拳皇98 COMBO教學/images/image_2_1758998113.jpg)

取消是《拳皇98》连段构成的基石。它指的是在一个招式击中对手的瞬间，立刻输入下一个招式的指令，从而「取消」前一招的收招动作，使连段得以成立。主要分为两种：

*   **普通技取消必杀技：** 这是最基础的取消。例如，草薙京的站立重拳（C）击中后，立刻输入「荒咬」（↓↘→ A），重拳的收招动画会被荒咬的动作覆盖，形成二段连击。
*   **必杀技取消超必杀技：** 在特定必杀技击中后，可以立即输入超必杀技指令来取消。这需要消耗一颗能量珠。

### 「目押」与「连打」的区别


*   **目押（Link）：** 指一个招式结束后，利用对手足够长的硬直时间，衔接下一个招式。这对时机要求极高，需要精准的节奏感。例如，某些角色的下轻脚（↓B）之后接下轻拳（↓A）。
*   **连打（Chain/Cancel）：** 指的是透过系统内建的取消机制来连续出招，也就是我们上面提到的「取消」。绝大多数的COMBO都建立在取消系统之上。

1.  **施展超必杀技：** 造成巨大伤害，通常作为连段的收尾。

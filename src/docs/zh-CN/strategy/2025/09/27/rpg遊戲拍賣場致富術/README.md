### 模式一：「倒卖」（Flipping）的艺术


这是最经典的拍卖场玩法：**低买高卖**。核心是找到那些被其他玩家以远低于市场均价上架的物品，迅速买下后再以正常价格或更高价格重新上架。

### 模式二：「垄断市场」（Market Cornering）


这是一种高风险、高回报的终极玩法。选择某一种特定且需求稳定的商品（例如某种重要的合剂或附魔材料），买断拍卖场上所有或绝大部分的供应，然后以你设定的更高价格重新上架，从而控制该市场。

*   **操作关键：** 需要庞大的初始资金和对市场供应链的深刻理解。通常选择那些产出途径有限的商品。
*   **风险提示：** 极度危险。如果其他玩家手中有大量库存并在你提价后抛售，或是有其他替代品，你的垄断会瞬间崩溃，导致巨额亏损。

### 模式三：「专业制造」（Crafting for Profit）


利用你的商业技能（如炼金、锻造、附魔）来创造价值。这不仅仅是制作东西然后卖掉。

*   **操作关键：** 核心是**计算利润**。利用工具或试算表，自动计算制作某件物品所需材料的总成本，并与该物品在拍卖场的售价进行比较。只制作那些有稳定利润空间的商品。
*   **专业化：** 专注于市场上最受欢迎的消耗品（药水、食物）或版本初期的毕业装备。

利用游戏的内在周期性来获利。市场需求并非一成不变。

*   **范例：** 在团队副本重置的晚上（例如周二、周四），合剂、药水、食物等消耗品的需求会达到顶峰，价格也最高。你可以在周末价格较低时入货，在副本活动日高价卖出。

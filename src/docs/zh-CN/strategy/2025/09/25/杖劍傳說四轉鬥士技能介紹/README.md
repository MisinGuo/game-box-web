  - **特点**：位移技能，能够帮助玩家快速接近目标或脱离危险。

  - **建议使用场景**：适合单体BOSS战斗和元素弱点目标。

- **技能2：无影剑阵**
  - **效果**：释放一个剑阵，持续造成范围伤害，并减少敌人防御。
  - **特点**：群体技能，适合多目标战斗。
  - **建议使用场景**：适合刷图或多BOSS战。

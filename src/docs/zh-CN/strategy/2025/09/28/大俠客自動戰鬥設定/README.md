### 战斗范围与索敌逻辑


决定了你的角色会在哪个范围内寻找并攻击敌人，直接影响清怪效率。

*   **索敌范围**：
    *   **推荐设定**：附近 / 当前萤幕
    *   **说明**：「全地图」模式会让角色疲于奔命，在怪物分散的地图效率极低，且容易引到精英怪或首领。 「附近」或「当前萤幕」能让角色专注于清理一小片区域的怪物，形成高效的刷怪循环。
*   **优先攻击**：
    *   **推荐设定**：优先攻击血量最少的目标
    *   **说明**：这个设定能确保快速减少怪物数量，降低自身受到的总伤害，是提升清怪效率的最佳选择。在特定任务需要击杀精英怪时，再手动切换为「优先攻击精英」。

这是自动战斗设定的灵魂。一个好的技能循环能让你的秒伤（DPS）最大化。

    3.  **单体高伤技能**：用于补刀或对付残血的精英怪。
    4.  **普通攻击**：通常不需特别设定，系统会在技能冷却时自动使用。

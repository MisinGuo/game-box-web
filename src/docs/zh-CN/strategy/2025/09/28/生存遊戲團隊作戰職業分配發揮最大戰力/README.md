根据2025年主流生存游戏（如《逃离塔科夫》、《Rust》、《浩劫前夕》等）的战术Meta演变，我们将团队角色归纳为四大核心职业。请注意，这些是职责划分，而非游戏内的固定职业。

### **突击者 (Assault)：团队的矛与盾**


突击者是战斗的先锋，永远在队伍的最前线。他们负责突破敌方防线、吸引主要火力，并为队友创造输出空间。

*   **核心职责：**
    *   **破点与推进：** 主动发起进攻，带领团队占领关键位置。
    *   **近距离作战 (CQB)：** 精通室内与复杂地形的战斗。
    *   **吸收伤害：** 利用重型护甲与血量优势，成为团队最可靠的屏障。
*   **装备建议：** 全自动步枪、冲锋枪、霰弹枪、重型防弹背心与头盔、投掷物（手榴弹、闪光弹）。
*   **适合玩家：** 反应迅速、打法激进、熟悉地图交战点、拥有强大心理素质的玩家。

### **哨兵 (Sentinel)：战场的眼睛与耳朵**


如果说突击者是利剑，那哨兵就是执剑人的眼睛。他们负责提供关键情报，从远处锁定威胁，并保护团队的侧翼安全。

*   **核心职责：**
    *   **侦察与情报：** 提前探查前方区域，标记敌人位置、数量与装备。
    *   **远程狙击：** 压制或消灭远处的高威胁目标（如敌方狙击手）。
    *   **警戒侧翼：** 在团队移动或交火时，确保不会被敌人包抄。
*   **装备建议：** 狙击步枪、精确射手步枪（DMR）、高倍率瞄准镜、望远镜、消音器。
*   **适合玩家：** 极具耐心、观察力敏锐、枪法稳定、善于利用地形隐蔽的玩家。

### **后勤官 (Quartermaster)：团队的生命线**


后勤官是团队能够持续作战的保障。他们或许不是击杀数最高的，但团队的续航力与财富积累都系于他们一身。

*   **核心职责：**
    *   **资源管理：** 高效搜刮物资，负责弹药、药品、食物的统一分配。
    *   **医疗支援：** 在战斗中或战后迅速为队友提供治疗。
    *   **建造与制作：** 在《Rust》这类游戏中，他们是基地建设与装备制作的核心。
*   **装备建议：** 大容量背包、医疗包、急救箱、工具，以及一把自卫能力尚可的卡宾枪。
*   **适合玩家：** 心思缜密、有条理、乐于奉献、熟悉游戏合成与经济系统的玩家。

### **战术师 (Tactician)：改变战局的X因子**


战术师是团队中的变数。他们不拘泥于特定打法，而是利用特殊装备或策略，在关键时刻打破僵局。

*   **核心职责：**
    *   **区域控制：** 使用地雷、陷阱或燃烧弹等道具封锁敌人路径。
    *   **特殊装备操作：** 操控无人机、迫击炮，或使用电子战设备干扰敌人。
    *   **战术破坏：** 负责爆破敌方掩体或防御工事。
*   **装备建议：** 榴弹发射器、爆炸物、战术装备（如夜视仪、热成像）、无人机。
*   **适合玩家：** 富有创造力、思路灵活、善于逆向思考的玩家。

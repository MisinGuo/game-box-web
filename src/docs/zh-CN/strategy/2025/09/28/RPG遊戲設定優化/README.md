这是设定选单中最复杂，也最影响效能的部分。我们将由高影响到低影响的顺序进行调整。

### 1. 基础解析度与显示模式 (Resolution


*   **显示模式 (Display Mode):** 永远优先选择 **全萤幕 (Fullscreen)** 模式。这能让系统将最多资源分配给游戏，减少输入延迟。无边框视窗模式虽然方便切换，但通常会牺牲些许效能。
*   **解析度 (Resolution):** 尽量设定为你萤幕的 **原生解析度**。降低解析度是提升帧数的最后手段，因为它会让画面变得模糊。在调整其他选项前，请保持原生解析度。

### 2. 效能杀手级选项：优先调整的关键设定


以下几个选项对帧数（FPS）影响巨大，是优化时首先要调整的目标。

*   **阴影品质 (Shadow Quality):**
    *   **影响：** 极高。即时运算的动态阴影需要大量GPU资源。
    *   **建議：** 從「高」降至「中」，通常能在不犧牲太多視覺效果的情況下，換取15-25%的效能提升。除非你的硬體頂尖，否則避免使用「極致」或「瘋狂」等級。

*   **环境光遮蔽 (Ambient Occlusion, AO):**
    *   **影响：** 高。它为物体接触面增加细微的阴影，提升画面的立体感。
    *   **建议：** 较新的HBAO 或更高效的演算法，在「中」等级就能提供很好的效果。如果帧数吃紧，这是可以考虑关闭的选项之一。

*   **反锯齿 (Anti-Aliasing, AA):**
    *   **影响：** 中至高。用于平滑物体边缘的锯齿。
    *   **建议：** TAA (Temporal Anti-Aliasing) 是目前的主流，效能与效果平衡最佳。避免使用旧的、资源消耗极大的MSAA。如果你的解析度在2K或以上，可以尝试将AA等级调低，因为高解析度本身就能减轻锯齿感。

### 3. 2025年必懂的次世代技术：DLSS, FSR 与光线追踪


*   ** nvidia dlss / amd fsr / intel xess：**
    *   **这是什么：** AI驱动的升频技术，能以较低解析度渲染画面，再放大至目标解析度，大幅提升帧数。
    *   **2025年优化核心：** 这已不是选项，而是必备工具。对于支援的游戏，请务必开启。
    *   **建议：**
        *   **品质模式 (Quality):** 最佳选择，画质损失极小，效能提升显著。
        *   **平衡模式 (Balanced):** 效能与画质的绝佳平衡点。
        *   **效能模式 (Performance):** 适用于4K解析度或追求极高帧率的玩家。

*   **光线追踪 (Ray Tracing):**
    *   **这是什么：** 模拟真实世界的光线路径，提供极致真实的反射、阴影和全域光照。
    *   **建议：** **谨慎开启**。光追是目前最大的效能消耗来源。建议在开启DLSS/FSR的前提下，再逐项尝试开启光追效果（如光追阴影、光追反射）。如果帧数低于你的流畅标准，请果断关闭它。

**Q1:​​ 为什么我把所有设定都调到最低，帧数还是很低？ **
A: 这很可能是CPU瓶颈。某些RPG（特别是拥有大量NPC或复杂物理运算的开放世界游戏）对CPU要求很高。降低阴影、LOD（细节层次）或NPC密度等与CPU相关的设定可能会有所帮助。

**Q2: 我应该锁定帧数吗？ **
A: 如果你的帧数波动很大（例如在60-90之间跳动），将其锁定在一个稳定的数值（如60 FPS）可以提供更平滑的体验，并减少画面撕裂。配合萤幕的G-Sync/FreeSync技术效果更佳。

**Q3: 纹理品质（Texture Quality）对效能影响大吗？ **
A: 纹理品质主要消耗的是显示卡的视讯记忆体（VRAM），对帧数的直接影响相对较小。只要你的VRAM充足，可以尽量开高以获得更清晰的材质细节。如果VRAM不足，则会导致游戏卡顿。

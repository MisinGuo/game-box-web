1.  **开启介面：** 按下键盘上的预设快捷键（通常是 `[ESC]` 选单内）或直接点击游戏介面下方的选单，找到「怪物收藏」图示。

*   **地图选择是关键：** 选择怪物密集、地形平坦、重生速度快的地图。一个好的挂机地图能让你在单位时间内击杀更多怪物，从而增加登录机会。
*   **善用辅助道具与系统：**
    *   **轮回碑石：** 大幅增加地图怪物重生数量与速度，是登录怪物的最佳伙伴。

*   **派遣探险：** 点击收藏介面下方的「去探险」按钮，选择你想派遣的队伍和探险时间（1小时、3小时、6小时、24小时）。

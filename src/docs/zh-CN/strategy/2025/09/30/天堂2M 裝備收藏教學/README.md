### 收藏基础与类型

装备收藏主要分为以下几种：

*   **一般收藏：** 透过收集特定数量或种类的普通装备来完成，通常提供基础能力值。
*   **套装收藏：** 收集同一套装备的不同部件，通常提供更强大的能力值加成。
*   **稀有收藏：** 涉及高阶或稀有度较高的装备，提供显著的能力值提升，是中后期玩家的追求目标。

在收藏中，**刻印装备**（可交易）和**非刻印装备**（绑定）的区别至关重要。大多数基础收藏可以使用非刻印装备完成，但部分高阶或活动收藏可能需要刻印装备。理解这一点，有助于您规划资源，决定哪些装备该自行取得，哪些可以透过交易所购买。

### 收藏能力值的叠加性

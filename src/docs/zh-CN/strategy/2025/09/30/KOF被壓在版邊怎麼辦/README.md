### 1. 精准格挡与防御取消（Guard Cancel）


*   **坚实的防御：** 这是最基本的生存之道。学会判断对手的攻击节奏，精准格挡上中下段攻击。
*   **防御取消（Guard Cancel/GC）：** 在防御硬直中，输入特定指令（通常是方向 AB或CD）消耗气槽进行反击。这是KOF系列中极其重要的脱困手段，能有效打断对手的压制，创造喘息空间。
    *   **用途：** 击退对手，拉开距离。
    *   **风险：** 若对手预判你使用GC并防御，你将处于硬直状态，容易被反击。
    *   **时机：** 在对手连段的空隙或收招硬直时使用，效果最佳。

### 2. 紧急回避（Emergency Evasion / AB Roll）


*   **前滚（Forward Roll）：** 消耗一小段气槽，向前滚动穿过对手。
    *   **用途：** 快速脱离版边，有短暂无敌帧。
    *   **风险：** 无敌帧结束后有硬直，若被对手预判并投掷或攻击，将陷入更大危机。
    *   **时机：** 在对手攻击间隙，或对手远距离发波时偷袭。
*   **后滚（Backward Roll）：** 消耗一小段气槽，向后滚动，通常用于在版中拉开距离。在版边时，后滚会让你继续贴在版边，但能调整位置或躲过一些攻击。
    *   **用途：** 躲避某些特定攻击，稍微调整站位。
    *   **风险：** 同前滚，无敌帧结束后有硬直。

### 3. 吹飞攻击反击（Blowback Attack / CD Counter）


*   **指令：** 在防御硬直中，输入CD键。
*   **效果：** 消耗一格气槽，发动一次带有无敌帧的吹飞攻击。
*   **用途：** 强行打断对手连段，将对手击飞至版中，为自己争取宝贵的脱困机会。
*   **风险：** 攻击判定较慢，若对手攻击判定快或有霸体，可能会被反击。
*   **时机：** 在对手连段的节奏点，或预判对手将要使用收招慢的攻击时。

### 4. 超必杀技/Climax超必杀技反击


*   **无敌帧利用：** 许多超必杀技和Climax超必杀技在发动时带有无敌帧。

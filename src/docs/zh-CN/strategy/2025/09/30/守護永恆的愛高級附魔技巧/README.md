*   **高级附魔：** 能洗出两条甚至三条属性（后续开放），且属性池更广泛，包含大量稀有、强力词条，如「魔力」、「物攻」、「生命上限百分比」、「忽视防御」等。其词条数值范围也更大，上限更高。

### 1.2 核心机制：随机性与词条等级


高级附魔的本质是「随机性」，但并非毫无章法。

*   **属性池：** 每件装备都有其特定的高级附魔属性池。例如，武器多出攻击向属性，防具多出防御或生命向属性。了解你装备的属性池是规划附魔的第一步。

*   **词条等级：** 每条属性都有其自身的「等级」，从最低的I级到最高的IV级（或更多）。等级越高，属性数值越高。例如，「物攻I」可能只加几十点，而「物攻IV」则能加数百点。
*   **稀有词条：** 部分极品词条（如「魔力」、「物攻」、「忽视防御」）出现机率较低，且通常只在较高等级的附魔中出现。

高级附魔是个「烧钱」的过程。你主要会消耗：

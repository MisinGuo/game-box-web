### 必备制作材料清单

![游戏场景](https://oss.2025.11.5yxy5.com/images/2025/10/24/%E6%96%B9%E8%88%9F%E9%BA%BB%E9%86%89%E7%AE%AD%E8%88%87%E9%BA%BB%E9%86%89%E9%8F%A2%E8%A3%BD%E4%BD%9C%E6%9D%90%E6%96%99/image_2_1761257834.jpg)

制作一支麻醉箭，你需要以下基础材料：

*   **石箭 (Stone Arrow) x 1：** 这是麻醉箭的基底。
    *   **材料：** 茅草 x 1，燧石 x 1，纤维 x 2，兽皮 x 1。
    *   **取得方式：** 茅草可从树木或灌木丛采集；燧石可从石头采集；纤维可从灌木丛采集；兽皮可从猎杀生物后取得。
    *   **制作场所：** 玩家背包或工作台 (Workbench)。
*   **腐肉 (Spoiled Meat) x 1：** 提供麻醉效果的关键成分。
    *   **取得方式：** 任何生肉（Raw Meat）放置在背包或储存容器中一段时间​​后会腐烂。
*   **黑色浆果 (Narcoberry) x 5：** 强化麻醉效果的主要材料。
    *   **取得方式：** 从灌木丛采集，呈现深紫色。

### 麻醉箭制作步骤


1.  **取得石箭：** 在你的背包或工作台中，使用茅草、燧石、纤维、兽皮制作石箭。
2.  **收集腐肉：** 让生肉自然腐烂，或利用研磨机 (Grinder) 加速腐肉的生成。
3.  **采集黑色浆果：** 寻找灌木丛，大量采集黑色浆果。
4.  **合成麻醉箭：** 将石箭、腐肉和黑色浆果放入你的背包或工作台的制作介面中，选择「麻醉箭」进行制作。

### 麻醉箭的应用与效率


麻醉箭需要搭配弓 (Bow)、十字弓 (Crossbow) 或复合弓 (Compound Bow) 使用。它能对目标造成一定的麻痹值 (Torpor)，同时伴随少量物理伤害。在游戏初期，面对渡渡鸟、三角龙、剑龙等生物时，麻醉箭是极佳的选择。务必留意目标生物的麻痹值衰减速度，适时补箭。

除了正确制作和选择麻醉工具，以下技巧能进一步提升你的驯服效率：

1.  **瞄准头部：** 对大部分生物而言，射击头部会造成额外的麻痹值，大幅缩短驯服时间。
2.  **利用地形与陷阱：** 将目标生物引诱至预设陷阱（如恐龙门、石墙围栏）中，可有效限制其行动，避免逃跑或反击，让你安全地进行射击。
3.  **麻醉药/黑色浆果的储备：** 在驯服过程中，即使生物昏迷，其麻痹值也会缓慢衰减。务必准备足够的麻醉药或黑色浆果，放入昏迷生物的背包中，强制喂食以维持麻痹状态。
4.  **观察生物麻痹值衰减：** 透过望远镜或驯服追踪器，密切监控目标生物的麻痹值衰减速度，适时补充麻醉箭或麻醉镖。
5.  **避免不必要的伤害：** 尽量减少麻醉工具以外的物理伤害，因为过高的伤害可能导致生物死亡，或降低驯服效率。麻醉镖的低伤害特性使其在这方面表现更优。
6.  **团队合作：** 驯服大型或高难度生物时，与队友分工合作，一人负责拉怪、一人负责射击，能大幅提高成功率。
7.  **考虑生物的抗性：** 某些生物（如泰坦巨蟒）对麻醉箭有较高抗性，而对麻醉镖效果更佳。驯服前可查阅相关资料。

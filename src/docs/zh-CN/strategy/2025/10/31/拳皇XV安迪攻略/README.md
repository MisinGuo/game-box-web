*   **中距离牵制：**
    *   利用**飞翔拳**限制对手移动，迫使对方跳入或防御。
    *   使用**轻斩影拳**快速突进​​试探，或用**重空破弹**进行远距离突袭。
    *   适时使用**下重脚**或**远重脚**进行远距离牵制，判定范围广。

    *   善用**瞬影移动**进行前后移动，调整与对手的距离，或躲避飞行道具。

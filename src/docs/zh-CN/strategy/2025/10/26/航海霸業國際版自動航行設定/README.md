*   **开启地图：** 在游戏介面下方导航栏或透过快捷键打开世界地图。

### 2. 设定多个中继点（航线规划）


单一目的地往往不足以满足复杂的航海需求。设定多个中继点是自动航行的核心策略之一。

*   **调整顺序：** 在航线列表中，你可以拖曳调整中继点的顺序，以优化航线。
*   **移除中继点：** 点击航线列表中特定中继点旁边的「X」按钮即可移除。
*   **环形航线：** 若要建立一条循环贸易路线，将起点和终点设定为同一个港口，并确保所有贸易港口都在中继点中。

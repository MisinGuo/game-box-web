圣遗物是赋予神姬额外属性与特殊效果的装备，通常分为不同部位（如头部、躯干、手部、脚部、饰品等），并拥有各自的基础属性与随机词条。集齐特定套装的圣遗物还能触发强大的套装效果。

### 词条：决定强度的核心要素


每个圣遗物都带有两种词条：

*   **主詞條 (Main Stat)：** 每個部位的主詞條類型是固定的，例如頭部通常為生命值%，軀幹為攻擊力%，但其數值範圍會隨稀有度變化。飾品類的主詞條種類較多，是搭配的關鍵。
*   **副词条 (Sub Stats)：** 每个圣遗物在获取时会随机生成3-4条副词条，强化时有机会提升现有副词条数值或解锁新的副词条。副词条的种类与数值范围极其广泛，是圣遗物「毕业」与否的决定性因素。

### 圣遗物套装效果：锦上添花的力量


除了单件圣遗物的属性外，集齐2件、4件或特定数量的同系列圣遗物，还能触发额外的套装效果。这些效果通常能大幅提升特定属性、提供特殊增益或改变技能机制，是构筑角色战斗风格的重要考量。

以下是《幻想神姬》中常见且重要的词条类型，并针对2025年游戏环境进行分析：

### 攻击类词条 (DPS核心)


*   **攻擊力% (ATK%)：** 最基礎的傷害提升詞條，適用於所有輸出角色。主詞條和副詞條都非常重要。
*   **暴擊率% (CRIT Rate%)：** 確保傷害穩定觸發暴擊的關鍵。一般建議堆疊至70-85%（配合部分角色技能或套裝可達更高）。
*   **暴擊傷害% (CRIT DMG%)：** 暴擊後傷害倍率的核心。在暴擊率達標後，暴擊傷害的收益極高。通常與暴擊率的比例為1:2或1:2.5（即1點暴擊率對應2-2.5點暴擊傷害）。
*   **增傷% (DMG Bonus%)：** 例如屬性傷害加成（火傷、冰傷等）、技能傷害加成等。通常在主詞條或特定套裝中出現，收益穩定且直接。
*   **速度 (SPD)：** 对于某些依赖行动频率或速度增益的输出角色（如部分追击型、技能循环型），速度词条能显著提升总体伤害。

### 防御/生存类词条 (坦克与辅助核心)


*   **生命值% (HP%)：** 提升角色最大生命值，直接增強生存能力。
*   **防禦力% (DEF%)：** 降低受到的物理和法術傷害。
*   **效果抗性% (Effect RES%)：** 降低被敵方施加負面狀態的機率。在PVP或面對大量控制敵人的PVE副本中至關重要。
*   **減傷% (DMG Reduction%)：** 某些套裝或稀有詞條會提供直接的傷害減免，是極其珍貴的生存屬性。

### 功能类词条 (辅助与特殊角色)


*   **能量恢復效率% (Energy Recharge%)：** 加速能量積累，讓神姬更快施放必殺技。對於依賴大招的輔助或輸出角色而言，收益極高。
*   **效果命中% (Effect HIT%)：** 提升角色施加負面狀態（如控制、減益）的成功率。對於控制型輔助或輸出角色至關重要。
*   **治療加成% (Healing Bonus%)：** 提升治療型神姬的治療量。

*   **挑战高难度副本：** 更高难度的圣遗物副本通常掉落更高稀有度（星级）的圣遗物，其主词条数值范围和副词条上限也更高。
*   **特定副本掉落特定套装：** 每个圣遗物副本都会掉落固定几套圣遗物。在刷取前，务必确认该副本是否掉落你所需的套装。
*   **体力规划：** 合理规划体力，优先刷取当前最急需的圣遗物部位和套装。

### 2. 强化与洗链技巧


*   **强化等级：** 圣遗物强化至 3、 6、 9、 12、 15时，会随机提升一条副词条的数值，或在副词条数量不足时解锁新词条。
*   **洗链 (Reroll)：** 游戏中通常有「洗链」或「重塑」功能，可以消耗特定材料来重新生成圣遗物的副词条。这是在获得主词条正确但副词条不佳的圣遗物后，追求「毕业」属性的重要手段。
*   **资源分配：** 不要将大量资源投入到主词条不对或副词条初始就极差的圣遗物上。优先强化主词条正确且至少有2-3条有效副词条的圣遗物。

### 3. 避免的误区


*   **盲目追求高星级：** 即使是五星圣遗物，如果主词条不对或副词条歪掉，其收益可能还不如四星的「极品」圣遗物。
*   **过度强化：** 在资源有限的情况下，不必追求所有圣遗物都强化到 15。先将核心输出角色的关键圣遗物强化到位，其他角色可先使用 9或 12的过渡装。
*   **忽视套装效果：** 有时一套完美的词条搭配，若能同时触发强力的套装效果，其整体收益会远超单纯堆叠词条。

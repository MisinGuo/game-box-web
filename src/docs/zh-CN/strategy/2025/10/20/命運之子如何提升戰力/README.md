角色是战力的核心载体，其基础养成至关重要。

### 1. 角色等级与突破（Uncap）


*   **等级提升：** 透过经验值素材或战斗获取经验，提升角色等级。等级是所有属性成长的基础。
*   **突破（Uncap）：** 使用同名角色或突破素材（如奥里哈根）来提升角色的突破等级。每次突破不仅能大幅提升基础属性上限，还能解锁新的技能等级上限，是战力飞跃的关键。优先将核心输出与辅助角色突破至满级（ 6）。

*   **普通技能与滑动技能：** 透过技能素材提升等级，增强伤害、治疗量或辅助效果。优先提升主力输出与关键辅助角色的核心技能。

### 3. 星级提升与进化


*   **星级提升：** 消耗进化素材和金币，将角色从低星进化到高星（最高6星）。每次进化都能大幅提升角色的基础属性，并解锁更高的等级上限。这是提升角色潜力的必经之路。
*   **进化优先级：** 优先将主力输出、强力辅助和坦度高的角色进化至6星。

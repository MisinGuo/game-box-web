全真教的武学体系主要由剑法、掌法和内功组成。理解每种武学的特点是构筑强力战斗风格的基础。

#### 1. 剑法：清静灵动，绵密不绝

*   **全真剑法**：作为全真教的入门剑法，其特色在于**持续伤害与控制**。在2025年版本中，全真剑法的小幅改动使其在累积剑气后能触发更高频率的内力伤害，并附带轻微的减速效果。这让它成为前期推图和PvE战斗中不可或缺的输出手段。
*   **三花聚顶剑法**：全真教的招牌剑法，以**爆发与范围控制**见长。其核心机制是在短时间内积蓄力量，对前方敌人造成高额伤害并附带定身效果。在2025年的竞技场环境下，精准释放的三花聚顶剑法是扭转战局的关键，尤其在面对多个近战敌人时，其群体控制能力尤为突出。

#### 2. 掌法：刚柔并济，以气御敌

*   **金雁功**：虽然名为“功”，实则兼具掌法与轻功的特性。在攻击端，金雁功提供**灵活的位移与反击能力**，能在闪避敌人攻击的同时造成伤害。作为轻功，它大幅提升了全真教弟子的机动性，无论是追击还是脱离战场都极为实用。 2025年版本强化了其闪避后的反击伤害，使其在对抗高爆发职业时更具生存力。
*   **同归剑法**：尽管名称带“剑”，但其发力方式与内力运用更偏向掌法。此武学以**牺牲自身部分防御换取极致的攻击力**为代价，是全真教少有的高风险高回报技能。在2025年的特定PvP战术中，与高防御内功搭配使用，能在关键时刻打出惊人的爆发。

#### 3. 内功：厚积薄发，源源不绝

*   **全真内功**：全真教的基础内功，提供稳定的**内力恢复与基础防御**。其精髓在于“清静无为”，让玩家在战斗中能持续输出，减少对药品的依赖。 2025年版本对其内力恢复速度进行了微调，使其在持久战中的优势更加明显。
*   **先天功**：全真教的镇派内功，是其**续航与反击的核心**。先天功不仅大幅提升内力上限与恢复速度，更能在受到伤害时有概率反弹部分伤害，并提供短暂的减伤效果。这使得全真教在面对高频率攻击时，能展现出惊人的韧性。 2025年版本强化了反弹伤害的比例，让先天功在PvP中成为一个让对手头疼的存在。

转珠是《神魔之塔》最核心的玩法，精通转珠技巧是称霸关卡的关键。

### 转珠基础概念


*   **符石种类：** 游戏中有火、水、木、光、暗五种属性符石，以及心符石（回复生命）。
*   **消除规则：** 只要将三颗或以上同属性符石连成一线（横向或纵向），即可消除并发动攻击或回复。
*   **COMBO（连击）：** 一次转珠消除越多组符石，COMBO数越高，攻击力加成也越多。
*   **移动时间：** 每次转珠有固定的移动时间，时间内可自由移动一颗符石，直到放手。

### 转珠实战技巧


1.  **路径规划：** 在移动符石前，先在脑海中规划好符石的移动路径，尽可能创造更多连击。
2.  **叠珠技巧：** 将同属性符石堆叠在一起，利用一次移动引发多次消除，制造高COMBO。这是进阶玩家必学的技巧。
3.  **首消策略：** 优先消除队伍主要攻击属性的符石，确保输出。同时也要注意消除心符石来回复生命。
4.  **特殊符石：**
    *   **强化符石：** 消除后攻击力更高。
    *   **问号符石：** 消除后会随机变为其他符石。
    *   **风化符石：** 消除后会留下痕迹，下次转珠碰到会扣血。
    *   **锁定符石：** 无法移动，需消除周围符石或特定技能解除。
    *   **电击符石：** 触碰会扣血，需消除周围符石解除。
了解这些特殊符石的应对方式，是应对高难度关卡的必备知识。

### 3.2 资源/经济流 (Resource/Economy Build)


*   **目标**：最大化金币、经验值、材料等资源的获取，加速角色成长与装备更新。
*   **核心技能**：

*   **目标**：提升角色生存能力，确保在面对强敌或长时间挂机时不易死亡，维持稳定收益。
*   **核心技能**：
    *   **生命值上限**：基础生存保障。
    *   **防御力/减伤**：降低受到的伤害。
    *   **生命恢复/吸血**：提升续航能力。
    *   **控制抗性**：减少被控制的时间，尤其在PVP或特定副本有用。
*   **分配策略**：通常作为辅助流派，在纯输出流卡关时，适当投资生存技能以求突破。某些游戏中，高生存力也能变相提升挂机效率（减少死亡次数）。

### 3.4 混合/平衡流 (Hybrid/Balanced Build)


*   **目标**：兼顾输出与生存，或输出与资源，以适应多样化的游戏内容。
*   **核心技能**：根据具体游戏玩法与个人喜好，从上述三种流派中选择核心技能进行搭配。
*   **分配策略**：
    *   **输出与生存平衡**：例如，在点满主要输出技能后，投资部分点数于生命值与防御，确保在推图时不易暴毙。
    *   **输出与资源平衡**：在保证一定输出能推进关卡的前提下，适当投入金币或经验值加成，加速整体进度。
*   **注意事项**：混合流容易陷入「样样通样样松」的困境，因此仍需有明确的主次之分。

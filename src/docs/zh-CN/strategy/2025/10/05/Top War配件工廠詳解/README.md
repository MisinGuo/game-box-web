了解配件工厂的基础运作，是高效利用它的第一步。

### 1. 配件生产


配件工厂的核心功能是生产配件。生产需要消耗时间和资源，通常是金币和加速道具。

*   **生产等级：** 工厂等级越高，能生产的配件品质越好，速度越快。务必优先提升配件工厂的等级。
*   **生产类型：** 根据你当前的需求，选择生产海军、陆军或空军配件。
*   **品质分级：** 配件从低到高分为：普通（白）、优秀（绿）、稀有（蓝）、史诗（紫）、传奇（橙）、神话（红）。目标始终是获取更高品质的配件。

### 2. 配件合成


当你拥有多个相同品质的配件时，可以将它们进行合成，以获得更高品质的配件。

*   **合成规则：** 通常是3个同品质配件合成1个更高品质的配件。例如，3个紫色配件可以合成1个橙色配件。
*   **合成策略：** 这是获取高阶配件的主要途径。积累低阶配件，然后批量合成是常见做法。合成时有一定机率跳级，直接获得更高品质的配件，但机率较低。

### 3. 配件分解


不想要的或低品质的配件可以分解，以回收部分资源。

*   **分解目的：** 清理背包空间，回收资源用于生产或升级其他更有用的配件。
*   **回收资源：** 分解通常会返还一定比例的生产材料或专属的「配件碎片」等资源，这些碎片可以用于在商店兑换特定配件或材料。

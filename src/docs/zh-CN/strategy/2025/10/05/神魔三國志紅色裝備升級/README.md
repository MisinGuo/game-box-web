#### 装备强化：基础属性稳步提升

装备强化是最直接的属性提升方式，消耗**强化石**与**铜钱**。

*   **强化上限与效益**：2025年版本中，各部位装备的强化等级上限通常与主公等级挂钩。建议优先将主力输出英雄的红色装备强化至最高等级，其次是坦克与辅助。
*   **材料获取**：强化石主要透过日常副本、活动奖励、商店购买及公会贡献获得。铜钱则可透过日常任务、副本、摇钱树等途径大量产出。
*   **策略建议**：强化是成本相对较低的提升手段，应持续进行。特别注意，在游戏前期不建议将过渡期的紫色或金色装备强化至太高，将资源集中于红色装备上。

#### 装备精炼：解锁关键属性，战力飞跃

装备精炼是红色装备独有的重要功能，它能解锁并提升装备的额外属性，如攻击力百分比、防御力百分比、生命值百分比、暴击率、暴击伤害、韧性等。精炼消耗**精炼石**与**精炼图纸**。

*   **精炼属性选择**：根据英雄定位选择精炼属性至关重要。
    *   **输出英雄**：优先选择攻击力、暴击率、暴击伤害等。
    *   **防御型英雄**：优先选择生命值、防御力、免伤等。
    *   **辅助英雄**：可考虑生命、防御或速度等。
*   **精炼材料获取**：精炼石和精炼图纸相对稀有，主要来源于高难度副本、限时活动、高级商店兑换和VIP礼包。 2025年新增的某些挑战副本会稳定产出精炼材料，务必参与。
*   **策略建议**：精炼的资源投入较大，应集中于核心英雄的核心装备上。避免平均分配，确保主力英雄的精炼等级领先。

装备进阶是将红色装备从R1提升至R2、R3等更高阶品质的关键步骤，每次进阶都会带来巨大的属性增长和潜在的技能强化。进阶通常需要消耗大量的**进阶石**和**同名红色装备碎片**，甚至需要完整的同名红色装备。

*   **进阶材料获取**：进阶石和同名红色装备碎片是稀有资源，主要透过限时抽奖活动、高阶商店兑换、跨服战奖励、以及一些2025年新增的特殊副本或活动产出。

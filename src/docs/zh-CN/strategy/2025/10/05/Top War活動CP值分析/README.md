*   **高价值奖励**：例如橙色英雄碎片、稀有装备模组、特殊建筑蓝图、大量加速道具（尤其是通用加速）。这些能直接提升战力或加速发展。

### 3. 活动周期与重复性


*   **常驻活动**：每周或每月固定出现的活动，其CP值相对稳定，可长期规划。
*   **限时活动**：节庆、联动或特殊事件，奖励通常较丰厚，但错过即不再。这类活动的CP值分析尤为重要。

    *   **低CP值**：若战力悬殊，投入大量兵力却收获甚微，则应考虑是否将资源投入其他活动。

        *   **抽奖/转盘**：若有保底机制，或你对某个特定奖励（如新英雄）有极高需求且预算充足，可适度尝试。
    *   **低CP值**：

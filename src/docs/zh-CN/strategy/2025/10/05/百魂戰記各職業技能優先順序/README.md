法师以其强大的范围魔法伤害和元素控制能力，在PVE清怪和PVP团战中扮演着不可或缺的角色。

    *   **「流星火雨」/「冰霜新星」：** 大范围高伤害魔法，通常附带灼烧或冰冻效果。法师的主要清怪与团战输出，优先点满。
    *   **「魔力弹幕」：** 快速施放的单体或小范围技能，作为普攻间隙的填充，并累积能量。

    *   **「冰冻术」/「雷电锁链」：** 强力单体或链式控制技能，能有效限制高威胁目标的行动。
    *   **「能量护盾」：** 吸收伤害或提供短暂减伤。法师较为脆弱，此技能能显著提升生存能力。
*   **被动技能 (优先度：B)**

    *   **「奥术专注」：** 提升魔法攻击力或能量恢复速度。

### 4. 游侠（Ranger/Archer）- 远程持续输出与风筝战术


游侠擅长保持安全距离进行持续输出，并利用控制技能风筝敌人，是PVE和PVP中稳定的伤害来源。

*   **核心输出技能 (优先度：S)**
    *   **「多重箭矢」：** 扇形范围攻击，清怪效率高，且能对多个目标造成伤害。
    *   **「精准射击」：** 高额单体伤害，通常附带穿透或暴击加成。针对BOSS或PVP高威胁目标的核心输出。
*   **关键控制/生存技能 (优先度：A)**
    *   **「陷阱设置」：** 放置陷阱，对触发敌人造成减速、束缚或伤害。 PVP中限制敌方走位，PVE中控制小怪。
    *   **「后跳射击」：** 向后位移并造成伤害，是游侠拉开距离、保持安全输出的重要技能。
*   **被动技能 (优先度：B)**
    *   **「鹰眼」：** 提升命中率、暴击率或射程。
    *   **「疾风步」：** 提升移动速度或闪避率。

**2025年特别提示：** 2025年版本中，一些新的辅助英雄拥有能聚怪或减速敌人的技能。游侠若能搭配这些英雄，将能最大化「多重箭矢」的范围输出效益，使其在PVE中的清怪能力更上一层楼。

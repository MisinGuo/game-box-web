术士的属性配点直接影响其生存和输出能力。以下是2025年推荐的配点策略：

1.  **智力 (INT)：** 术士最重要的属性，直接影响法术攻击力。初期和中期都应优先点满。
2.  **体能 (CON)：** 增加生命值上限。作为脆皮法师，适量的体能能显著提升生存能力。
3.  **耐力 (STA)：** 增加护甲。在面对物理攻击为主的敌人时有不错的效果，但优先级略低于体能。
4.  **敏捷 (AGI)：** 增加暴击率和闪避。暴击对于术士的爆发输出有显著提升，后期可以考虑投入。

### 推荐配点比例（2025年版本）：


*   **新手期（0-60级）：**
    *   **3 智力，1 体能 (3:1)**：确保输出基础，同时提升少量生存。
    *   **2 智力，1 体能，1 耐力 (2:1:1)**：如果你觉得生存压力较大，可以这样分配。
*   **中期（60-100级）：**
    *   **3 智力，1 体能 (3:1)**：继续保持高输出。
    *   **2 智力，1 体能，1 敏捷 (2:1:1)**：如果你装备已经不错，开始追求暴击伤害，可以尝试加入敏捷。
*   **后期（100级以上）：**
    *   **主要依赖装备词条调整。 ** 基础属性点可能维持 3 智力 1 体能，或根据团队配置和个人需求，微调智力与敏捷的比例，追求极致暴击或生存。

**E-E-A-T提示：** 根据资深玩家的经验，过度追求单一属性往往适得其反。智力是基石，但适当的体能能确保你在战斗中站得住脚，才有机会打出更多伤害。敏捷的收益在后期装备词条堆叠起来后会更明显。

无论你的玩家等级如何，以下几个核心原则是进行武器COST分配时必须考量的：

### 1. 主手武器优先原则


你的主手武器（即战斗中实际挥舞的武器）是输出或辅助的核心，其COST通常较高，但其提供的基础攻击/魔攻、技能发动率以及秘技效果至关重要。

*   **高COST高稀有度：** 优先将最高的COST分配给你当前最强的主手武器。这通常是SSR或SS级别，且技能等级较高的武器。
*   **职业契合度：** 确保主手武器与你选择的职业（Job）相符，例如破坏者（Breaker）应装备物理攻击武器，而魔术师（Sorcerer）则应装备魔法攻击武器。

### 2. 副手武器的COST效率考量


副手武器（即自动发动秘技的武器）的选择是COST分配的学问所在。它们虽然不直接参与普攻，但其秘技对整体战力影响巨大。

*   **秘技优先于基础面板：** 选择副手武器时，应优先考虑其秘技的实用性与强度，而非单纯的攻击力或防御力。例如，能够提升全队攻击、减少敌人防御、或增加自身SP恢复的秘技都极具价值。
*   **COST与秘技的平衡：** 寻找那些COST不高但秘技效果卓越的武器。这能让你用更少的COST装备更多有用的秘技，最大化COST效率。
*   **属性与武器类型平衡：** 根据你主要面对的敌人属性，适当配置不同属性的副手武器。同时，也要考虑物理/魔法伤害的平衡，以应对不同敌人的防御类型。

### 3. 职业特性与队伍定位导向


不同的职业在队伍中扮演的角色不同，其武器COST分配策略也应有所区别。

*   **输出型职业（破坏者、魔术师、粉碎者）：** 优先将COST分配给高攻击力、高伤害技能的武器，确保主手武器达到最高输出。副手武器则选择能辅助提升自身伤害或降低敌人防御的秘技。
*   **辅助型职业（圣职者、吟游诗人、守护者）：** 优先确保治疗、增益（Buff）、减益（Debuff）技能的武器能装备到位。主手武器可能不追求极致输出，但副手武器的治疗量、Buff效果、Debuff命中率等秘技至关重要。
*   **混合型职业：** 需根据实际需求在输出与辅助之间取得平衡。例如，某些情况下可能需要牺牲部分输出，以换取更高的生存能力或控制能力。

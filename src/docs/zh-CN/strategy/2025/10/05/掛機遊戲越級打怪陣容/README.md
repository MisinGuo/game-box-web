    *   **后排：** 1-2名持续性伤害输出（如毒系法师、DOT弓箭手）或具备吸血能力的战士。
*   **策略要点：** 确保队伍的续航能力，注意敌方是否有清除Debuff的能力。

### 3. 「单点突破流」：精准打击敌方核心


*   **核心思路：** 针对敌方阵容中有特别棘手的核心（如高输出、高治疗或关键控场），集中火力优先击杀，从而瓦解敌方整体战力。

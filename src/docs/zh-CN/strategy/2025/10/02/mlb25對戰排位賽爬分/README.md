### 2. 投球策略：掌握球路组合与心理战


*   **球路组合与配球：**
    *   **开局试探：** 用不同球种试探打者弱点，观察其对哪些球路反应不佳。

    *   **高低速差：** 快速球与慢速变化球的搭配，能有效制造挥空或软弱击球。
    *   **眼位欺骗：** 让不同球种在出手时看起来相似，但最终轨迹差异巨大。
*   **投球位置与控球：**
    *   **边角球：** 尽量将球投向好球带边缘，增加打者挥空或打成界外球的机率。

*   **心理战：**
    *   **节奏变化：** 故意放慢或加快投球节奏，打乱打者专注力。
    *   **局间观察：** 观察对手上一局的打击表现，调整本局的配球策略。
    *   **假牵制：** 适时的假牵制可以有效限制跑者盗垒。

*   **精准守备：** 熟悉球员的守备范围与传球能力，选择最适合的传球目标。对于高飞球，提前判断落点并移动到位。

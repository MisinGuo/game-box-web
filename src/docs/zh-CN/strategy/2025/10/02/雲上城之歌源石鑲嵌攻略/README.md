### 什么是源石？


源石是镶嵌在装备孔位上，用以提升角色各项属性（如攻击、防御、生命、暴击、韧性等）的特殊道具。它们通常具有独特的属性加成，有些甚至能触发强大的套装效果。

### 源石的种类与品质


源石依据其属性加成方向大致可分为：
*   **攻击型源石**：提升物理攻击、魔法攻击、暴击率、暴击伤害、穿透等。
*   **防御型源石**：提升物理防御、魔法防御、生命值、韧性、格挡等。
*   **功能型源石**：提升命中、闪避、移动速度、技能冷却缩减等。

品质上，源石通常有不同等级，如普通、稀有、史诗、传说等，品质越高，基础属性加成越强，并可能拥有更强大的套装潜力。 2025年可能新增了更高阶的「神话源石」或「觉醒源石」，这些通常伴随着更独特的属性或机制。

### 如何获取源石？ 2025年最新途径


在2025年，获取源石的途径依然多元，但效率与重点可能有所变化：
*   **副本挑战**：高难度团队副本、深渊挑战、限时活动副本是获取高品质源石的主要来源。
*   **活动奖励**：版本更新活动、节庆活动、储值活动等常会提供稀有源石或源石宝箱。
*   **商店兑换**：公会商店、竞技场商店、荣誉商店等，定期更新可兑换的源石种类。
*   **合成与分解**：低品质源石可合成高品质源石，多余源石可分解获得材料，用于强化或再次合成。
*   **交易行/拍卖行**：玩家间自由交易高品质源石，是快速补齐所需源石的重要途径。

1.  **开启镶嵌界面**：点击角色装备栏，选择带有镶嵌孔位的装备。

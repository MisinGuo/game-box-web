### 1. 核心输出/治疗技能优先


无论你的职业是物理输出、魔法输出还是治疗者，那些直接影响你主要战斗表现的技能，始终是附魔的首要目标。更高的伤害、更强的治疗量，意味着更快的击杀速度和更高的团队贡献。

对于辅助职业或那些拥有强大控制技能的输出职业，提升这些技能的成功率、持续时间或减益效果，能显著提升团队的战术灵活性与PVP竞争力。

### 3. 生存能力提升技能随后


在确保输出和辅助能力后，提升自身的生存能力（如防御技能、保命技能、反击技能等）能让你站得更稳，在长时间战斗或高强度PVP中发挥更大作用。

每个技能可选择的附魔类型（如：威力、消耗、机率、时间等）都不同。在选择时，应权衡其对技能的实际提升幅度与资源消耗。例如，对于高消耗的核心输出技能，「消耗」附魔可能不如「威力」附魔来得直接有效。

### 5. 资源管理与风险评估


技能附魔存在失败机率，且失败会导致附魔等级归零。在附魔高阶技能时，务必衡量自身资源（金币、强化卷轴）是否充足，并做好风险管理，避免因一次失败而导致进度停滞。

此阶段主要针对那些使用频率最高、影响最直接的技能进行附魔，以快速提升基础战力。

2.  **主要治疗技能 (治疗量)：**
    *   **补师职业：** 优先附魔你的主要治疗技能，例如：圣骑士（治愈）、主教（高阶治愈）。选择「威力」附魔，确保队友的生存能力。
3.  **核心增益/减益技能 (成功率/持续时间)：**
    *   对于辅助或具有关键控制技能的职业，优先附魔那些能提升队友能力或削弱敌人的技能，例如：先知（精神冲击、加速术）、狂咒术士（黑暗冲击、诅咒）。选择「机率」或「时间」附魔，确保效果稳定。

当你的核心技能已有一定附魔等级后，此阶段应开始强化那些能提升战术深度、扩展角色功能的技能。

    *   强化那些作为主要输出技能补充的次要攻击技能，或是PVP中用于牵制、消耗的技能。

3.  **移动/逃脱/追击技能 (消耗/冷却时间)：**
    *   例如：盗贼的「隐身」、弓手的「后退」。这些技能的附魔可能选择「消耗」或「冷却」，以提高其使用频率与灵活性。
4.  **特殊增益/减益技能 (机率/时间)：**
    *   那些在特定场合（如副本、PVP）能发挥奇效的技能，例如：魅影（沉默术）、死灵法师（吸血术）。

2.  **高阶被动技能 (威力/数值)：**
    *   许多高阶被动技能附魔后能提供可观的属性加成，例如提升攻击力、防御力、施法速度等，这些都是后期追求极致战力的重要环节。

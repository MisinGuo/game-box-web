### 1. 目标锁定优先级（Targeting Priority）


这是自动战斗中最基础也最重要的设定之一，决定了您的勇者会优先攻击哪个敌人。

*   **血量最低（Lowest HP）：** 快速击杀残血敌人，减少敌方输出单位。适用于清怪效率优先的场合。
*   **血量最高/精英怪（Highest HP/Elite）：** 优先处理威胁较大的敌人，避免被高血量单位拖延战斗。
*   **距离最近（Nearest）：** 清理周围敌人，避免被包围或拉到过多仇恨。
*   **威胁度最高/施法者（Highest Threat/Caster）：** 优先击杀对团队威胁最大的敌人，如治疗者或高伤害输出者。
*   **随机（Random）：** 通常不推荐，除非有特殊战术需求。

**2025年建议：** 针对一般练功刷怪，推荐「血量最低」或「距离最近」以提升清怪效率。面对精英或BOSS战，则应调整为「威胁度最高」或「血量最高」，确保优先处理关键目标。

### 2. 技能使用策略（Skill Usage Strategy）


自动战斗不再是无脑放技能，精准的技能配置能让您的勇者在适当时机释放关键技能。

*   **技能优先级（Skill Priority）：** 设定各技能的施放顺序。高优先级技能会在满足条件时优先施放。
*   **技能冷却（Cooldown）：** 某些设定允许您选择「冷却结束立即施放」或「保留技能」。
*   **魔力/能量门槛（MP/Energy Threshold）：** 设定当魔力或能量达到一定百分比时才施放特定技能，避免空蓝。
*   **條件觸發（Conditional Trigger）：** 例如，當隊友血量低於X%時施放治療技能；當敵人有Debuff時施放增傷技能。

**2025年建议：** 务必为核心输出技能、保命技能和控制技能设定恰当的优先级与触发条件。例如，将范围技能设为高优先级用于清怪，而单体爆发技能则设为对精英怪或BOSS优先。

确保您的勇者在危急时刻能自动补充状态，是生存的保障。

*   **生命值門檻（HP Threshold）：** 當生命值低於X%時自動使用生命藥水。
*   **魔力值門檻（MP Threshold）：** 當魔力值低於X%時自動使用魔力藥水。
*   **异常状态解除（Status Effect Removal）：** 自动使用解毒剂、解控药水等。

**2025年建议：** 根据勇者的坦度与队伍配置，设定合理的HP/MP门槛。避免将门槛设得太低导致猝死，也避免太高造成药水浪费。

### 4. 移动与走位设定（Movement


部分游戏允许玩家设定自动战斗时的移动模式。

*   **自动寻路/挂机范围（Auto-Pathing/Grinding Area）：** 设定勇者在指定区域内活动，避免拉到过多怪物或偏离练功点。
*   **躲避技能（Dodge Skills）：** 部分高端AI能设定自动躲避敌方范围技能或预警区域。
*   **近战/远程距离（Melee/Ranged Distance）：** 确保近战角色贴近目标，远程角色保持安全距离。

*   **目标优先级：** 血量最低 > 距离最近。

*   **移动：** 设定明确的挂机范围，避免不必要的移动。

### 2. 精英/BOSS战（Elite/Boss Fights）


*   **目标优先级：** 威胁度最高 > 血量最高。确保优先攻击BOSS或其核心机制单位。

*   **移动：** 如果游戏有躲避机制，务必开启自动躲避。

### 3. 资源采集/生活技能（Resource Gathering/Life Skills）


*   **目标优先级：** 通常无需战斗设定，但可设定为「不攻击」或「只攻击主动攻击的敌人」。

*   **移动：** 设定自动寻路至采集点，或在特定区域内循环采集。

### 4. 团队/PVP（Team/PvP - 如游戏内有此功能）


*   **目标优先级：** 优先攻击敌方治疗者、高输出单位或控制单位。在PVP中，针对性地攻击对手核心角色至关重要。
*   **技能使用：** 协同队友，例如在队友控制住敌人后，自动释放爆发技能。保命技能和控制技能的释放时机更为关键。
*   **移动：** 维持阵型，或根据职业特性保持距离。

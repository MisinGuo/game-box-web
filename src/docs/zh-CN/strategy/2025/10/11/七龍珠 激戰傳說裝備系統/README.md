### 装备强化 (Enhancement)


装备强化是提升装备基础属性的过程。

1.  **消耗与效益：** 强化需要消耗Zeni和强化灵魂。前期可以随意强化，但到了后期，为了追求S或Z 评级，资源消耗会非常巨大。
2.  **随机数值与固定数值：** 装备的每个能力值槽位都有一个数值范围。有些槽位是固定数值，有些则是随机数值。随机数值是决定装备强度的关键。
3.  **目标：S/Z/Z 评级：** 装备强化后会根据其总体属性评级为C、B、A、S、Z或Z 。 S级以上通常代表该装备的随机数值都达到了较高的水平。 Z 是最高评级，意味着所有随机数值都接近或达到最大值，是顶级玩家的追求目标。
4.  **「再链」(Reroll) 机制与技巧：**
    *   **概念：** 当装备的随机数值不理想时，可以使用「再链」功能，消耗「再链宝石」或「清除槽位」来重新随机化某个槽位的数值。
    *   **策略：** 通常建议先将装备强化到所有槽位都开启，然后专注于提升最关键的随机槽位。例如，如果一件装备的第二个槽位是攻击力加成，而其数值偏低，就应优先「再链」这个槽位，直到接近最大值。

    *   **资源管理：** 再链宝石是稀有资源，应优先用于具有潜力冲击Z 的白金或专属装备。

### 如何为不同属性/标签角色选择装备


*   **输出型角色：** 优先选择提升打击攻击力、射击攻击力、特殊攻击力、暴击率的装备。

*   **辅助型角色：** 除了基础防御，可考虑提升气回复速度、技能卡抽取速度或特定团队增益的装备。

### 队伍协同效应与装备配置


*   **平衡性：** 考虑整个队伍的平衡性。如果队伍中输出角色较多，可以为其中一两个角色配置一些提供防御或生命值的装备，增加队伍的生存能力。

### PVP与PVE装备选择差异


*   **PVP (玩家对战)：** 强调爆发力、生存能力和应对Meta的能力。 PVP装备通常追求极致的攻击或防御属性，以及针对特定Meta的抵抗能力。 Z 评级的装备在PVP中至关重要。
*   **PVE (玩家对环境)：** 更注重稳定输出、特定关卡的应对。 PVE装备可以更灵活，例如在需要打断敌人特定行动的关卡中，选择提升气回复速度的装备可能更有效。

### 进阶玩家的装备管理技巧


*   **定期清理：** 定期清理低稀有度、低评级且无潜力的装备，避免背包爆满。
*   **标记重要装备：** 将S级以上或潜力巨大的装备进行标记，防止误操作。
*   **分批强化：** 集中资源强化少数几件核心装备，而不是分散强化大量装备。
*   **关注更新：** 游戏每次更新都可能引入新的强力装备或调整现有装备的获取方式，保持关注是关键。

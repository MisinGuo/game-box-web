#### 通用输出策略：DPS最大化基础

无论你选择哪个职业，以下几点是所有输出玩家都应遵循的原则：

1.  **技能循环与优先级：**
    *   **核心理念：** 永远不要让你的主输出技能闲置，尤其是那些高伤害、短CD的技能。
    *   **爆发技能：** 通常有较长的冷却时间，应在BOSS易伤阶段、队友增益叠加时，或特定机制处理后集中释放，以达到最大效益。
    *   **填充技能：** 在核心技能CD时使用，保持伤害不断档。
    *   **资源管理：** 注意MP/能量条，避免在关键时刻因资源不足而影响输出。
2.  **走位与站位：**
    *   **生存优先：** 活着才有输出。熟悉BOSS技能预警，提前走位躲避伤害。
    *   **输出窗口：** 在安全的前提下，尽可能靠近BOSS（近战）或保持最佳输出距离（远程），确保技能命中率与输出时间。
    *   **背后输出：** 许多BOSS在背后受到的伤害更高，或某些技能只能从背后施放。
3.  **爆发时机：**
    *   与队友的增益技能（如增伤、破防）同步开启自己的爆发，能让DPS瞬间达到巅峰。
    *   利用BOSS的易伤阶段，例如破甲后、控制效果期间，集中火力。
4.  **队伍协作：**
    *   了解队友的职业特性，与辅助职业的增益技能、坦克的聚怪技能形成配合。
    *   在语音或文字频道中沟通，协调爆发时机与集火目标。

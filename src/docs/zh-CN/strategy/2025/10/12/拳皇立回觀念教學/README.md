「立回」（たちまわり, Tachima-wari）源自日语，在格斗游戏中意指「中距离的攻防、移动、牵制与心理博弈」。它不是指一套固定的连段，而是一种动态的、以「控制」为核心的战略思维。在KOF中，立回的目标是：

*   **控制空间：** 透过走位、跳跃、前后 dash 等移动，限制对手的行动范围，或为自己创造有利的进攻或防御位置。
*   **制造机会：** 利用牵制技、假动作、防御反击等手段，引诱对手露出破绽，从而发动连段或造成伤害。
*   **读取对手：** 观察对手的习惯、反应模式，预判其下一步行动。

简单来说，立回就是一场无形的「棋局」，双方玩家在萤幕中段不断试探、博弈，争夺主动权，为最终的击杀创造条件。

### 1. 基础立回元素：移动与攻击距离的掌握


精通立回，首先要从最基本的移动与攻击距离开始。

*   **精准走位与移动：**
    *   **前后走动（Walk）：** KOF中最重要的移动方式，用于微调距离，观察对手。熟练地前后走动，能让你随时保持在最佳攻击或防御位置。
    *   **前后冲刺（Dash/Backdash）：** 快速接近或拉开距离。前冲能迅速缩短与对手的距离，制造突袭机会；后冲则用于紧急回避或创造安全空间。
    *   **跳跃（Jump）：** KOF的跳跃高度与距离都较大，是重要的进攻与防御手段。但同时也是风险最高的移动，容易被对空技击落。学会判断跳跃时机，以及不同跳跃（小跳、中跳、大跳）的应用，是立回的关键。
    *   **紧急回避（Roll）：** 快速穿过对手攻击的无敌移动，但有恢复硬直。善用于躲避弹道攻击或穿过对手防线。

*   **攻击距离掌握：**
    *   **普通攻击（Normal Attacks）：** 轻拳、重拳、轻脚、重脚。每个角色不同方向的普攻都有其射程、速度与判定。熟悉你角色的普攻特性，知道在什么距离下能命中对手，什么距离下会落空，是立回的基础功。
    *   **特殊技（Special Moves）：** 许多特殊技（如波、突进技、对空技）在立回中扮演重要角色。了解它们的有效射程、启动速度、收招硬直，并在适当的距离使用，是牵制对手、制造压力的关键。

### 2. 进阶立回观念：心理博弈与资源管理


当你掌握了基础移动与攻击距离后，便可进一步深入立回的心理层面与资源运用。

#### 2.1 空间控制与心理博弈

*   **压制与退让：**
    *   **压制（Pressure）：** 利用攻击迫使对手防御，限制其行动。例如，用波或长手普攻让对手不敢轻举妄动。
    *   **退让（Zoning/Keepaway）：** 透过远距离攻击或移动，保持安全距离，消耗对手或等待机会。这是许多波升角色的立回核心。
    *   **引诱（Whiff Punish）：** 故意在对手攻击范围外晃动，引诱其出招落空（whiff），然后抓住破绽进行惩罚（punish）。这是高端立回的常见技巧。

*   **假动作与预判：**
    *   **假动作（Feint）：** 透过看似要攻击却取消，或做出特定动作引诱对手反应。例如，假装要跳入，却在空中紧急回避落地。
    *   **预判（Prediction）：** 根据对手的习惯（例如，喜欢在特定情况下跳入或发波），提前做出应对。这需要大量的对战经验和观察力。

#### 2.2 帧数知识与应用

虽然KOF的帧数资讯不如一些游戏那么普及，但了解「有利帧」和「不利帧」的概念，能让你更精准地判断攻防时机。

*   **有利帧（Plus Frames）：** 当你的攻击命中或被防御后，你比对手更快恢复行动，这时你处于有利帧。可以继续压制或发动另一次攻击。
*   **不利帧（Minus Frames）：** 当你的攻击命中或被防御后，你比对手慢恢复行动，这时你处于不利帧。如果对手反应够快，可能会反击你。
*   **确认连段（Confirm Combos）：** 在立回中，许多攻击（如轻攻击）可以作为确认连段的起手。当轻攻击命中后，你有足够的时间判断是否接续更长的连段。这避免了在攻击被防御时，因强行连段而暴露破绽。

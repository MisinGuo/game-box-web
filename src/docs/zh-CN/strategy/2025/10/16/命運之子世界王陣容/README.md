一个理想的世界王阵容，通常由五个角色组成，每个角色都扮演着不可或缺的功能。理解这些定位，是构筑高效队伍的第一步。

### 1. 核心攻击手（Attacker/DPS）


*   **职责：** 队伍的主要伤害来源，透过普攻、滑条技能（Slide Skill）和狂热时间（Fever Time）造成大量伤害。
*   **选择要点：** 需具备高攻击力、强力的单体或范围伤害技能，并能与辅助角色产生良好协同。属性与世界王相克者优先。

### 2. 关键加速手（Speed Buffer/Charger）


*   **职责：** 加速队伍的技能条充能，让攻击手能更频繁地施放技能，尤其是滑条技能。
*   **选择要点：** 拥有能大幅提升技能条充能速度或直接增加技能条的技能。其存在直接影响队伍的总输出上限。

### 3. 强力辅助手（Supporter/Buffer）


*   **职责：** 提升队伍的攻击力、暴击率、暴击伤害，或提供其他增益效果，间接提升整体输出。
*   **选择要点：** 寻找能提供多种强力增益，且增益覆盖率高的角色。

### 4. 有效妨碍手（Debuffer/Disabler）


*   **职责：** 对世界王施加减防、减攻、中毒、诅咒等负面效果，削弱Boss的防御或攻击，或造成持续伤害。
*   **选择要点：** 减防效果通常最为关键，能显著提升全队伤害。其他如中毒、诅咒等持续伤害也能有效累积输出。

### 5. 生存位/弹性位（Healer/Tank/Flex）


*   **职责：** 确保队伍的生存能力，或根据战斗需求提供额外输出或辅助。
*   **选择要点：** 若Boss伤害较高，可选择补师或护盾角色。若队伍生存无虞，此位置可弹性选择第二个攻击手、加速手或更强的辅助/妨碍手来最大化输出。

*   **魂之卡：** 根据角色定位选择。攻击手优先选择增加攻击力、暴击率、暴击伤害的卡片；辅助与妨碍手可选择增加技能条充能、生存或强化自身技能效果的卡片。

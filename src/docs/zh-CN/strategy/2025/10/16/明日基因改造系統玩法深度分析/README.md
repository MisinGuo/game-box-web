基因根据其效果分为多种类型，并有不同的稀有度。

3.  **基因洗练：** 这是提升基因品质的关键环节。洗练可以改变基因的附加词条（副属性）和数值。洗练需要消耗特定的洗练材料，并且结果具有随机性。
    *   **2025年更新：** 新增了「锁定词条」功能，允许玩家消耗更多资源来锁定满意的词条，避免被洗掉，大大降低了洗练的随机性风险。
4.  **基因升级：** 消耗基因经验或重复的基因，可以提升基因的主属性数值。等级越高，增益越显著。

**玩家攻略提示：** 在洗练时，优先锁定对您的职业和流派最有价值的核心词条。对于非核心基因，可以先镶嵌使用，待核心基因成型后再逐步优化。

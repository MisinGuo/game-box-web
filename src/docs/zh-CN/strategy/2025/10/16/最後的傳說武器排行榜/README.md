*   **玩家攻略：** 建议搭配「力量型」部件，将其物理攻击力推向极限。由于其高伤害，可作为快速击杀高血量Boss的核心武器。

*   **最佳使用者：** 物理与魔法输出兼备的单位，如Wyngale、Emmy等。由于其多样化的战技，能适应各种战斗情境。

    4.  **获得贤者之泪：** 完成所有试炼后，迷雾贤者会现身，并将「贤者之泪」赠予你。同时，祭坛后方会开启一道隐藏通道。

*   **任务名称：** 吟游诗人的秘密
*   **触发条件：**
    1.  在绿洲城酒馆内，至少聆听过三位不同吟游诗人的所有故事。

*   **起始地点：** 绿洲城「月光酒馆」内的特定吟游诗人「艾伦」。
*   **详细流程：**
    1.  **收集乐谱：** 前往绿洲城外，找到流浪乐师，完成他给予的「失落的旋律」系列任务，最终获得「黄金乐谱」。
    2.  **聆听故事：** 回到月光酒馆，在不同时段与吟游诗人艾伦对话，聆听他所有的故事。确保你已听过其他至少两位吟游诗人的所有故事。
    3.  **交付乐谱：** 当艾伦的故事重复时，再次与他对话，此时会出现「交付黄金乐谱」的选项。
    4.  **解锁秘密：** 交付乐谱后，艾伦会向你透露一个关于「古代歌谣」的秘密，并要求你前往「回音谷」寻找线索。
    5.  **回音谷探险：** 进入回音谷，按照艾伦的提示，在特定的石壁前使用「黄金乐谱」。石壁会显现出一段古老歌谣的歌词。
    6.  **完成任务：** 将歌谣的歌词带回给艾伦，他会将他的「秘密」传授给你。
*   **任务奖励：**
    *   **吟游诗人专属时装：** 穿戴后魅力值大幅提升，并在与NPC对话时有更高机率触发隐藏选项。
    *   **魅惑之歌（稀有技能）：** 可使周围一定范围内的敌人陷入混乱状态，持续数秒。
    *   **声望：** 大幅提升在绿洲城的声望。
*   **E-E-A-T小撇步：** 「失落的旋律」系列任务需要玩家完成一些乐器演奏小游戏，对节奏感有一定要求。我建议在进行这些小游戏前，先调整游戏音量，确保能清晰听到节拍。艾伦的故事有时需要多次对话才会全部解锁，要有耐心。

理解御魂词条，是精准选择的第一步。每个御魂都有一个「主属性」和数个「副属性」。

### 主属性（固定位置与类型）

![游戏相关](https://oss.2025.10.5yxy5.comstrategy/2025/10/16/陰陽畢業裝備詞條選擇/images/image_5_1760564282.jpg)
御魂共有六个位置，其中1、3、5号位的主属性是固定的：
*   **1号位：攻击**
*   **3号位：防御**
*   **5号位：生命**

而2、4、6号位的主属性则有多种选择，这是我们进行词条选择时的重点：
*   **2号位：** 速度、攻击加成、防御加成、生命加成
*   **4号位：** 攻击加成、防御加成、生命加成、效果命中、效果抵抗
*   **6号位：** 攻击加成、防御加成、生命加成、暴击、暴击伤害

### 副属性（随机生成与强化）

每个御魂在获取时会随机生成2-4条副属性，并在每次强化（ 3、 6、 9、 12、 15）时，随机强化其中一条副属性或新增一条副属性（若不足四条）。理想的毕业御魂往往需要副属性也完美契合式神需求，这也是御魂刷取的难点所在。

### 1. 输出型式神（DPS）


输出式神的核心目标是造成最大伤害，因此词条选择围绕攻击、暴击、暴击伤害展开。

*   **2号位：** 速度（先手输出）、攻击加成（后手或慢速输出）
*   **4号位：** 攻击加成
*   **6號位：** 暴擊（優先堆滿100%）、暴擊傷害（暴擊達標後）

**副屬性優先級：** 暴擊傷害 > 暴擊率（補足100%） > 速度 > 攻擊加成

**2025年趋势：**
*   **高爆伤队伍：** 若能确保暴击率达标（如配合荒骷髅、鬼灵歌伎等御魂），则6号位直接选择暴击伤害，副属性狂堆暴击伤害。
*   **速度输出：** 对于需要先手爆发的输出（如一些斗技式神），2号位速度，副属性追求高速与高暴伤。

### 2. 控制型式神（CC）


控制型式神需要确保其控制技能能够命中敌人，并在敌人行动前出手。

*   **2号位：** 速度（绝对优先，越快越好）
*   **4号位：** 效果命中
*   **6号位：** 生命加成 / 防御加成（提升生存能力）

**副属性优先级：** 速度 > 效果命中 > 生命加成 > 防御加成

**2025年趋势：**
*   **极速高命中：** 在高抵抗环境下，控制式神对速度和效果命中的要求越来越高。副属性务必追求双高。
*   **兼顾生存：** 部分控制式神（如星熊童子）可能需要一定的生命抵抗来保证自身不被反控或秒杀。

### 3. 辅助型式神（Support）


辅助式神的作用是提供增益、拉条、驱散等，其核心在于稳定性和速度。

*   **2号位：** 速度（PVP辅助必备，越快越好；PVE可考虑生命加成）
*   **4号位：** 生命加成 / 防御加成 / 效果抵抗（视功能而定）
*   **6号位：** 生命加成 / 防御加成

**副属性优先级：** 速度 > 生命加成 > 效果抵抗 > 防御加成

**2025年趋势：**
*   **高速高抵抗：** 在斗技中，辅助式神往往是敌方集火或控制的目标，高速度确保先手，高效果抵抗则能有效反制控制。
*   **特定功能辅助：** 部分辅助（如驱散类）可能需要堆叠一定的效果抵抗来确保自身行动。

### 4. 治疗/坦伤型式神（Healer/Tank）


这类式神的核心是生存能力，确保队伍的续航。

*   **2号位：** 速度（部分治疗需抢先手）、生命加成（纯粹堆肉）
*   **4号位：** 生命加成 / 防御加成
*   **6号位：** 生命加成 / 防御加成

**副属性优先级：** 生命加成 > 防御加成 > 速度 > 效果抵抗

**2025年趋势：**
*   **极致生命/防御：** 在面对高爆发输出时，纯粹的生命与防御加成是保证生存的基石。
*   **抵抗治疗：** 部分治疗式神（如花鸟卷）在斗技中可能需要堆叠一定的效果抵抗，以避免被控制而无法治疗。

### 1. 速度阈值与配速


在PVP中，理解速度阈值至关重要。你的拉条式神速度必须快过对面，输出式神速度必须紧随拉条式神。这要求精确计算每个式神的速度，通过副属性调整来达到最佳配速。 2025年的斗技环境对速度的要求只增不减，极速御魂依然是稀有资源。

### 2. 效果命中与效果抵抗的博弈


随着新式神不断推出，高抵抗阵容和高命中控制的对抗日益激烈。
*   **效果命中：** 建議控制式神效果命中堆疊至120%-150%以上，以應對普遍的高抵抗環境。
*   **效果抵抗：** 輔助、治療和部分輸出式神，若有自保需求，可適當堆疊效果抵抗，避免被敵方關鍵控制打亂節奏。通常建議堆疊至100%以上才會有明顯效果。

### 3. 特定御魂套装与词条搭配


某些御魂套装本身就带有属性加成，这会影响你的词条选择。例如：
*   **破势、心眼：** 这些输出御魂通常会与暴击、暴伤词条搭配，追求极致伤害。
*   **招财猫、火灵：** 这些供火御魂，2号位通常选择速度，副属性优先速度，以确保供火稳定。
*   **地藏像、薙魂：** 这些生存御魂，通常与生命、防御、抵抗词条搭配，提升坦度。

### 4. PVP与PVE的微调


*   **PVP：** 更注重速度、效果命中/抵抗的平衡，以及针对特定阵容的克制。往往需要多套御魂方案。
*   **PVE：** 则更强调极致输出或稳定续航。例如，在魂土、逢魔等副本中，追求单一极限伤害词条的输出式神更受欢迎。

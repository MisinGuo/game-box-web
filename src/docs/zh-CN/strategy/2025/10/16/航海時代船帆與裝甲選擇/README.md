### 主要船帆类型与特性


*   **方帆（Square Sails）：**
    *   **优点：** 顺风航速极快，适合长距离直线航行，尤其是在贸易路线上效率卓越。
    *   **缺点：** 逆风性能差，转向迟钝，不适合频繁变向或近战缠斗。
    *   **适用船舰：** 大型贸易船、远洋探险船。
*   **三角帆（Lateen Sails）：**
    *   **优点：** 逆风性能优异，转向灵活，操控性极佳，适合近岸航行或海战中频繁调整角度。
    *   **缺点：** 顺风航速不如方帆，直线加速较慢。
    *   **适用船舰：** 小型战斗船、侦察船、近海贸易船。
*   **混合帆（Mixed Sails）：**
    *   **优点：** 结合方帆与三角帆的优点，在顺风与逆风性能上取得平衡，提供较好的综合表现。
    *   **缺点：** 在单一极致性能上不如专精帆种。
    *   **适用船舰：** 中型多功能船、平衡型战斗舰。
*   **特殊/传说帆（如：魔帆、风精之帆等）：**
    *   **优点：** 拥有独特的属性加成，例如额外加速、风力抗性、甚至特殊技能，能大幅提升船舰性能。
    *   **缺点：** 获取难度高，通常需要参与特定活动、高难度副本或稀有掉落。
    *   **适用船舰：** 追求极致性能的旗舰、PvP专用船。

*   **木质装甲（Wooden Armor）：**
    *   **优点：** 重量轻，修理成本相对较低，适合初期或追求轻量化的船舰。

*   **铁质装甲（Iron Armor）：**
    *   **优点：** 防御力显著提升，性价比高，是中期船舰的标准配置。

    *   **适用船舰：** 中型贸易船、一般战斗船。
*   **钢质装甲（Steel Armor）：**
    *   **优点：** 高防御力，耐久度极佳，能有效抵御猛烈炮火。

    *   **适用船舰：** 大型战列舰、旗舰、PvP专用重甲船。
*   **特殊合金装甲（如：秘银装甲、钛合金装甲等）：**

    *   **缺点：** 获取难度极高，通常是顶级玩家的追求。
    *   **适用船舰：** 追求极致生存能力与综合性能的顶级船舰。

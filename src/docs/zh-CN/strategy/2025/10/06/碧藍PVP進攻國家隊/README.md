    *   **前排：** 安克拉治 (或 普利茅斯)、岛风、海伦娜改
*   **战术思路：**
    *   **信浓/白龙：** 提供绝对的航空伤害输出，信浓的特殊弹幕和白龙的鱼雷机爆发是主要输出点。

    *   **前排：** 强化生存能力的装备（如：维修工具、灭火器、高性能舵机）与提升输出或辅助能力的装备（如：SG雷达、改良版液压弹射器）。

    *   **厌战改/君主：** 提供精准的炮击输出，厌战改的「神射手」技能在PVP中能有效打击敌方关键舰船。

    *   **普利茅斯：** 作为前排核心，其辅助与承伤能力极佳。
    *   **伦敦改：** 强力的重巡炮击输出与破甲能力。
    *   **贝尔法斯特改：** 烟雾弹能有效保护前排，并提供稳定的输出。
*   **装备配置重点：**
    *   **后排：** 优先选择高爆、高射速的主炮，搭配提升航空或生存的装备。
    *   **前排：** 强化生存、鱼雷输出或控制能力的装备。
*   **优势：** 攻防兼备，具有一定的控制能力，能应对多种PVP阵容。
*   **劣势：** 爆发力不如极速航空流，对操作和时机把握要求较高。

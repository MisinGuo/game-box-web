### 1. 输出型侠客（例如：郭靖、杨过、黄蓉、小龙女等）


**核心目标：** 最大化伤害输出，确保技能命中与爆发。

*   **加点顺序建议：**
    1.  **主动攻击技能（核心输出技）：** 优先加满，这是伤害的主要来源。例如郭靖的「亢龙有悔」、杨过的「黯然销魂掌」。
    2.  **与主动技能联动的被动技能：** 提升主动技能的伤害、触发概率或附加效果。例如提升暴击率、暴击伤害、无视防御等。
    3.  **增伤型光环/被动：** 提升自身或队友的攻击力、最终伤害等。
    4.  **生存/续航被动：** 提供吸血、减伤或复活等，确保输出环境。
    5.  **次要控制/辅助技能：** 在输出技能加满后，再考虑。

*   **2025年加点提示：** 部分输出侠客在2025年版本中，会因特定状态（如中毒、灼烧）而获得额外伤害加成。若有此类特性，应确保队伍中有能施加这些状态的侠客，并优先提升与状态联动的技能。

### 2. 防御型侠客（例如：周伯通、洪七公、欧阳锋等）


**核心目标：** 最大化生存能力，有效承担伤害，保护队友。

*   **加点顺序建议：**
    1.  **减伤/免伤被动技能：** 优先加满，这是坦克生存的基石。例如提升防御、韧性、生命值上限或直接提供减伤效果。
    2.  **嘲讽/聚怪主动技能：** 确保能有效吸引敌人火力，保护后排。
    3.  **生命恢复/护盾技能：** 提升自身或队友的续航能力。
    4.  **反击/反伤被动：** 在保证生存的基础上，提供少量输出或控制。
    5.  **增益/控制辅助技能：** 例如提供队友防御加成或对敌方施加弱化。

*   **2025年加点提示：** 随着敌方伤害的普遍提高，坦克的“硬度”比以往任何时候都重要。优先投资于直接减伤和生命值的技能，其次才是反击或控制。部分侠客的“援护”或“格挡”机制在2025年得到了强化，与之相关的技能应给予更高权重。

### 3. 辅助型侠客（例如：瑛姑、一灯大师、黄药师等）


**核心目标：** 提升队友整体战力，提供治疗、增益或清除负面状态。

*   **加点顺序建议：**
    1.  **核心治疗/护盾技能：** 优先加满，确保队伍续航。例如瑛姑的治疗、一灯大师的群体回复。
    2.  **关键增益/减益技能：** 提升队友攻击、防御、速度或降低敌方属性。例如黄药师的控制与增益。
    3.  **净化/免疫负面状态技能：** 在面对多控制或多负面状态的敌人时尤为关键。
    4.  **能量恢复/技能冷却缩减：** 帮助队友更快释放技能。
    5.  **次要输出/控制技能：** 在辅助能力足够后，可适当提升。

*   **2025年加点提示：** 辅助侠客的“功能性”在2025年版本中被进一步强调。除了治疗和增益，能否有效应对敌方的控制和负面状态，成为衡量一个辅助侠客强弱的重要标准。优先提升净化和免疫相关的技能。

### 4. 控制型侠客（例如：裘千仞、金轮法王、部分黄药师等）


**核心目标：** 有效限制敌方行动，打断关键技能，创造输出环境。

*   **加点顺序建议：**
    1.  **核心控制技能（眩晕、冰冻、沉默、变形等）：** 优先加满，提升控制概率、持续时间和作用范围。
    2.  **与控制技能联动的被动：** 提升控制命中率、控制后的额外效果（如减防、易伤）。
    3.  **次要控制技能：** 提供补充性的控制效果。
    4.  **生存/速度被动：** 确保控制侠客能活下来并先手释放控制。
    5.  **少量输出技能：** 在控制能力足够后，可适当提升。

*   **2025年加点提示：** 控制型侠客的“效果命中”属性在2025年版本中变得更加重要。除了技能等级，装备和阵容搭配也应考虑提升效果命中，以确保控制技能的稳定性。

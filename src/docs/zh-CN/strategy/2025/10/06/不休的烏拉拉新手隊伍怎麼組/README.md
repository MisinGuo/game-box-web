### 推荐配置一：均衡推进型 (1坦2输出1治疗)

这是一个非常经典且稳定的配置，适合绝大多数新手玩家。它提供了足够的坦度与治疗，同时拥有不错的输出能力，推图效率高。

    *   **战士**：技能组较为全面，能抗能打，是早期不错的选择。
*   **输出1：猎人 / 法师**
    *   **猎人**：单体输出稳定，且拥有宠物协同伤害，推图打BOSS都很出色。
    *   **法师**：群体输出能力强，在清理小怪关卡时效率极高。
*   **输出2：刺客 / 术士**
    *   **刺客**：高爆发单体伤害，适合针对单一强敌。
    *   **术士**：持续伤害（DOT）能力突出，且有召唤物分担伤害，生存力较强。
*   **治疗：萨满 / 德鲁伊**

**缺点**：爆发力可能不如极端输出队伍，但对新手而言更为友好。

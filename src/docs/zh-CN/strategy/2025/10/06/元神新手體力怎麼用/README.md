这个阶段应以推进主线任务、探索地图、开启传送点和神像为主。体力（树脂）非常有限，且此时掉落的材料等级较低，不建议大量投入。

*   **体力优先用于：**
    *   **摩拉地脉（启示之花）：** 当急缺摩拉时，少量刷取。

    *   **世界首领（如无相系列、急冻树、爆炎树）：** 仅在您急需特定角色的突破材料时，针对性刷取。例如，抽到主C角色需要突破。

    *   **圣遗物秘境：** 此阶段不会掉落金色圣遗物，投入体力是巨大浪费。
    *   **天赋培养秘境：** 同理，材料等级低，效率不高。

    *   **摩拉地脉与经验书地脉：** 根据需要适量刷取，保证角色和武器的升级消耗。

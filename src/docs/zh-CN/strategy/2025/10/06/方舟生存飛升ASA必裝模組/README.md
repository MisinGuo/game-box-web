1.  **启动游戏**：进入《方舟：生存飞升》。
2.  **进入主选单**：在主选单界面，您会看到一个「Mod」或「模组」选项。
3.  **浏览与订阅**：点击进入模组页面，您可以浏览CurseForge上的所有ASA模组。找到您想安装的模组后，点击「订阅」或「安装」。

### 1. 目标选定策略：优先攻击的艺术


这是自动战斗的首要决策点，直接影响战斗效率与战损。游戏通常提供多种目标选定逻辑：

*   **血量最低（Lowest HP）**：适合快速击杀敌方弱势单位，或集火目标。在PVE中，这有助于加速清怪。
*   **威胁最高（Highest Threat）**：优先攻击对我方造成最大威胁的敌人，例如敌方输出核心或控场单位。在PVP中，这能有效解除威胁。
*   **优先攻击类型（如：炮舰、战列舰、商船等）**：针对特定目标进行打击。例如，在打劫商船时，优先攻击商船。
*   **距离最近（Closest Target）**：最直接的选择，但可能导致火力分散。

**2025年建议**：对于PVE练功，通常选择「血量最低」或「威胁最高」以提高清怪速度。 PVP则需根据舰队配置和战术，灵活切换，例如优先集火敌方高输出船只。

### 2. 技能施放逻辑：智慧运用舰队能力

*   **指定技能自动施放（Specific Skill Auto Cast）**：允许您选择哪些技能自动施放，哪些保留手动。这是最佳化自动战斗的关键。
*   **CD优先（Cooldown Priority）**：技能冷却时间一到就施放。
*   **條件觸發施放（Conditional Cast）**：例如，當我方單位血量低於X%時施放治療技能。

**2025年建议**：建议仔细检视每个船员的技能。将**增益效果（Buff）、持续伤害（DoT）**或**低冷却时间的输出技能**设定为自动施放。而**高爆发、强控场或关键保命技能**，则考虑保留手动，以便在关键时刻精准介入。例如，船长的战术技能往往需要手动判断施放时机。

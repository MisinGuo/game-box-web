### 任务流程与类型详解：知己知彼，百战不殆


国家舰队任务的具体内容会随机变动，但通常围绕以下几种核心类型展开：

1.  **海战型任务：**
    *   **目标：** 击败特定海域的海盗舰队、敌国舰队或特殊NPC船只。
    *   **特点：** 强调船只性能、炮术技巧与团队协同作战。
2.  **贸易型任务：**
    *   **目标：** 在限定时间内，将某种交易品运送到指定港口，或在特定港口进行买卖以达到利润目标。
    *   **特点：** 考验玩家对市场行情的判断、贸易路线的规划以及船只载货量。
3.  **冒险探索型任务：**
    *   **目标：** 发现特定地点、寻找隐藏宝藏、调查遗迹或生物，通常需要解谜或战胜守护者。
    *   **特点：** 需要玩家具备丰富的冒险技能与地理知识。
4.  **生产型任务：**
    *   **目标：** 收集特定材料并在指定地点生产出任务所需的物品。
    *   **特点：** 需要玩家具备相关生产技能，并熟悉材料来源。

对于新手玩家而言，加入一个活跃的家族是快速成长的最佳途径：

3.  **选择「创建家族」**：输入你家族的名称（需符合游戏规范，且名称唯一）。
4.  **设定家族宣言与加入条件**：这是吸引其他玩家加入的关键。

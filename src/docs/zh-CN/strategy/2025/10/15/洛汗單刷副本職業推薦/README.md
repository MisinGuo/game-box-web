*   **核心优势：** 极高的生存能力、优秀的范围伤害与自我恢复。圣骑士是单刷的「铁壁」，能轻松应对大量怪物的围攻。其神圣系的技能不仅能造成伤害，还能提供治疗与防御增益。

    *   **生存核心：** 「圣盾术」（大幅减伤）、被动技能提升防御与血量上限。 「生命祝福」提供持续恢复。
    *   **战术：** 利用「神圣之怒」聚怪后，开启「圣盾术」硬吃伤害并循环输出。面对Boss时，依靠高防御与治疗技能进行磨耗战。
*   **装备与属性建议：**
    *   **属性：** 主点力量（提升攻击力与防御），次点体质（提升血量与防御）。
    *   **装备：** 优先选择增加防御、血量、减伤率的重甲。武器选择高攻击力的单手剑，搭配盾牌。
*   **适合副本类型：** 任何需要高生存与稳定输出的副本，尤其是在怪物数量多、伤害高的地图表现出色。
*   **操作难度与注意事项：** 操作相对简单，容错率高。但清怪速度可能不如纯输出职业，需要耐心。

### 2. 刺客 (Assassin)


*   **核心优势：** 极致的单体爆发、高机动性与优秀的闪避能力。刺客擅长逐个击破，或利用爆发技能快速秒杀精英怪与Boss。其隐身技能在某些副本中也具有战术价值。
*   **技能配置与战术：**
    *   **主要输出：** 「致命打击」、「背刺」、「影袭」等高倍率单体技能，配合「毒液」持续伤害。
    *   **生存核心：** 「闪避强化」、「疾风步」（提升闪避率与移动速度），「烟雾弹」提供短暂无敌或脱战。
    *   **战术：** 利用高机动性拉开距离，精准计算技能CD，对高威胁目标进行爆发输出。面对群怪时，可利用位移技能风筝或优先击杀远程怪。
*   **装备与属性建议：**
    *   **属性：** 主点敏捷（提升攻击力、暴击率、闪避），次点力量（提升基础攻击）。
    *   **装备：** 优先选择增加攻击力、暴击率、暴击伤害的轻甲。武器选择双持匕首。
*   **适合副本类型：** 适合需要快速击杀单一高威胁目标的副本，或对操作要求较高、能发挥闪避优势的场景。
*   **操作难度与注意事项：** 操作难度较高，需要精准的走位和技能释放时机。生存能力依赖闪避，一旦被命中可能较脆弱。

*   **核心优势：** 毁灭性的范围魔法伤害、强大的控制技能。魔导师是清怪效率的王者，能够瞬间蒸发大量怪物。
*   **技能配置与战术：**
    *   **主要输出：** 「陨石术」、「暴风雪」、「连锁闪电」等大范围毁灭性魔法。
    *   **生存核心：** 「魔力护盾」（吸收伤害）、冰系技能的减速与冰冻控制。
    *   **战术：** 利用控制技能将怪物聚集，然后释放高伤害范围魔法进行清场。面对Boss时，保持距离，利用单体高伤魔法（如「火球术」强化版）与控制技能风筝。
*   **装备与属性建议：**
    *   **属性：** 主点智力（提升魔法攻击力、MP），次点体质（提升血量）。
    *   **装备：** 优先选择增加魔法攻击力、暴击率、MP恢复的法袍。武器选择魔杖或法杖。
*   **适合副本类型：** 任何怪物数量多、需要快速清场的副本。在熟悉怪物刷新点后，能达到极高的刷怪效率。
*   **操作难度与注意事项：** 缺乏物理防御，生存能力较差，需要良好的走位和控场意识。一旦被怪物近身，容易陷入危机。

### 4. 祭司 (Priest) - 暴力奶妈流


*   **核心优势：** 独特的「暴力奶妈」流派在2025年版本逐渐受到玩家青睐。祭司不仅拥有强大的治疗和辅助能力，部分攻击技能配合特定装备和属性加点后，也能提供不俗的输出，同时保持极高的续航。
*   **技能配置与战术：**
    *   **主要输出：** 「神圣之光」（强化后伤害可观），搭配「惩戒之锤」或「神圣冲击」。
    *   **生存核心：** 「治愈术」、「恢复术」等强大单体/范围治疗，以及「防御结界」等防御增益。
    *   **战术：** 利用自身的治疗能力，几乎可以无视小怪的伤害，边打边回。面对Boss时，保持治疗不断，利用攻击技能慢慢磨耗。
*   **装备与属性建议：**
    *   **属性：** 智力与精神并重（智力提升攻击与治疗量，精神提升MP与魔法防御）。部分玩家会投入少量力量提升物理防御。
    *   **装备：** 选择增加魔法攻击、治疗量、MP恢复、防御力的法袍。武器选择权杖。
*   **适合副本类型：** 适合需要极高续航、稳定推进的副本。清怪速度可能较慢，但胜在安全无虞。
*   **操作难度与注意事项：** 输出效率不如纯输出职业，需要较长的副本时间。但其安全性极高，是新手玩家单刷的稳妥选择。

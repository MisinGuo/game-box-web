### 什么是防御？理解你的保护盾

![游戏相关](https://oss.2025.10.5yxy5.com/strategy/2025/10/01/KOF防禦技巧入門/images/image_5_1759301369.jpg)

在KOF中，防御（Guard）是指在对手攻击来临前，透过方向键指令来抵挡伤害的行为。当你成功防御时，角色会进入防御状态，虽然不会受到伤害，但会累积防御硬直（Guard Stun）并可能被破防（Guard Crush）。

*   **后方向键防御：** 最基本的防御方式，向后按住方向键即可。
*   **空中防御：** 部分KOF作品允许空中防御，但通常有较多限制或特殊条件。对于入门玩家，先专注地面防御即可。

### 站立防御 vs. 下蹲防御：应对上中下段攻击


这是KOF防御中最重要的区分，也是新手最常犯错的地方。

*   **站立防御（Standing Guard）：** 按住后方向键。可以防御所有「上段」和「中段」攻击。
*   **下蹲防御（Crouching Guard）：** 按住下后方向键。可以防御所有「中段」和「下段」攻击。

**核心原则：**
*   **上段攻击：** 站立防御。
*   **中段攻击：** 站立或下蹲防御皆可。
*   **下段攻击：** 必须下蹲防御。

**入门玩家提示：** 由于下段攻击通常起手较快且容易被忽略，新手建议在不确定对手攻击类型时，优先选择下蹲防御，因为它能防御中段和下段，而中段攻击相对较少需要站防才能防御的。但切记，面对跳入攻击或特定中段技能时，仍需及时切换至站防。

### 防御硬直与破防：防御的代价


即使成功防御，你也不是完全安全的。

*   **防御硬直（Guard Stun）：** 每次成功防御攻击时，你的角色会进入短暂的硬直状态，无法立即行动。攻击力越强或段数越多的招式，造成的防御硬直通常也越长。
*   **破防（Guard Crush）：** KOF角色都有一个隐藏的防御槽。当你持续防御大量攻击，或防御了某些具有「破防属性」的招式时，防御槽会耗尽，角色会进入一段长时间的晕眩状态，此时完全无法行动，任人宰割。

**入门玩家提示：** 了解防御硬直和破防的存在，是避免被对手压制致死的关键。不要盲目地按住防御键不放，学会适时反击或回避，才是上策。

### 2. 滚动回避（Roll）：KOF独有的机动性防御


滚动回避是KOF系列最具代表性的防御/回避机制之一，也是入门玩家必须熟练掌握的技巧。

*   **指令：** 轻拳 轻脚（A B）
    *   **前滚（Forward Roll）：** 向前按住方向键   A B。角色向前翻滚，期间有短暂无敌时间。
    *   **后滚（Backward Roll）：** 向后按住方向键   A B。角色向后翻滚，期间有短暂无敌时间。
*   **应用场景：**
    *   **回避波升类飞行道具：** 在对手发出飞行道具的瞬间，透过前滚穿过道具，接近对手。
    *   **脱离版边压制：** 当你被逼到版边时，后滚可以帮助你拉开距离，避免被持续压制。
    *   **回避投技：** 投技无法被防御，但可以在对手发动投技的瞬间使用滚动回避来躲开。
    *   **绕背偷袭：** 在某些情况下，前滚可以绕到对手身后，制造意想不到的攻击机会。

**入门玩家提示：** 滚动回避虽然有无敌时间，但结束后会有硬直。因此，不能无脑乱滚。学会判断滚动的时机和方向，是从新手迈向进阶的标志。

### 3. 紧急回避（Emergency Evasion / Blowback Attack）：攻防转换的利器


紧急回避在KOF中通常指代「重拳 重脚」（C D）的指令，它既是防御手段，也是反击手段。

*   **指令：** 重拳 重脚（C D）
    *   **站立紧急回避：** 站立状态下按C D。角色会向前冲刺并发动一个吹飞攻击，期间有短暂无敌时间。
    *   **下蹲紧急回避：** 下蹲状态下按C D。角色会原地发动一个吹飞攻击，期间有短暂无敌时间。
*   **应用场景：**
    *   **打断对手连段：** 在对手连续攻击的空档，利用紧急回避的无敌时间打断其攻势并将其吹飞。
    *   **反击跳入攻击：** 当对手跳入攻击时，可使用站立紧急回避将其击落。
    *   **创造距离：** 在被压制时，紧急回避可以将对手击退，为自己争取喘息空间。

**入门玩家提示：** 紧急回避的吹飞攻击在命中后，对手会被击飞到版边，为你争取到宝贵的追击或调整时间。但若未命中，其硬直时间较长，容易被对手惩罚。

### 4. 防御反击（Guard Cancel）：进阶防御的序曲


防御反击是KOF系列中一种高阶的防御技巧，但入门玩家也应了解其概念，并尝试在简单情境下应用。

*   **指令：** 在防御硬直中，输入特定指令（通常是方向键前   轻拳 重拳 (A C) 或 轻脚 重脚 (B D)）。
*   **效果：** 消耗一格能量槽（Power Gauge），在防御硬直中立即发动一次反击攻击。
*   **应用场景：**
    *   **打断对手压制：** 当对手持续用连段压制你时，防御反击可以强制打断其攻势，并造成伤害。
    *   **创造连段机会：** 在某些情况下，防御反击命中后可以衔接自己的连段，实现攻防转换。

**入门玩家提示：** 防御反击需要消耗能量，且时机要求较高。对于新手，先熟悉基本防御和滚动回避，之后再逐步尝试防御反击。在训练模式中多加练习，掌握在防御硬直中输入指令的感觉。

### 5. 受身（Ukemi）：倒地后的快速起身


受身不是直接的防御，但它能有效减少你倒地后的被动时间，避免被对手追击。

*   **指令：** 倒地前瞬间，按任意两个攻击键（通常是A B或C D）。
*   **效果：** 角色在被击倒的瞬间，会快速弹起并恢复站立状态，缩短倒地时间。
*   **应用场景：**
    *   **避免追击：** 对手很多连段在击倒后会进行追击，受身可以让你快速起身，打乱对手的节奏。
    *   **调整位置：** 受身后可以选择向前或向后移动，调整有利位置。

**入门玩家提示：** 受身是KOF中非常基础但极其重要的生存技巧。务必在训练模式中多加练习，让它成为你的肌肉记忆。

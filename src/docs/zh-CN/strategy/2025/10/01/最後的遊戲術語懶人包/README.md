*   **Quest (任务)**：游戏中需要完成的目标，通常有奖励。

*   **Cloud Gaming (云端游戏)**：透过网路串流在远端伺服器上运行游戏，玩家无需高性能硬体。
*   **Latency (延迟)**：操作指令从玩家端发送到伺服器，再返回画面的时间差，对云端游戏体验至关重要。

*   **Web3 Gaming (Web3 游戏)**：基于区块链技术，强调去中心化和玩家拥有游戏资产的游戏。

### 4. 人工智慧 (AI) 与游戏设计

### 承伤舰船的类型与特性

理解不同承伤类型，是做出明智选择的第一步：

*   **高血量/重甲型（如超巡、重巡）**：依赖高生命值和重型装甲硬吃伤害。她们通常拥有较高的防御属性，能有效抵御物理伤害，但面对穿甲弹或无视防御的攻击时可能较为脆弱。
*   **高回避型（如驱逐、轻巡）**：透过极高的回避值来规避敌方攻击。这类舰船的生存能力波动较大，在面对高命中率敌人或技能锁定时可能表现不佳，但在闪避成功时几乎无伤。
*   **护盾/减伤型（如科研超巡、部分重巡）**：透过技能生成护盾、提供减伤Buff或Debuff敌人来降低所受伤害。这是当前版本最主流且高效的承伤方式，能有效应对各种伤害类型。
*   **自我恢复型（如部分重巡、轻巡）**：透过技能在战斗中恢复自身血量。这类舰船能延长战斗续航，但通常需要一定的时间才能发挥最大效益。

### 2025年承伤舰船选择的关键考量


*   **泛用性**：能否应对多数战场环境，包括主线推图、活动高难关卡、作战档案、演习等。
*   **稳定性**：承伤能力是否稳定可靠，不易受敌方配置或随机性影响。
*   **功能性**：除了承伤，是否能提供额外的辅助（如增伤、减速、控制）。
*   **获取与培养成本**：对于新手玩家，易于获取和培养的舰船更具吸引力。

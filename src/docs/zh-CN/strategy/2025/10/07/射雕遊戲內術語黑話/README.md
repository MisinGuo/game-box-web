### (一) 战斗与角色定位篇


这是游戏中最频繁使用的术语，直接关系到团队合作与战斗成败。

*   **T / 主T / 副T：**
    *   **解释：** 「Tank」的缩写，指队伍中负责吸引敌人火力、承受伤害的「坦克」角色。主T是主要承受伤害者，副T则在必要时支援或分担压力。
    *   **情境：** 「团本缺个T，来个能扛的！」
*   ** dd / dps：**
    *   **解释：** 「Damage Dealer」或「Damage Per Second」的缩写，指队伍中负责输出伤害的角色。
    *   **情境：** 「这波集火DD跟上，秒掉那只精英怪！」
*   **奶 / 补：**
    *   **解释：** 指队伍中负责治疗、恢复生命值的「治疗」角色。
    *   **情境：** 「T血量危险，奶妈注意抬血！」
*   **控场 / CC：**
    *   **解释：** 「Crowd Control」的缩写，指拥有控制技能（如晕眩、冰冻、沉默等）的角色或其技能，用于限制敌人行动。
    *   **情境：** 「优先控场对面奶妈，断她技能！」
*   **集火：**
    *   **解释：** 指队伍成员将所有攻击目标集中在一个敌人身上，以最快速度击杀。
    *   **情境：** 「BOSS出小怪了，优先集火击杀！」
*   **走位：**
    *   **解释：** 指玩家在战斗中移动、调整位置的技巧，以规避伤害、创造输出空间或执行战术。
    *   **情境：** 「BOSS范围技，注意走位躲开！」
*   **秒伤：**
    *   **解释：** 「每秒伤害」的缩写，用于衡量角色或技能的输出效率。
    *   **情境：** 「这套心法搭配能大幅提升我的秒伤！」

### (二) 装备与经济篇


理解这些术语能帮助你更好地进行装备搭配、资源管理与交易。

*   **毕业装 / 神装：**
    *   **解释：** 指当前版本中，角色能够获得的最顶级、最适合其流派的装备组合。
    *   **情境：** 「我这套毕业装刷了快一个月才凑齐。」
*   **白嫖：**
    *   **解释：** 指不花费任何现实金钱，仅透过游戏内活动、任务或肝度来获取资源、装备或福利。
    *   **情境：** 「这次活动奖励不错，白嫖党也能拿到稀有材料。」
*   **肝帝：**
    *   **解释：** 形容那些投入大量时间、精力在游戏中，透过长时间游玩来获取资源或提升实力的玩家。
    *   **情境：** 「那位肝帝一天能刷十几个小时的副本。」
*   **金砖 / 绑金 / 元宝：**
    *   **解释：** 游戏内货币的不同形式。金砖通常指可自由流通、交易的游戏币；绑金是绑定角色、无法交易的游戏币；元宝通常是透过储值获得的高级货币。
    *   **情境：** 「急需金砖收个材料，私聊报价！」
*   **洗练 / 附魔 / 镶嵌：**
    *   **解释：** 装备强化系统中的不同环节。洗练通常指重随装备属性；附魔指为装备增加额外属性；镶嵌指将宝石等物品嵌入装备以提升能力。
    *   **情境：** 「这件装备洗练出极品词条了，终于毕业！」
*   **狗粮：**
    *   **解释：** 指用于强化其他装备或道具的低阶材料，通常会被消耗掉。
    *   **情境：** 「包里一堆蓝装狗粮，该清空了。」

这些术语能帮助你更好地参与帮派、组队活动与社群交流。

*   **帮战 / 团本 / 世界王：**
    *   **解释：** 游戏内不同规模的PVP（玩家对战）或PVE（玩家对环境）活动。帮战是帮派间的对抗；团本是需要多个队伍协作的副本；世界王是全服玩家共同挑战的强大敌人。
    *   **情境：** 「今晚八点帮战，所有兄弟上线集合！」
*   **招募 / 喊人：**
    *   **解释：** 在世界频道或特定频道发布组队或招募帮派成员的讯息。
    *   **情境：** 「XX副本招募DD奶妈，来个高战力！」
*   **亲友团：**
    *   **解释：** 指由现实朋友或游戏内关系密切的玩家组成的固定队伍或帮派。
    *   **情境：** 「我们是亲友团，只收熟人。」
*   **PY交易：**
    *   **解释：** 源自「屁眼交易」的谐音，指私下、非公开的交易或合作，带有一定程度的戏谑与不正当性，但游戏中多指私下协商而非公开竞价的物品分配。
    *   **情境：** 「这件稀有装备我们PY一下，你出多少？」
*   **日常 / 周常 / 限时：**
    *   **解释：** 游戏中每天、每周或在特定时间段内可完成的任务或活动。
    *   **情境：** 「日常还没清完，今天又得熬夜了。」

理解这些有助于你规划角色发展方向。

*   **流派：**
    *   **解释：** 指角色在技能、心法、装备搭配上的特定倾向，形成一套独特的战斗风格或功能。
    *   **情境：** 「我玩的是毒攻流派，爆发很高。」
*   **属性 / 面板：**
    *   **解释：** 角色各项能力值的总称，如攻击、防御、生命、暴击等。面板通常指角色资讯介面显示的总体数据。
    *   **情境：** 「这件装备加的属性很适合我的面板。」
*   **内功 / 外功：**
    *   **解释：** 《射雕》武学体系中的两种主要攻击类型，分别对应不同的门派或技能加成。
    *   **情境：** 「我是外功门派，需要叠高外功攻击。」

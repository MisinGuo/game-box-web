*   **阵地防守：** 在有利地形建立防线，利用坦克宠物吸引伤害，输出宠物在后方稳定输出。

*   **面对速攻流：** 优先保护我方核心，利用控制技能打断敌方爆发，并迅速反击。
*   **面对消耗流：** 集中火力突破其防线，击杀治疗或辅助宠物，避免被拖入持久战。
*   **面对控制流：** 优先击杀敌方控制宠物，或利用免疫控制的技能/宠物进行反制。

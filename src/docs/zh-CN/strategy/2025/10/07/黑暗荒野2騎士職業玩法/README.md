骑士的能力值配点应根据你的主要玩法流派来决定。以下是2025年主流的配点策略：

### 1. 力量 (Strength)


*   **核心收益**：直接提升物理攻击力，并少量增加生命值。
*   **推荐度**：所有骑士流派都应优先考虑。对于输出型骑士，力量是主要属性；对于防御型骑士，在满足生存的基础上，适量力量也能提高清怪效率。

### 2. 体质 (Constitution)


*   **核心收益**：大幅提升生命值上限，增加物理防御与魔法防御。
*   **推荐度**：防御型骑士的首要属性，能显著提升坦度。输出型骑士也应投入一定点数，以确保基本的生存能力。

### 3. 敏捷 (Dexterity)


*   **核心收益**：提升攻击速度、暴击率，并少量增加闪避率。
*   **推荐度**：对于追求高频输出与暴击伤害的骑士流派有较高价值。若选择双手武器或重视技能爆发，可适当投入。

### 4. 智力 (Intelligence) 与 精神 (Spirit)


*   **核心收益**：智力提升魔法攻击力，精神提升魔法防御与法力值。
*   **推荐度**：骑士基本不依赖这两个属性，除非有特殊装备或技能搭配，否则不建议投入。

### 2025年主流配点建议：


*   **防御型骑士 (主坦)**：主体质，副力量。例如，每升一级3点体质、1点力量。
*   **输出型骑士 (狂暴)**：主力量，副体质或敏捷。例如，每升一级3点力量、1点体质或敏捷。
*   **均衡型骑士 (PVE)**：力量与体质均衡发展，或略偏力量。例如，每升一级2点力量、2点体质。

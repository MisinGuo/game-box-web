防御型英雄负责吸收伤害，保护后排输出，是队伍稳定的基石。

*   **代表类型：** 装甲指挥官、高生命值与防御力的英雄。
*   **选择标准：** 具备减伤、回血、嘲讽或控制技能，能有效限制敌人输出。
*   **培养重点：** 优先提升生命值、防御力、减伤率，以及生存相关的技能等级。

#### 3. 实用辅助型英雄 (Support)

辅助型英雄能为队伍提供增益、治疗或控制，是提升整体战力的关键。

*   **代表类型：** 医疗兵、战术支援型英雄。
*   **选择标准：** 拥有队伍增益（攻防加成）、敌方减益（降低属性）、治疗或强力控制技能。

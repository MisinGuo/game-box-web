*   **解释：** 快速移动一段距离，用以躲避敌人攻击或调整战位。通常有冷却时间或消耗体力。

*   **解释：** 举起武器或身体进行防御，减少或完全抵挡受到的伤害。部分游戏格挡成功后有反击机会。

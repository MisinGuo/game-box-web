*   **类型特点**：以极速直球和强劲的速球系变化球为主，追求三振出局。拥有高球速和力量属性，能有效压制打者。

    *   **能力配点**：优先提升「控球」与「变化球」，其次是「精神」。控球决定进垒点，变化球影响球种威力。

        *   **精准控球/球路变化**：提升控球精准度与变化球轨迹。
        *   **滚地球制造者/飞球诱捕**：增加特定出局方式的机率。
        *   **稳定投球**：减少因疲劳导致的控球失准。
    *   **推荐球种**：曲球、滑球、变速球、切球、螺旋球等，至少掌握3-4种不同轨迹的变化球。

*   **核心养成策略**：
    *   **能力配点**：优先提升「变化球」与「控球」，适当分配「精神」以维持特殊球种的稳定性。
    *   **核心技能**：
        *   **特殊球种精通**：大幅提升特定特殊球种的威力与控球。

*   **精神 (MEN)**：影响投手在危机时刻的表现，以及能否稳定发挥。高压局面下显得尤为重要。

**2025年建议**：均衡发展体力与精神，即使是力量型投手也需要足够的控球来避免保送。

技能是提升投手实力的关键。除了上述类型专属技能外，以下是通用且高效的**投手技能**：

**2025年建议**：优先升级能直接提升投球表现或稳定性的技能。考虑搭配能弥补自身弱点的技能。

*   **投手手套**：通常提升控球、变化球或力量。
*   **球棒/护具**：部分装备会提供额外的体力或精神加成。
*   **服装/鞋子**：可能提供速度、守备或特殊效果。

**2025年建议**：关注2025年赛季推出的新装备，它们往往具有更强的属性加成或套装效果。根据你的投手类型，选择能最大化其核心属性的装备。

### 4. 球种搭配艺术


单一强力的球种不足以应对高阶打者。有效的**投手球种**搭配是致胜的关键。

*   **直球系 (四缝线、二缝线、切球)**：作为基础，用来抢好球数或迷惑打者。
*   **变化球系 (滑球、曲球、变速球)**：制造挥空或破坏打者节奏。
*   **特殊球种 (指叉、蝴蝶)**：作为杀手锏，在关键时刻使用。

**2025年建议**：至少掌握一种高速球、一种横向变化球、一种纵向变化球和一种变速球。尝试不同的球路组合，观察打者的反应。

### 5. 实战投球策略

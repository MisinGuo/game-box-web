### 1. 旋律掌控者（Control Dancer）


*   **核心特色：** 擅长透过持续性的节奏干扰、减速或暂停效果，打乱对手的舞步连贯性，为自己创造输出或恢复的机会。通常拥有较高的韧性或自保能力。
*   **对战优势：**
    *   **强大控场：** 能有效限制高爆发或快节奏职业的发挥。
    *   **战术主导：** 透过控场技能主导战局节奏，迫使对手跟随你的步伐。
    *   **持久战能力：** 通常具备一定的恢复或防御技能，适合拉长战线。
*   **对战劣势：**
    *   **爆发不足：** 自身输出能力相对较弱，需要较长时间才能击败对手。
    *   **操作要求：** 精准的技能施放时机是关键，失误可能导致节奏全失。
*   **实战心得与攻略（2025年版本）**
    *   **核心战术：** 利用「混乱乐章」或「节奏冻结」等技能，在对方准备施放关键高分舞步前进行干扰。
    *   **关键技能：** 优先升级具备范围控场或长时间减益效果的技能。
    *   **应对快攻：** 善用位移或无敌技能规避初期爆发，待对手技能空档再反手控场。
    *   **2025年调整：** 部分控场技能的冷却时间略微缩短，提升了持续压制能力，但其基础伤害仍维持中等水平。

### 2. 节奏爆发者（Burst Dancer）


*   **核心特色：** 以瞬间高分舞步连击为核心，追求在短时间内击败对手。通常拥有较高的敏捷或攻击加成，但相对脆弱。
*   **对战优势：**
    *   **极致爆发：** 能在极短时间内打出海量分数，让对手措手不及。
    *   **压制力强：** 一旦抓住机会，能迅速建立巨大分数优势。
    *   **操作爽快：** 连招流畅，给予玩家极大的成就感。
*   **对战劣势：**
    *   **生存力低：** 容易被控场或高持续伤害职业针对。
    *   **容错率低：** 一旦连招中断或技能空放，会陷入被动。
*   **实战心得与攻略（2025年版本）**
    *   **核心战术：** 抓住对手技能空档或失误，一套「极限连舞」或「终极音浪」带走。
    *   **关键技能：** 着重提升高伤害或具备破防效果的舞步技能。
    *   **应对控场：** 预留解控或位移技能，在被控制后迅速脱离并寻找反击机会。
    *   **2025年调整：** 爆发技能的动画帧数略有优化，使得连招衔接更为流畅，但同时也增加了部分技能的能量消耗。

### 3. 魅惑辅助者（Support Dancer）


*   **核心特色：** 不以直接输出为主要手段，而是透过增益自身或削弱对手，为队友（或自身在PVP中创造有利环境）提供支援，或在持续战中展现强大韧性。
*   **对战优势：**
    *   **战术多样：** 能根据战局变化，提供治疗、护盾、加速或减益效果。
    *   **持续作战：** 具备较强的生存与恢复能力，能拖垮对手。
    *   **团队核心：** 在团队战中作用巨大，但在PVP中也有一席之地。
*   **对战劣势：**
    *   **单挑能力弱：** 缺乏直接的爆发或控制手段，难以快速击败对手。
    *   **依赖战术：** 需要精准判断施放技能的时机，否则效果不彰。
*   **实战心得与攻略（2025年版本）**
    *   **核心战术：** 在PVP中，辅助者更像是一种「消耗战术」的执行者。利用「治愈之舞」或「魅惑光环」等技能，不断恢复自身状态或削弱对手，等待对方失误。
    *   **关键技能：** 优先提升具备护盾、治疗或持续减益效果的技能。
    *   **应对爆发：** 在对手爆发前预判并开启护盾或减伤技能，拖延时间。
    *   **2025年调整：** 部分辅助技能的持续时间增加，并且新增了「反击节奏」效果，在自身受到攻击时有机会触发小幅反击，提升了PVP中的威胁性。

### 4. 战术舞者（Tactical Dancer）


*   **核心特色：** 技能组均衡，既有一定输出，也具备控制或自保能力。擅长根据战局变化灵活切换策略，是多面手型的职业。
*   **对战优势：**
    *   **适应性强：** 能应对多种对手，战术选择丰富。
    *   **攻守兼备：** 既能输出，也能在关键时刻自保或控制。
    *   **稳定性高：** 不容易被单一战术完全克制。
*   **对战劣势：**
    *   **精通难度：** 需要对所有技能有深刻理解，才能发挥最大潜力。
    *   **上限不明显：** 在某个单一领域可能不如专精职业。
*   **实战心得与攻略（2025年版本）**
    *   **核心战术：** 观察对手，利用「动态舞步」或「变奏冲击」等技能，在攻击、防御和控制之间灵活切换。
    *   **关键技能：** 均衡提升各类技能，特别是那些具备多重效果的复合技能。
    *   **应对所有职业：** 根据对手职业特性，选择相应的开局策略。例如，对抗爆发者时优先防御，对抗控场者时寻找输出机会。
    *   **2025年调整：** 战术舞者的「舞步转换」技能获得强化，现在切换形态后会短暂提升下一个技能的效果，鼓励玩家更多地进行战术变换。

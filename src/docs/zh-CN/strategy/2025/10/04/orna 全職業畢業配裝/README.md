在深入探讨各职业配装前，有些通用原则是所有Orna毕业玩家都必须掌握的：

### 1. 属性优先级：理解你的职业核心


*   **物理输出职业（如Beowulf, Heretic, Nekromancer）**：优先追求**攻击力（Attack）**、**敏捷（Dexterity）**（影响命中与闪避）、**暴击率（Crit Rate）**。

*   **防御型职业（如Titanguard, Grand Attuner）**：优先追求**防御（Defense）**、**抗性（Resistance）**、**生命值（HP）**、**守护值（Ward）**及其回复。
*   **辅助型职业（如Deity, Realmshifter）**：根据其辅助类型，可能需要**魔法力（治疗）**、**防御/抗性（生存）**，或特定的**状态抗性**。

### 2. 装备品质与锻造等级：神铸是基本


所有毕业装备都应至少是**传奇（Legendary）**或**完美（Perfect）**品质，并尽可能提升至**神铸（Godforged）**等级。神铸能极大提升装备基础属性，是毕业配装的基石。

### 3. 饰品镶嵌（Adornments）：微调与强化


饰品（Adornments）是微调属性、弥补短板的关键。选择与职业核心属性相符的饰品，例如：
*   **攻击/魔法输出**：**魔眼（Eye of Anu）**、**龙眼（Dragon Eye）**、**暴击宝石（Crit Jewel）**。
*   **防御/抗性**：**守护石（Ward Stone）**、**抗性宝石（Resistance Jewel）**、**防御宝石（Defense Jewel）**。
*   **特殊需求**：**状态抗性宝石（Status Resist Jewels）**、**经验/金币/Orn宝石（Exp/Gold/Orn Jewels）**（用于刷图）。

### 4. 武器与副手搭配：双持或盾牌？


*   **双持（Dual Wield）**：提供高额伤害加成，适合追求极致输出的职业。
*   **盾牌（Shield）**：提供大量防御、抗性与守护值，适合坦克或需要更高生存能力的职业。
*   **副手武器（Off-hand Weapon）**：部分职业（如Beowulf的兽王爪）或特定战术会选择副手武器以获取额外属性或技能。

### 5. 元素与状态异常：针对性防御与进攻


针对特定副本或PvP对手，准备一套具有**元素抗性**或**状态异常抗性**的装备至关重要。同时，高阶武器自带的**元素伤害**或**状态异常触发**也能有效提升输出。

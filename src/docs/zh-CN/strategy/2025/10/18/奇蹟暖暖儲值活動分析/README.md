*   **评估方式：** 计算每1元台币（或等值货币）能获得多少钻石。

*   **美观度：** 最主观但也最重要的标准。一套你真心喜欢的套装，其价值远超数据。

*   **评估方式：** 将附赠的金币、体力、幻阁券等资源，换算成等价钻石，再加总计算总体性价比。

### 4. 个人预算与游戏目标：量力而为


*   **评估方式：** 结合自身经济能力与游戏中想达成的目标（如集齐特定套装、冲刺竞技场排名）。

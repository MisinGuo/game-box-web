### 实战操作技巧


1.  **速度规划**：
    *   **供火/拉条**：通常要求一速，确保开局有火。
    *   **控制**：紧随供火之后，抢先控制敌方。
    *   **增伤**：在输出之前，最大化伤害。
    *   **输出**：在所有增益和控制之后，打出最高伤害。
    *   **治疗/驱散**：根据战况调整，确保队伍生存。
2.  **技能施放时机**：
    *   **鬼火分配**：协同对弈中鬼火资源共享，需要与队友沟通，避免抢火。
    *   **关键技能**：SP缘结神的结缘、千姬的潮汐之音、帝释天的金莲等，要在最佳时机施放。
    *   **观察敌方行动条**：预判敌方行动，提前布置控制或防御。
3.  **与队友沟通**：
    *   **选人阶段**：提前与队友商量好选人方向，避免式神重复或职能缺失。
    *   **战斗中**：利用游戏内置的语音或文字交流，提示队友集火目标、鬼火使用、技能施放等。
4.  **应对突发状况**：
    *   **减员**：优先复活或保护核心输出/辅助。
    *   **鬼火不足**：优先普攻，等待供火式神行动。
    *   **被控**：利用驱散式神或等待控制结束。

5.  **预览效果：** 染色介面会即时显示染色后的预览效果。这是非常重要的步骤，务必仔细观察。

### 4. 不同部位染色技巧


*   **头饰：** 通常作为点睛之笔，可选择与主体服装形成对比的亮色，或与发色、瞳色呼应。
*   **上衣/下装：** 这是时装的主体，决定了整体风格。建议先确定这两个部位的颜色，再搭配其他部件。
*   **鞋子：** 常常被忽略，但一双配色得体的鞋子能让造型更加完整。可选择与裤子/裙子同色系，或作为小面积撞色。

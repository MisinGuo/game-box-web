### 技能类型与属性


宠物技能主要分为：

*   **主动技能：** 宠物在战斗中会自动施放，通常具有冷却时间。例如「连击」、「破甲」、「治愈之歌」。
*   **被动技能：** 持续生效的增益或减益效果，无需施放。例如「坚韧」、「弱点」、「鼓舞」。

根据效果，技能可概括为：

*   **输出型：** 提升伤害，如「连击」、「破甲」、「狂暴」。
*   **辅助型：** 治疗、增益己方，如「治愈之歌」、「鼓舞」、「净化」。
*   **控制型：** 晕眩、沉默、减速等，如「冲撞」、「冰冻」。
*   **防御型：** 提升生存能力，如「坚韧」、「格挡」、「护盾」。

### 技能搭配原则：效益最大化


选择宠物技能时，应遵循以下原则：

1.  **互补性：** 弥补队伍短板或强化优势。例如，队伍输出不足时，选择输出型宠物技能；生存压力大时，选择防御或治疗型。
2.  **叠加效益：** 某些技能组合能产生「1 1>2」的效果。例如，破甲与高伤害技能搭配，能显著提升总体输出。
3.  **针对性：** 根据敌人特性（单体/群体、高防/高攻、有无控制）调整技能。
4.  **宠物定位：** 不同宠物有其天生属性与成长偏向，选择与其定位相符的技能，效益最佳。例如，攻击型宠物搭配输出技能，防御型宠物搭配防御技能。

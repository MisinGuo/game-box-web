### 1. 主词条与副词条的区别

![游戏场景](https://oss.2025.11.5yxy5.com/images/2025/10/22/%E5%85%83%E7%A5%9E%E8%81%96%E9%81%BA%E7%89%A9%E8%A9%9E%E6%A2%9D%E9%81%B8%E6%93%87/image_2_1761068596.jpg)

*   **主词条 (Main Stat):** 每个圣遗物部位都有一个主要属性，其数值固定且远高于副词条。生之花固定为生命值、死之羽固定为攻击力。时之沙、空之杯、理之冠则有多种主词条选项，选择正确的主词条是圣遗物搭配的第一步。
*   **副词条 (Substats):** 每个圣遗物初始会随机生成2-4条副词条。在强化过程中​​，每提升4级，会随机强化一条现有副词条或新增一条副词条（如果不足4条）。副词条的随机性是圣遗物培养最「痛苦」也最「刺激」的部分。

### 2. 核心词条类型与作用


理解不同词条的作用，是进行元神圣遗物词条选择的基石：

*   **攻擊力百分比 (ATK%) / 基礎攻擊力 (Flat ATK):** 增加角色攻擊力。百分比詞條收益更高。
*   **生命值百分比 (HP%) / 基礎生命值 (Flat HP):** 增加角色生命值。百分比詞條收益更高，對依賴生命值增傷或提供護盾的角色至關重要。
*   **防禦力百分比 (DEF%) / 基礎防禦力 (Flat DEF):** 增加角色防禦力。百分比詞條收益更高，主要用於依賴防禦力增傷（如荒瀧一斗、諾艾爾）或提供護盾的角色。
*   **元素精通 (Elemental Mastery):** 提高元素反应（蒸发、融化、感电、超导、扩散、燃烧、绽放、超绽放、烈绽放）的伤害和效果。
*   **元素充能效率 (Energy Recharge):** 提高角色获取元素能量的速度，影响元素爆发的循环频率。
*   **暴击率 (CRIT Rate) / 暴击伤害 (CRIT DMG):** 暴击率决定攻击暴击的机率，暴击伤害决定暴击时造成的额外伤害。这对所有输出角色都是最重要的输出词条。
*   **治疗加成 (Healing Bonus):** 提高治疗角色提供的治疗量。
*   **物理伤害加成 (Physical DMG Bonus):** 提高物理攻击造成的伤害，主要用于物理输出角色。
*   **元素伤害加成 (Elemental DMG Bonus):** 提高特定元素攻击造成的伤害（如火元素伤害加成、水元素伤害加成等），对元素输出角色至关重要。

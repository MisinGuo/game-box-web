*   **显示模式：全萤幕模式**

### 2. 画面品质细项调整


这些是影响FPS的关键设定，建议根据自身硬体配置进行调整。

*   **材质品质：中 / 低**
    *   **优化理由：** 高材质品质会大量占用显示记忆体。对于《希望恋曲》这类游戏，中或低品质已足够清晰，且能显著提升FPS。

    *   **优化理由：** 降低角色细节可以减少游戏在处理大量角色模型时的负担，尤其在城镇或团战中效果显著。
*   **背景细节：低**
    *   **优化理由：** 背景细节对游戏体验影响较小，但对FPS影响较大。设为低可以大幅减轻显卡压力。
*   **特效显示：低 / 关闭**
    *   **优化理由：** 技能特效、环境特效是FPS杀手。将其设为低或完全​​关闭，能确保你在战斗中视野清晰，且不会因特效过多而卡顿。
*   **阴影品质：关闭**
    *   **优化理由：** 阴影是消耗GPU资源的大户，且对游戏战斗体验影响微乎其微。关闭阴影能带来显著的FPS提升。
*   **观看距离：中/低**
    *   **优化理由：** 适度降低视野距离可以减少游戏需要渲染的物件数量。在PvE或PvP中，中等视野已足够观察周围情况。
*   **垂直同步 (V-Sync)：关闭**
    *   **优化理由：** 垂直同步虽然能消除画面撕裂，但会增加输入延迟并可能限制FPS。对于竞技性较强的玩家，建议关闭以追求最低延迟和最高FPS。

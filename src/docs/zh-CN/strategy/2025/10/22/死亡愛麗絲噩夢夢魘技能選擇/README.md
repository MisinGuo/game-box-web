要精准选择噩梦，首先必须理解其技能的运作机制。噩梦技能主要分为主动与被动两大类，而主动技能又可细分为多种效果。

### 主动技能：扭转战局的爆发点


噩梦的主动技能是其最直接、最显著的影响力来源，通常伴随着华丽的动画和强大的效果。

*   **全体增益/减益型：** 对我方全体施加攻击/防御/魔攻/魔防提升，或对敌方全体施加相同属性降低。这是最常见也最实用的类型，是公会战中集火或防御的关键。
*   **单体爆发型：** 对单一敌人造成巨额伤害，或对单体我方提供巨额恢复。多用于针对高威胁目标或紧急治疗。
*   **状态控制型：** 使敌人陷入沉默、麻痹、束缚等状态，或解除我方负面状态。在面对特定敌人或PvP中，这类技能具有决定性作用。
*   **资源管理型：** 恢复SP、降低技能冷却时间等。这类噩梦能有效提升队伍的续航能力和技能循环。
*   **特殊环境型：** 根据战场环境（如特定属性增伤、特定武器类型增伤）提供特殊效果。

### 被动技能：持续增益与策略布局


噩梦的被动技能虽然不如主动技能那般显眼，但在战斗全程中提供稳定且持续的加成。这些加成通常与噩梦的属性、武器类型或特定条件相关。

*   **基础属性提升：** 提升召唤者或全队的攻击、防御等基础属性。
*   **特定武器/属性强化：** 提升特定武器类型（如刀剑、法书）或特定属性（如火、水）的伤害或防御。
*   **特殊条件触发：** 例如，当HP低于一定百分比时触发回复，或在公会战特定时间段内提供额外加成。

### 技能发动时机与战术考量


精通噩梦技能的关键在于「时机」。

*   **集火阶段：** 在公会战的集火指令下，同时发动多个增伤型噩梦，能瞬间爆发出毁灭性伤害。
*   **防御阶段：** 敌人集火时，发动减伤或提升防御的噩梦，能有效减少我方损失。
*   **特殊Boss战：** 针对Boss的特定技能，发动解除状态或免疫状态的噩梦。
*   **PvP对决：** 预判对手行动，在对方准备爆发时施加控制，或在我方关键角色濒危时提供保护。

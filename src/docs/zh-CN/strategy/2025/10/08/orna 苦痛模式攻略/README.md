*   **盔甲：**
    *   **高防御/抗性：** 优先选择提供大量防御（DEF）和抗性（RES）的重甲或轻甲。
    *   **状态免疫：** 这是苦痛模式的生命线！至少要有一件装备提供「麻痹免疫」（Paralysis Immunity）或「中毒免疫」（Poison Immunity）。 「燃烧免疫」（Burn Immunity）、「睡眠免疫」（Sleep Immunity）也极为重要。

*   **饰品：**
    *   **状态免疫饰品：** 弥补防具的不足，确保你对多种致命状态免疫。
    *   **属性增强饰品：** 提升关键属性，如暴击率、穿透、吸血或法力回复。
    *   **经验/奥恩加成：** 在确保生存的前提下，佩戴这类饰品能最大化收益。

苦痛模式要求你的技能栏位必须高效。

    *   **减益（Debuffs）：** 「战吼」（Warfare）、「破甲」（Shatter）等降低敌人属性的技能在对抗高防御敌人时非常有效。

*   **位移/控制：** 部分职业拥有位移或控制技能，能帮助你在战斗中调整位置或限制敌人。

**玩家建议：** 根据你即将挑战的敌人类型，预设多套技能配置。例如，面对高物理防御的敌人时，切换到魔法伤害技能；面对高魔法抗性的敌人时，则切换到物理伤害技能。

### 4. 宠物与追随者：忠实的战场伙伴


宠物在苦痛模式中扮演着至关重要的角色。

*   **输出型宠物：** 如「魔眼」（Greater Mimic）、「巴哈姆特」（Bahamut）等，提供稳定的额外伤害。
*   **辅助型宠物：**
    *   **治疗型：** 如「仙女」（Fairy）、「独角兽」（Unicorn），能提供持续的生命回复，大大增加续航。
    *   **增益型：** 部分宠物能为你施加增益效果，如「巨龙」（Dragon）的防御提升。
    *   **减益型：** 部分宠物能对敌人施加减益效果。

**玩家建议：** 在苦痛模式中，一只能提供治疗或关键增益的宠物往往比纯输出宠物更具价值。特别是对于缺乏自我治疗能力的职业，治疗宠物几乎是必备。

### 5. 战斗策略与资源管理：精打细算，步步为营


*   **优先击杀威胁：** 在面对多个敌人时，优先击杀那些能施加致命状态或拥有高爆发伤害的敌人。
*   **利用属性弱点：** 观察敌人的属性，使用克制该属性的技能或武器能造成更多伤害。
*   **保持增益：** 在战斗开始前和战斗中，时刻保持你的关键增益效果。
*   **适时减益敌人：** 对于高血量或高防御的敌人，先施加减益效果再输出，会显著提高效率。
*   **药水与卷轴：** 不要吝啬使用生命药水、法力药水和各种增益卷轴（如「巨人药剂」（Scroll of Giants）、「魔法药剂」（Scroll of Magic））。它们是你在关键时刻扭转战局的关键。

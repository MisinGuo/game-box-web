十天众是游戏中一组独特的限定角色，每位都代表着一种武器类型，并拥有极其强大的战斗能力与独特辅助效果。他们在各种高难度副本、古战场（Guild War）以及日常推图中都能发挥关键作用，是构筑强大队伍不可或缺的核心成员。拥有十天众，不仅能显著提升您的战力，更是一种个人成就的展现。

### 制作流程总览


碧蓝幻想十天众的制作流程主要分为以下几个核心阶段：

1.  **荣誉武器取得与强化：** 获取并将特定「荣誉武器」强化至SSR。
2.  **天星器制作与强化：** 将荣誉武器转化为「天星器」，并进行三凸、四凸。
3.  **天星器真化：** 消耗大量素材，将天星器「真化」并进行属性变更。
4.  **依代武器制作：** 制作与真化天星器同属性的「依代武器」。
5.  **天星器解放战：** 完成特定的角色解放战（フェイトエピソード）。

这是一个需要极大耐心与策略的过程，但每一步的努力都将让您更接近目标。

### 荣誉武器的取得与强化


十天众的制作始于「荣誉武器」（朽ち果てた武器）。这些武器是制作天星器的基础，共有十种，每种对应一位十天众。

*   **取得途径：** 荣誉武器主要透过以下方式获得：
    *   **社区使命：** 可转换为共产党商店期间使用的“勇气之光”。
    *   **活动奖励：** 部分游戏活动会提供荣誉武器作为奖励。
    *   **古战场商店：** 在古战场活动期间，可使用「战货」兑换。

*   **强化至SSR化（段阶突破）：** 获得荣誉武器后，需要将其强化至SSR等级。这通常需要消耗大量的武器狗粮和卢比。请确保您的武器库中有足够的强化素材。

### 制作所需稀有素材一览


在制作十天众的过程中，您将会面临各种稀有素材的收集挑战。以下是几个关键且需求量大的素材：

    *   **获取之旅：**普通商店（兑换勇气光辉）、高难度活动副本、荣宇商店（兑换荣誉光辉）。
*   **究竟之证（究竟の证）：** 每位十天众需要3个。
    *   ** Catch：**研讨会商店（改变战争的光彩），高活动的主管和Yongshun Shoten（改变了宝石的光彩）。
*   **各属性星晶之欠片：** 数量需求巨大，用于天星器真化及属性变更。
    *   **获取途径：** 每日岛主、方阵（Magna）副本、活动、商店兑换。
*   **属性元素（エレメント）：** 大量用于武器强化与真化。
    *   **获取途径：** 分解SSR武器与召唤石、活动商店、共斗商店。

**效率刷法提示：**

### 天星器基础强化


当您的荣誉武器达到SSR后，下一步是将其转化为「天星器」。这一步骤通常在游戏内的「武器」介面进行。

*   **天星器化：** 消耗少量素材后，荣誉武器会转变为天星器。
*   **三凸（3★）：** 将天星器强化至满级，并使用相同的天星器或「ダマスカス钢」（大马士革钢）进行三次突破上限。
*   **四凸（4★）：** 达到三凸后，需要使用「金刚晶」或特定活动素材将其突破至四凸。

### 天星器真化流程


天星器真化是制作十天众最为耗费资源的环节之一。真化后的天星器将获得更强大的能力，并为后续的十天众解放奠定基础。

*   **真化所需素材：**
    *   **真なるアニマ（真之魂）：** 每个十天众需要10个，透过击败特定的高难度星晶兽（如方阵HL）或在荣誉商店兑换。
    *   **各属性星晶之欠片：** 数量庞大，通常需要数千个。
    *   **属性元素：** 数千个。
    *   **卢比：** 数百万。
    *   **其他特定素材：** 依据不同天星器可能会有额外需求。

*   **属性变更的策略建议：**
在真化过程中，您需要为天星器选择一个属性。这个属性将决定您最终招募的十天众的属性。
    *   **考量因素：**
        *   **队伍需求：** 您的哪个属性队伍最需要补强？
        *   **素材库存：** 您哪个属性的星晶之欠片和元素最充足？
        *   **古战场准备：** 下一次古战场的有利属性是哪个？
    *   **重要提示：** 属性一旦选定，将无法更改，请务必慎重选择。

这是招募十天众的最后一道门槛——一场特殊的角色解放战。

*   **战斗内容：** 解放战通常分为几个阶段，您需要使用自己的队伍击败特定的敌人。这些战斗往往具有一定的难度，考验玩家的队伍构筑和战斗技巧。

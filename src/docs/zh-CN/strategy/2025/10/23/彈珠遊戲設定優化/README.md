### 画面设定：清晰度与流畅度的黄金平衡


画面设定直接影响你的视觉体验与反应时间。找到效能与画质的最佳平衡点是关键。

*   **解析度 (Resolution)：** 始终选择显示器原生解析度。若效能不足，可适度降低，但会牺牲画面清晰度。 2025年，主流显示器已普及2K甚至4K，确保你的游戏能充分利用这些高解析度。
*   **垂直同步 (V-Sync) 与帧率限制 (FPS Cap)：**
    *   **V-Sync：** 开启可消除画面撕裂，但可能引入输入延迟。若显示器支援G-Sync或FreeSync，优先启用这些技术。
    *   **FPS Cap：** 建议将帧率限制在显示器刷新率（如144Hz）或略低于此值，以保持稳定，避免不必要的硬体压力。
*   **纹理品质 (Texture Quality)：** 影响机台表面、弹珠细节的清晰度。若显示卡记忆体充足，建议设为「高」或「极高」。
*   **反锯齿 (Anti-aliasing, AA)：** 消除画面边缘的锯齿感，使图像更平滑。 FXAA、TAA是常见选项，TAA效果通常更好但可能略微模糊。根据效能选择。
*   **阴影 (Shadows) 与反射效果 (Reflections)：** 这些效果能极大提升视觉真实感，但也最消耗效能。在保证流畅的前提下，尽量调高。
*   **2025年趋势：高刷新率萤幕、HDR与光线追踪**
    *   **高刷新率萤幕 (120Hz/144Hz )：** 为弹珠游戏带来前所未有的流畅感，让弹珠移动更平滑，判断更精准。务必确保游戏帧率与之匹配。
    *   **HDR (高动态范围)：** 提供更广阔的色彩范围与对比度，让机台灯光、特效更加绚丽夺目。
    *   **光线追踪 (Ray Tracing)：** 部分高端弹珠模拟游戏已开始支援，能带来极其真实的光影与反射效果，但对硬体要求极高。权衡效能与视觉享受。
